# ロビー階段〜ホール階段統合図構造.md — まとめ（要点）
ロビー階段は吹き抜け側へ張り出す“開放の縦軸”、ホール階段は内部へ沈む“閉じた縦軸”として対照的に構成される

境界は壁・段差・天井ラインで明確に分離され、動線が混ざらない

ロビー階段はロビーの明るさと視界の広がりを保ち、ホール階段は音と光を抑えた静かな導線になる

両階段は吹き抜けを介して立体的に独立し、視界の衝突が起きない

2Fに上がった後の動線も分離され、ロビー側は通路へ、ホール側は客席へ自然に誘導される

統合図の目的は“開放の階段 → 境界 → 静けさの階段”という段階的な空間切り替えを成立させること
