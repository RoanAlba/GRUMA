
# ロビー階段・ホール階段・バルコニー 三層立体図（文章版）.md
1. 三層の全体構造（縦軸のまとめ）
下層：ロビー階段（白層・生命感）  
↓
中層：ホール階段（黒層・静けさ）  
↓
上層：バルコニー（黒層の最深部・音の中心）

GRUMA の縦軸は、
白 → 黒 → 黒の最深部  
という “沈み込むような上昇” で構成されている。

2. 下層：ロビー階段（白層・生命感）
・白層の空気の中を上昇
・階段（内側）
・手すり（低い白金）
・滑り台（外側）
・猫の動きが儀式として見える
・上昇の終点はロビー2F通路（白→黒の移行帯）

ロビー階段は 生命感の層。

3. 中層：ホール階段（黒層・静けさ）
・黒層の空気の中を上昇
・階段は黒大理石
・手すりは白金の儀式線（内側のみ）
・外側は黒層の余白
・滑り台は存在しない
・静けさが支配する
・上昇の終点はバルコニー後方通路

ホール階段は 静けさの層。

4. 上層：バルコニー（黒層の最深部）
・ホール階段の最上段と水平に接続
・黒層の最深部
・白金の手すりが半円状に広がる
・音の中心へ向かう最終領域
・縦軸の空気が最も美しく流れる場所

バルコニーは 音の中心の層。

5. 三層の立体関係（文章で崩れない図）
下層：ロビー階段（白層）  
生命感・動き・猫の儀式
→ 白層の上昇

中層：ホール階段（黒層）  
静けさ・集中・儀式の線
→ 黒層の上昇

上層：バルコニー（黒層の最深部）  
音の中心・縦軸の終点
→ 黒層の最深部の水平

この三層が “縦軸の儀式” を成立させている。

6. 三層の意味（三層視点の総括）

ロビー階段（下層）
　白層・生命感・動き・猫の儀式  
　→ 白層の上昇として、世界の入口を象徴する。

ホール階段（中層）
　黒層・静けさ・集中・儀式の線  
　→ 黒層の上昇として、音の中心へ向かう“沈み込み”を担う。

バルコニー（上層）
　黒層の最深部・音の中心・純粋な終点  
　→ 黒層の水平として、縦軸の儀式が完結する場所。

この三層は  
「白から黒へ沈み込み、最後に水平へと抜ける」  
という GRUMA の縦軸の構造性を最も純粋に表している。

---

7. 三層構造の意義（世界観上の役割）

・ロビー階段は “生命感の入口”  
・ホール階段は “静けさの通路”  
・バルコニーは “音の中心の到達点”

三層はそれぞれ独立した意味を持ちながら、  
縦軸として一つの儀式を形成している。

白層 → 黒層 → 黒層最深部  
という流れは、  
GRUMA の世界が持つ  
「沈み込む上昇」「静けさの深化」「音の純化」  
を象徴する。
