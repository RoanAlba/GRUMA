
# ロビー階段〜ホール階段 光の流れ（文章版）.md
1. 全体の光の流れ（縦軸のまとめ）
ロビー1F（白層の柔らかい光）
→ ロビー階段（白層の光が動きに沿って上昇）
→ ロビー2F通路（光が落ち着き、影が増える）
→ ホール2F入口（光が一段暗くなる境界）
→ ホール階段（黒層の深い陰影）
→ バルコニー（黒層の最深部の光）

GRUMA の光は、
明 → 半明 → 暗 → 深暗  
という “沈み込む光の軸” を持っている。

2. ロビー階段の光（白層・動きの光）
・白層の柔らかい光
・滑り台と猫の動きで光が揺れる
・階段の黒大理石が光を反射し、生命感が強まる
・上昇しても光は明るいまま
・ロビー2F通路に近づくと光が少し落ち着く

ロビー階段は 動きのある明るい光。

3. ロビー2F通路の光（落ち着きの帯）
・階段の終点から光が落ち着き始める
・白層の光が弱まり、影が増える
・通路中央は “白と黒の中間の光”
・ホール側に近づくほど暗さが増す
・視界が静かになり、集中が始まる

ここは 光が変質する帯。

4. ホール2F入口の光（境界の光）
・黒木の扉が光を吸う
・白金の枠がわずかに光を返す
・影が濃くなり、空気の密度が上がる
・音と同時に光も静まる
・扉の向こうは完全に黒層の光

入口は 光が “暗さ” に切り替わる結界。

5. ホール階段の光（黒層・静けさの光）
・黒層の深い陰影
・階段の黒大理石が光を吸収する
・白金の手すりだけが細く光る
・外側の余白はほぼ光を持たない
・上昇するほど光は深く沈む

ホール階段は 静けさの光。

6. バルコニーの光（黒層の最深部）
・黒層の最も深い光
・白金の手すりが半円状に光を縁取る
・音の中心へ向かうための “儀式の暗さ”
・縦軸の光がここで収束する

バルコニーは 光の終点＝深暗の中心。

7. 光の流れのまとめ（文章で崩れない図）
明（ロビー1F）
→ 明の上昇（ロビー階段）
→ 半明（ロビー2F通路）
→ 暗への切り替え（ホール2F入口）
→ 深暗の上昇（ホール階段）
→ 深暗の中心（バルコニー）

この光の流れが、
GRUMA の “沈み込む世界観” を成立させている。
