# ロビー〜ホール動線総まとめ.md（構造レイヤー完成版）
1. 全体構造の概要
GRUMA の動線は、
ロビー → ロビー階段 → ロビー2F → ホール2F入口 → ホール階段 → バルコニー  
という “二段階構造” で成立している。

ロビー階段とホール階段は役割が異なるため、
物理的にも世界観的にも分離されている。

2. ロビー階段（滑り台あり）
位置：ロビー左側

役割：ロビー1F → ロビー2F

構造：階段＋滑り台（猫の動線）

手すり：低い白金の線（75〜80cm）

滑り台：手すりより少し高い縁（85〜90cm）

終点：ロビー2F通路

ロビー階段は “動き・生命感” を象徴し、
猫の儀式的な動線が存在する唯一の階段。

3. ロビー2F通路
ロビー階段の終点

ホール2F入口へ続く水平動線

幅：1.2m

黒層の空気が薄く入り、ホール側の静けさへ移行する領域

ここが ロビー階段とホール階段の境界。

4. ホール2F入口
ロビー2F通路の先にある

ホール内部の空気に切り替わる “結界” のような位置

ここから先は滑り台などの動的構造は存在しない

儀式性・静けさ・集中の領域に入る

5. ホール階段（滑り台なし）
位置：ホール1F後方

役割：ホール1F → バルコニー

構造：黒層の静かな階段

手すり：白金の儀式線（90cm）

終点：バルコニー後方通路

ホール階段は “音の中心へ向かう儀式” を象徴し、
静けさを守るため滑り台は存在しない。

6. バルコニー後方通路
ホール階段の終点

バルコニー席を一周するリング状の通路

白金の手すりが半円状に広がる

ホール内部の縦軸の空気が最も美しく流れる領域

7. 動線のまとめ（視覚的に理解しやすい形）
【ロビー1F】
      │
      ▼
ロビー階段（滑り台あり）
      │
      ▼
【ロビー2F通路】
      │
      ▼
ホール2F入口（境界）
      │
      ▼
ホール階段（滑り台なし）
      │
      ▼
【バルコニー後方通路】

ロビー階段とホール階段は 直接繋がらない

ロビー2F通路が “橋渡し” の役割

滑り台はロビー側にのみ存在

ホール内部は静けさを守るため階段のみ

# ロビー〜ホール動線総まとめ.md — まとめ（要点）
ロビー → ロビー階段 → ロビー2F通路 → ホール2F入口 → ホール内部  
　という動線は“迷いゼロ”の一本線で構成される

ロビーは明るく開放、階段は縦の静けさ、通路は水平の静けさ、入口は導入の暗さ  
　という 光と音の段階変化 が体験の軸になる

動線は水平（ロビー2F通路）と垂直（ロビー階段・ホール階段）が立体的に分離  
　し、視界の衝突が起きない

ホール2F入口で視界が一度絞られ、ホール内部で一気に開ける  
　という“劇場的な開き”が動線のクライマックス

全体の目的は、ロビーの賑わいからホールの静寂へ
　“段階的に空気を切り替える” こと
