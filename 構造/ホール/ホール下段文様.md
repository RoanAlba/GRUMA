# ホール下段文様.md（下段装飾帯の文様）

1. 下段の基本理念
下段は “消えていく帯” として扱う

装飾は最も軽く、密度は上段・中段より低い

黒い影を作らず、影は“薄い銀影”

中段の流れを床へ自然に落とす

空気の透明度を守るため、余白を多く残す

2. 構成要素
■ A：細い蔦（E2 の終端）
中段から降りてきた蔦が、下段でさらに細くなる

彫りは極浅く、影はほとんど線として見える

床に向かって“消えていく”ようにデザイン

下段の蔦は中段よりも本数が少ない

■ B：小さなダイヤ（B1）
点光は最小限

空気を濁らせないため、配置は控えめ

光が当たると、床との境界が淡くきらめく

中段 → 下段で密度が自然に減るように配置

■ C：白い余白（空気の層）
下段は余白が最も多い

文様よりも“空気の透明度”が主役

余白が床の静けさを守る

3. 流れ（D1）
中段の水平の流れが、下段で“細く弱く”なる

下段はホール全体の流れの終点

流れの強い部分：ステージ側の正面

流れの弱い部分：左右の壁

流れが床へ自然に落ちることで、空気の層が安定する

4. 彫りと影
彫りはホール全体で最も浅い

影は“薄い銀影”として床に落ちる

黒い影は絶対に作らない

ダイヤの点光が影を分断し、空気を澄ませる

床の素材（薄い灰白石）と相性が良い

5. 床との接続
下段の蔦は床に触れる直前で“消える”

床の影と自然に溶け合う

下段 → 床の境界は曖昧で、空気の層として見える

床の反射が弱いため、文様が強く出ない

6. 観客からの見え方
■ 1F
下段は“ほとんど見えない薄い帯”

床との境界が柔らかく、空気が澄んで見える

中段 → 下段 → 床の流れが自然

■ 2F
下段は細い影の帯として見える

文様は強く見えず、空気の透明度が優先される

床の静けさが際立つ

■ B1
下段はほぼ見えず、床の光だけが感じられる

空気の層が静かに広がる
