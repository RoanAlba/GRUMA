# ロビー階段図（文章版） — 初期ドラフト
● ロビー階段の基本構造
ロビー階段は 1F ロビー中央から 2F ロビー通路へ上がる主要動線で、
ホール階段とは独立した構造を持つ。
段数・角度・幅はホール階段より緩やかで、
ロビー空間の開放感を保つために視界を遮らない設計。

● ホール階段との関係
ロビー階段とホール階段は “隣接しているが別構造”。
境界は壁・段差・天井ラインで明確に分離され、
動線が混ざらないように設計されている。

● 立体構造
ロビー階段は 1F → 2F の単純な縦軸ではなく、
ロビー吹き抜けの空間に沿って “外側へ張り出す形” で構成される。
そのため、階段下の空間はロビーの視界として利用される。

● 動線の特徴
ロビー階段は “ホールへ向かう動線” と “ロビー内移動” を分離する役割を持つ。
