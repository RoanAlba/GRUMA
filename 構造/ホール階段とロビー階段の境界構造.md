# ホール階段とロビー階段の境界構造.md
1. ロビー階段の終点
ロビー階段（滑り台を含む動線）は、
ロビー2F通路で終わる。

ここから先はホール内部には直接入らず、
ロビー2Fの通路を経由して
ホール2Fの入口へ向かう。

2. ホール階段の起点
ホール階段は、
ホール1F後方 から始まる独立した構造であり、
ロビー階段とは物理的に接続しない。

ホール内部の静けさと儀式性を守るため、
滑り台などの付帯構造は存在しない。

3. ロビー2F → ホール2F の動線
ロビー階段の終点（ロビー2F）

ロビー2F通路

ホール2F入口

ホール階段（バルコニーへ）

この順で移動するため、
ロビー階段とホール階段は
直接は繋がらず、通路を介して分離されている。

4. 滑り台の存在範囲
滑り台は ロビー階段の外側 にのみ存在し、
ホール内部には一切入らない。

ホール階段は黒層の静けさを保つため、
滑り台のような動的構造は排除されている。

5. 世界観上の意味
ロビー階段：動き・生命感・猫の儀式

ホール階段：静けさ・集中・音の中心へ向かう儀式

両者は役割が異なるため、
構造的にも世界観的にも分離されている
