# ロビー〜ホール 最終統合まとめ（文章版）.md
1. 全体の流れ（水平＋垂直の統合）
ロビー1F（白層）
→ ロビー階段（白層の上昇・滑り台あり）
→ ロビー2F通路（白→黒の移行帯）
→ ホール2F入口（結界）
→ ホール階段（黒層の上昇・滑り台なし）
→ バルコニー後方通路（黒層の最深部・音の中心）

GRUMA の動線は
白 → グレー → 黒 → 黒の最深部  
という “沈み込むような上昇” で構成されている。

2. ロビー階段（白層・生命感）
・階段（内側）
・手すり（低い白金）
・滑り台（外側）
・猫の動きが儀式として見える
・白層の光と音が広がる
・上昇の終点はロビー2F通路

ロビー階段は 生命感の層。

3. ロビー2F通路（白→黒の移行帯）
・幅1.2mの水平通路
・白層の光と音が弱まり始める
・空気が白→グレー→黒へ変化
・動きが止まり、静けさが始まる
・正面にホール2F入口が現れる

ここは 空気・光・音が変質する帯。

4. ホール2F入口（結界）
・黒木の扉
・白金の細い枠
・光が落ち着き、影が増える
・音が急に弱まる
・扉の向こうは完全に黒層
・右側にホール階段が現れる

入口は 白→黒の境界点。

5. ホール階段（黒層・静けさ）
・黒大理石の階段
・手すりは白金の儀式線（内側のみ）
・外側は黒層の余白
・滑り台は存在しない
・音が吸い込まれる
・光が深く沈む
・上昇の終点はバルコニー後方通路

ホール階段は 静けさの層。

6. バルコニー後方通路（黒層の最深部）
・黒層の最も濃い空気
・白金の手すりが半円状に広がる
・音の中心へ向かう最終領域
・縦軸の空気・光・音がここで収束する

バルコニーは 縦軸の終点＝音の中心。

7. ロビーとホールの違い（最終比較）
ロビー（白層）  
・生命感
・動き
・猫の儀式
・滑り台
・広がる光と音

ホール（黒層）  
・静けさ
・集中
・儀式の線
・余白
・沈む光と音

この対比が GRUMA の世界観を成立させている。

8. ロビー〜ホールの縦軸（最終まとめ）
白層（ロビー1F）
→ 白層の上昇（ロビー階段）
→ 白→黒の移行帯（ロビー2F通路）
→ 黒層への切り替え（ホール2F入口）
→ 黒層の上昇（ホール階段）
→ 黒層の最深部（バルコニー）

白から黒へ沈み込む上昇  
これが GRUMA の縦軸の正体。
