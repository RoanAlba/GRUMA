# GRUMA動線総合構造.md（最終統合版）
1. 全体概要
GRUMA の動線は、
ロビー（生命感） → ホール（静けさ） → バルコニー（音の中心）  
という三段階の空気の変化に合わせて設計されている。

動線は以下の順で進む：
ロビー1F
   ↓
ロビー階段（滑り台あり）
   ↓
ロビー2F通路（空気の境界帯）
   ↓
ホール2F入口（結界）
   ↓
ホール階段（滑り台なし）
   ↓
バルコニー後方通路
この流れが、
GRUMA の “儀式としての移動” を成立させている。
2. ロビー1F → ロビー階段（滑り台あり）
ロビーは白層の軽い空気

左側に階段

階段の外側に滑り台（猫の動線）

手すりは低い白金の線（75〜80cm）

滑り台は手すりより少し高い（85〜90cm）

猫の動きが視界に入る構造

ここは 生命感の領域。

3. ロビー階段 → ロビー2F通路
ロビー階段の最上段はロビー2F通路と段差なしで接続

滑り台はここで終わる

手すりもここで終わる

通路は水平で幅1.2m

空気は白層→黒層へ移行する “薄いグレーの帯”

ここは 空気の移行帯。

4. ロビー2F通路 → ホール2F入口（結界）
通路の突き当たりに黒木の扉

扉の枠は白金の細い線

扉の向こうは完全に黒層

光が落ち着き、影が増える

音が急に弱まる

滑り台や動的構造は一切入らない

ここは 儀式空間への入口。

5. ホール2F入口 → ホール階段（滑り台なし）
入口を通過すると空気が黒層に切り替わる

右側にホール階段

黒大理石の静かな階段

手すりは白金の儀式線（90cm）

滑り台は存在しない

音が吸い込まれるように静か

ここは 静けさの領域。

6. ホール階段 → バルコニー後方通路
階段最上段はバルコニー後方通路と水平に接続

バルコニー後方通路はホールを一周するリング

白金の手すりが半円状に広がる

縦軸の空気が最も美しく流れる場所

ここは 音の中心へ向かう最終領域。

7. 動線の視覚的まとめ（文章図）
【ロビー1F】
   白層の空気
   猫の滑り台がある領域
        │
        ▼
ロビー階段（滑り台あり）
   手すり＜滑り台の高さ
        │
        ▼
【ロビー2F通路】
   白→黒の空気の移行帯
        │
        ▼
ホール2F入口（結界）
   黒木の扉
   空気・光・音が切り替わる
        │
        ▼
ホール階段（滑り台なし）
   黒層の静けさ
   白金の儀式線
        │
        ▼
【バルコニー後方通路】
   音の中心へ向かう最終領域
8. 世界観上の意味
ロビー：生命感・動き・猫の儀式

通路：空気の移行

入口：結界

ホール：静けさ・集中

バルコニー：音の中心

この動線は、
GRUMA の世界観そのものを体験として成立させる構造。

# ✦ GRUMA動線総合構造.md — まとめ（要点）

動線は “ロビー → ホール → 回廊 → 各部屋” の一方向性で構成される

1Fと2Fは階段で分岐し、互いに干渉しない独立ルート

ホール入口は1Fと2Fで別動線になり、混雑が発生しない

回廊はホールと部屋をつなぐ“緩衝帯”として機能し、音と光を整える

ロビー階段は中央軸で分岐し、上層と下層の流れを自然に分ける

ステージ周辺は動線が交差しないように設計され、裏動線は存在しない

全体の動線は“中央軸・上下分岐・回廊緩衝”の三要素で成立する
