# ロビー階段〜ホール階段 音の流れ（文章版）.md
1. 全体の音の流れ（縦軸のまとめ）
ロビー1F（白層：音が広がる）
→ ロビー階段（白層：音が動きに沿って上昇）
→ ロビー2F通路（音が弱まり始める帯）
→ ホール2F入口（音が一段落ちる境界）
→ ホール階段（黒層：音が吸い込まれる）
→ バルコニー（黒層の最深部：音の中心）

GRUMA の音は、
広がる → 弱まる → 落ちる → 吸われる → 集中する  
という “沈む音の軸” を持っている。

2. ロビー階段の音（白層・動きの音）
・白層は音が軽く広がる
・人の足音も猫の動きも柔らかく響く
・滑り台の存在で “動きの音” が生まれる
・階段を上がるほど音は少しずつ落ち着く
・ロビー2F通路に近づくと音が弱まる

ロビー階段は 動きのある音。

3. ロビー2F通路の音（弱まりの帯）
・階段の終点から音が弱まり始める
・白層の響きが薄くなる
・通路中央は “音が半分だけ残る” 地帯
・ホール側に近づくほど音が落ちる
・静けさが始まる準備の帯

ここは 音が変質する帯。

4. ホール2F入口の音（境界の音）
・扉の手前で音が急に落ちる
・黒木の扉が音を吸う
・白金の枠がわずかに響きを返す
・扉の向こうは完全に黒層の静けさ
・空気と光と音が同時に切り替わる

入口は 音が “静けさ” に切り替わる結界。

5. ホール階段の音（黒層・静けさの音）
・黒層は音を吸収する
・足音がほとんど響かない
・白金の手すりが “儀式の線” として静けさを縁取る
・外側の余白が音の吸収を強める
・上昇するほど音は深く沈む

ホール階段は 静けさの音。

6. バルコニーの音（黒層の最深部）
・黒層の最も深い静けさ
・音の中心へ向かうための “儀式の静寂”
・縦軸の音がここで収束する
・舞台の音が最も美しく届く場所

バルコニーは 音の終点＝中心。

7. 音の流れのまとめ（文章で崩れない図）
広がる（ロビー1F）
→ 動く音（ロビー階段）
→ 弱まる音（ロビー2F通路）
→ 落ちる音（ホール2F入口）
→ 吸われる音（ホール階段）
→ 集中する音（バルコニー）

この音の流れが、
GRUMA の “沈み込む世界観” を完成させている。
