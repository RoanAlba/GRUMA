# 白孔雀.md（完成版）
1. 役割
白孔雀は、GRUMAホールにおける「神の使い」として配置される。
天井画が表す“神の空”を補完し、空間全体の神聖さを完成させる存在である。
主役ではなく、天井画を引き立てる補助的・象徴的役割を担う。

2. 数量（確定：6体）
白孔雀は合計6体とする。

四隅：4体

扇形ホールの左右中間：2体

扇形ホールの広がりを自然に満たし、
天井画の主役性を損なわず、空間のバランスを最も美しく保つ構成。

3. 大きさ（1.6m寄り）
白孔雀は存在感を持たせるため、1.6m寄りのサイズとする。

全高：1.5〜1.6m

羽根の最大長：60〜80cm

体の高さ：40〜50cm

全幅：70〜90cm

近距離での圧巻の精密さと、遠距離での静かな神聖さを両立する。

4. 羽根の構造とディテール
4-1. 曲線（自然で神聖）
羽根の流れは、直線でも過度な曲線でもなく、
“自然で神聖”な中間のカーブを採用する。
柔らかさを保ちながら、光の流れを導く形状とする。

4-2. 極細線（銀・プラチナ）
太さ：0.1〜0.2mm

役割：羽根内部の“光の骨格”

通常時は透明に溶け込み、光が当たった瞬間だけ細い閃光として浮かぶ

金属感が出ないよう、マット寄りの質感を採用する

柔らかさを損なわず、近距離での精密さを生むための最小限の使用とする。

4-3. 極小の光の粒
サイズ：1mm以下

材質：透明片・極小ダイヤ

配置：極細線の上に星座のように点在させる

効果：光の跳ね返りによる情報量の増加、近距離での圧倒的な精密感

4-4. 多層構造
羽根は以下の3層で構成する。

上層：極細線（気配の線）

中層：極小の光の粒（情報量）

下層：乳白ガラス（柔らかい影）

柔らかさを保ちながら、立体的で神聖な印象を生む。

4-5. 先端のしずく型
大きめの透明しずく型を採用

周囲に極小片を数粒配置し、“光の霧”を形成

降臨感を象徴する要素として機能する

5-2. 天井画との関係
白孔雀：点の光

天井画：面の世界

白孔雀は白と透明、天井画は青の深さ

役割・色・スケールが異なるため、
白孔雀が天井画を超えることはなく、
むしろ天井画の神聖さを補完する。
