2. ロビー階段（1F → 2F）
・ロビーの中心軸に位置する主要階段
・上昇方向は「ホールへ向かう儀式的な導線」
・空気層は白層が強く、開放感がある
・2Fに上がるとホール世界への境界が始まる

3. ロビー2F通路（水平移動）
・ロビー世界とホール世界をつなぐ水平の結界ライン
・空気層は白層から中間層へ移行
・ホール2F入口が世界の切り替わり点になる

4. ホール2F入口（境界）
・ロビー世界からホール世界へ入る最重要の境界点
・空気層が中間層から黒層へ変化
・音響・光・空気の密度が変わる
・ここから儀式空間が始まる

5. ホール階段（2F → バルコニー）
・ホール内部の縦軸を象徴する階段
・上昇するほど黒層が濃くなり、視界が舞台方向へ収束
・バルコニーは最も高い視点の観覧層

6. バルコニー（最上層）
・ホールの縦軸構造の終点
・黒層の最深部
・観客の視線が舞台へ最も美しく落ちる位置
・ロビー → ホール → バルコニーの儀式的上昇の完成点

# ロビー階段〜ホール階段統合図構造（決定版）.md — まとめ（要点）
ロビー階段は吹き抜けに開く“開放の縦軸”、ホール階段は内部へ沈む“閉じた縦軸”として対照的に構成される

境界は壁・段差・天井ラインで明確に分離され、動線が混ざらない

ロビー階段は光と視界が広がり、ホール階段は音と光が抑えられた静かな導線になる

両階段は吹き抜けを介して立体的に独立し、視界の衝突が起きない

2Fに上がった後の動線も分離され、ロビー側は通路へ、ホール側は客席へ自然に誘導される

“開放 → 境界 → 静けさ” の段階的な空間切り替えが統合図の主目的
