# ロビー2F通路＋ホール2F入口 統合図（文章版）.md

→ ホール2F入口（結界）
→ ホール階段（滑り台なし）

この4つが一直線に並ぶ “水平の儀式動線”。

2. ロビー2F通路の構造（文章図）
ロビー階段の最上段
→ 水平の通路（幅1.2m）
→ 白層 → 薄いグレー → 黒層へと空気が変化
→ 正面に黒木の扉（ホール2F入口）

3. ホール2F入口の構造（文章図）
ロビー2F通路の突き当たり
→ 黒木の扉（枠は白金の細い線）
→ 扉の向こうは完全に黒層
→ 光が落ち着き、影が増える
→ 音が急に弱まる
→ 右側にホール階段が現れる

入口は 空気・光・音が切り替わる結界。

4. 空気の変化（境界の帯）
ロビー側：白層
通路中央：白＋黒が混ざる薄いグレー
入口：黒層が強くなる
ホール側：完全な黒層

# ロビー2F通路＋ホール2F入口統合図構造.md — まとめ（要点）
ロビー2F通路は明るい水平動線、ホール2F入口は暗く静かな縦開口として対照的に構成される

境界は“角度・天井ライン・光の落差”で分離され、空間の性質が切り替わる

ロビー側は吹き抜けに開き、ホール入口側は天井が低く視界が一度絞られる

通路から入口までは完全な直線動線で、迷いが生まれない

入口内部は光が抑えられ、ホールへ入る瞬間に視界が一気に開く構造

音はロビー側で拡散し、入口で一度吸われ、ホール内部で再び広がる

統合図の目的は“明るい水平動線 → 静かな導入 → ホールの開放”という三段階の体験を成立させること

このグラデーションが
“儀式空間へ入る” 感覚を自然に作る。
