# ロビー2F通路＋ホール2F入口 統合図（文章版）.md
1. 全体の流れ（水平動線）
ロビー階段（滑り台あり）
→ ロビー2F通路（白→黒の移行帯）
→ ホール2F入口（結界）
→ ホール階段（滑り台なし）

この4つが一直線に並ぶ “水平の儀式動線”。

2. ロビー2F通路の構造（文章図）
ロビー階段の最上段
→ 水平の通路（幅1.2m）
→ 白層 → 薄いグレー → 黒層へと空気が変化
→ 正面に黒木の扉（ホール2F入口）

通路は 段差なし・水平・余白多め の構造。

3. ホール2F入口の構造（文章図）
ロビー2F通路の突き当たり
→ 黒木の扉（枠は白金の細い線）
→ 扉の向こうは完全に黒層
→ 光が落ち着き、影が増える
→ 音が急に弱まる
→ 右側にホール階段が現れる

入口は 空気・光・音が切り替わる結界。

4. 空気の変化（境界の帯）
ロビー側：白層
通路中央：白＋黒が混ざる薄いグレー
入口：黒層が強くなる
ホール側：完全な黒層

このグラデーションが
“儀式空間へ入る” 感覚を自然に作る。

5. 横方向の構造（文章で崩れない図）
ロビー2F通路
＝ 白層 → グレー → 黒層の帯
＝ 余白の多い水平動線
＝ 滑り台は入らない

ホール2F入口
＝ 黒木の扉
＝ 白金の細い枠
＝ 黒層の空気が支配
＝ 静けさの始まり

6. 世界観上の意味
ロビー2F通路：
・白層から黒層へ移行する “準備の帯”
・動きが止まり、静けさが始まる場所

ホール2F入口：
・空気・光・音が切り替わる “結界”
・儀式空間への正式な入口

この2つが連続することで、
GRUMA の世界は “濁りゼロ” のまま繋がる。
