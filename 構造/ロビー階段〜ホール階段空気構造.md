# ロビー階段〜ホール階段 空気の流れ（文章版）.md
1. 全体の空気の流れ（縦軸のまとめ）
ロビー1F（白層）
→ ロビー階段（白層の上昇）
→ ロビー2F通路（白→黒の移行帯）
→ ホール2F入口（黒層へ切り替わる結界）
→ ホール階段（黒層の上昇）
→ バルコニー（黒層の最深部・音の中心）

GRUMA の空気は、
白 → グレー → 黒 → 黒の最深部  
という “沈み込むような流れ” を持っている。

2. ロビー階段の空気（白層・生命感）
・白層の軽い空気
・猫の動きが空気を揺らす
・滑り台の存在が生命感を強調
・上昇しても空気は白層のまま
・階段の終点で空気が変わり始める

ロビー階段は 白層の動きの空気。

3. ロビー2F通路の空気（白→黒の移行帯）
・階段の最上段から空気が変わり始める
・白層が薄まり、グレーが混ざる
・通路の中央で “白と黒の中間” の空気
・ホール側に近づくほど黒層が強くなる
・動きが止まり、静けさが始まる

ここは 空気が変質する帯。

4. ホール2F入口の空気（結界）
・扉の手前で空気が重くなる
・光が落ち着き、影が増える
・音が急に弱まる
・扉の向こうは完全に黒層
・空気の密度が一段階上がる

入口は 空気・光・音が同時に切り替わる境界点。

5. ホール階段の空気（黒層・静けさ）
・黒層の静かな空気
・音が吸い込まれるように消える
・白金の手すりが “儀式の線” として浮かぶ
・外側の余白が空気の静けさを強調
・上昇しても空気は黒層のまま濃くなる

ホール階段は 黒層の静けさの空気。

6. バルコニーの空気（黒層の最深部）
・黒層の最も濃い空気
・音の中心へ向かうための静けさ
・縦軸の空気がここで収束する
・白金の手すりが空気の流れを縁取る

バルコニーは 空気の終点＝音の中心。

7. 空気の流れのまとめ（文章で崩れない図）
白層（ロビー1F）
→ 白層の上昇（ロビー階段）
→ 白→黒の移行帯（ロビー2F通路）
→ 黒層への切り替え（ホール2F入口）
→ 黒層の上昇（ホール階段）
→ 黒層の最深部（バルコニー）

この空気の流れが、
GRUMA の “儀式としての移動” を成立させている。
