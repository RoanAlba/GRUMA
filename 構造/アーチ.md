# ✦ 構造／アーチ.md（最終仕様）
◆ 1. アーチの本体（黒アイアン・固定）
アーチ本体は 黒アイアン。
これは世界の“線”をつくる最重要要素で、変更しない。

色：深い黒（マット寄り）

反射：弱い

太さ：細い（重くならない）

存在感：光が弱い時に最も美しい

外が明るい時でも輪郭が消えない

→ 白金アーチにはしない。
　黒が世界の基準であり、やっちゃんの好みに完全一致。

◆ 2. 白金の役割（縁のハイライトのみ）
白金は“アーチ本体の色”ではなく、
縁にだけ入る極細の光のライン。

本体は黒

縁だけ白金の細い光

光が当たった時だけ浮く

外が明るい時はほぼ見えない（自然）

夜景では白金が静かに光る

→ 白金は“輪郭補助”。
　黒アイアンを引き立てるためだけに存在する。

◆ 3. アーチの形
形：縦長の丸アーチ（比率 1 : 1.25）

厚み：14cm

曲線：やや深めのカーブ

影：床側に落ちる（壁には落とさない）

→ 黒大理石の壁でも沈まないように設計。

◆ 4. 壁との相性（素材は部屋側で変更可能）
アーチは黒アイアンなので、
壁が黒でも白でも成立する。

● 初期値
黒大理石（弱反射）

● 部屋側で選べる素材
白大理石

灰を含まない白石

深木（ダークウッド）

金属壁（サテン）

マット黒石

→ アーチは黒なので、どの素材でも相性が良い。

◆ 5. アーチと窓の関係
アーチは“外へ出る境界”。
窓は“外を見る境界”。

アーチ：黒アイアン

窓：ガラスが“雨の時だけ出現”

どちらも外の景色と連動する

→ 黒アイアンのアーチが、窓の透明感を引き締める。

◆ 6. 光の処理
アーチが沈まないように、光の扱いは重要。

白金の縁が光を拾う

黒アイアンは光を吸う

壁の反射は弱め

影は床に落とす

→ 黒 × 白金 × 外光 の三角形で成立する。

◆ 7. 雨の時の特別仕様
窓ガラスが“雨の時だけ出現”するため、
アーチの見え方も変わる。

ガラスに雨粒の揺らぎ

アーチの黒が濡れたように深く見える

白金の縁が雨光を拾う

→ 雨の時が一番ドラマチック。

✦ アーチ.md — まとめ（要点）

アーチは“室内と外の境界”をつくる黒アイアンのフレーム

厚みは14cmで、境界の“深さ”と影の落ち方を決める

床とは段差なく接続し、外側はそのままバルコニーへ連続する

光はアーチ上部から入り、影は床側に細く落ちる（壁には落とさない）

縁の白金ラインは必要な時だけ微光を帯び、夜の境界を示す

装飾ではなく“構造”として存在し、空間の軸と雰囲気を決定する
