# ロビー階段 vs ホール階段（比較図・文章版）
1. 目的の違い
ロビー階段：生命感・動き・猫の儀式

ホール階段：静けさ・集中・音の中心へ向かう儀式

2. 構造の違い（横並びで比較）
ロビー階段（滑り台あり）  
階段（内側） → 手すり（低い白金） → 滑り台（外側）

ホール階段（滑り台なし）  
客席側（内側） → 手すり（白金） → 黒層の余白（外側）

3. 高さの違い（文章でズレない表現）
ロビー階段  
手すり：80cm
滑り台の縁：90cm
→ 滑り台の方が高い（猫が見える）

ホール階段  
手すり：90cm
外側：余白のみ
→ 静けさを守るため “何も置かない”

4. 空気の違い
ロビー階段  
白層の空気の中で動きがある
猫の滑りが儀式的に見える

ホール階段  
黒層の空気の中で静けさが支配
白金の手すりが儀式の線として浮かぶ

5. 終点の違い
ロビー階段  
ロビー2F通路（白→黒の移行帯）で終わる
滑り台もここで終了

ホール階段  
バルコニー後方通路（黒層の最深部）に繋がる
静けさの頂点へ向かう動線

6. 世界観上の意味
ロビー階段  
・動き
・生命感
・猫の儀式
・白層の空気

ホール階段  
・静けさ
・集中
・音の中心へ向かう儀式
・黒層の空気
