# ホールステージ：世界観（完全版）
■ 1. ステージの役割
ホールのステージは、
“天井画の神聖さを受け止めるための静かな台座” として存在する。
主張せず、沈黙し、空気の透明さを保つことが最優先。
ステージそのものが目立つのではなく、
天井画 → 空気 → 光 → ルカ  
という流れを邪魔しない構造である。

■ 2. ステージの色と素材
色：白に近い無彩色

質感：マット（反射を抑える）

素材：白木または白に近い木材

目的：天井画の光を柔らかく受け、空気の透明感を壊さない

“白を濁らせない白” が基準。
青白くも黄白くもならない、完全中立の白。

■ 3. ステージの高さ・幅
高さ：低め（存在感を出さない）

幅：ホールの中央に静かに広がる

段差：最小限

目的：ルカが“浮いているように見える”ための高さバランス

ステージは“舞台”ではなく
“静かな床の延長” として扱う。

■ 4. 光の構成
ステージの光は、
ルカと天井画の両方を壊さない“透明の光” が基準。

メイン光：右前45度（無色）

補助光：左後ろ弱め（影を柔らかくするだけ）

空気の色：完全無色

色温度：5000〜5500K（白でも青でもない中立光）

光は“照らす”のではなく
“存在を浮かび上がらせる” ために使う。

■ 5. 空気の世界観
ホールの空気は
“色を持たない透明” が絶対条件。

白い空気 → 毛が黄ばむ

青白い空気 → 世界が冷たくなる

無色 → 透明世界が最大化される

空気は存在しないように見えるほど薄く、
光だけが静かに流れる。

■ 6. ハープの位置
ステージ左奥

ルカの右側

角度：正面に対して少し斜め

光：金属光が右前45度で自然に光る位置

ハープは“主役”ではなく
“神聖な空気を支える柱” として存在する。

■ 7. ルカとの距離感（最重要）
横から見たときの距離は
ハープから20〜30cm。

理由：

毛がつぶれない

宝石が“点”で光る

ハープの形が隠れない

ルカのS字シルエットが崩れない

空気の層が生まれる

浮遊感が最大化される

“近すぎず、離れすぎず、重ならず、浮く”  
これがホールのルカの距離感。
