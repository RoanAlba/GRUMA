# ロビー2F通路〜ホール階段 立体関係（文章版）.md
1. 全体の立体構造（上から見た関係）
ロビー階段（上昇）
→ ロビー2F通路（水平）
→ ホール2F入口（境界）
→ ホール階段（上昇）
→ バルコニー後方通路（水平）

つまり、
上昇 → 水平 → 境界 → 上昇 → 水平  
という “立体のリズム” で構成されている。

2. ロビー2F通路（水平の帯）
・幅1.2m
・白層 → グレー → 黒層へ空気が変化
・動きが止まり、静けさが始まる
・滑り台はここで終了
・正面に黒木の扉（ホール2F入口）

通路は 水平のまま空気が変わる帯。

3. ホール2F入口（立体の境界点）
・通路の突き当たり
・黒木の扉
・白金の細い枠
・光が落ち着き、影が増える
・音が急に弱まる
・扉の向こうは完全に黒層
・右側にホール階段が現れる

入口は 水平→垂直の切り替え点 であり、
同時に 白→黒の空気の切り替え点。

4. ホール階段（垂直の帯）
・黒層の空気の中を上昇
・階段は黒大理石
・手すりは白金の儀式線（内側のみ）
・外側は黒層の余白
・静けさが支配する
・滑り台は存在しない

ホール階段は 黒層の垂直軸。

5. 立体的なつながり（文章で崩れない図）
ロビー階段（上昇）  
白層の動き →
ロビー2F通路（水平）  
白→黒の移行帯 →
ホール2F入口（境界）  
空気・光・音が切り替わる →
ホール階段（上昇）  
黒層の静けさ →
バルコニー（水平）  
黒層の最深部・音の中心

この “上昇→水平→上昇” の立体構造が
GRUMA の儀式性を強くしている。

6. 世界観上の意味（立体視点）
・ロビー階段：生命感の上昇
・ロビー2F通路：空気の移行
・ホール2F入口：結界
・ホール階段：静けさの上昇
・バルコニー：音の中心

立体的に見ると、
GRUMA の世界は 白層→黒層へ沈んでいくような構造 になっている。

# ロビー2F通路〜ホール階段立体構造.md — まとめ（要点）
ロビー2F通路は“水平軸”、ホール階段は“垂直軸”として独立した構造を持つ

両者は吹き抜け空間を介して立体的に接続され、視界が重ならない

ロビー2F通路は明るく開放、ホール階段は静かで閉じた縦動線として対照的

境界は壁・天井ライン・段差で明確に分離され、動線が混ざらない

ロビー側から見ると階段は“奥へ沈む縦の穴”として見え、視覚的に干渉しない

ホール階段を上がると、ロビー2F通路とは別方向へ自然に誘導される

立体構造の目的は“水平動線と垂直動線の完全分離”と“視界の衝突回避”にある
