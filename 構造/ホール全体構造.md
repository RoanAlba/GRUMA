# ホール全体構造（B1・1F・2F）
このホールは 三層構造で成り立つ。
中心となるのは B1（地下1階） に広がる“神域”であり、
1F・2Fはその神域を 見下ろすための階層として存在する。

■ B1：ホール床面（神域）
● 役割
B1は ホールの中心であり、神域そのもの。
天井画の光が落ち、透明の空気が満ち、
ルカとステージが静かに存在する場所。

● 特徴
出入口は置かない

生活動線を入れない

空気の純度を守る

天井画 → 空気 → ルカ → ステージ の縦軸が成立する

客席は空気に溶けるように配置される

● 理由
B1に出入口を作ると

空気が濁る

世界観が崩れる

上階とのつながりが死ぬ
ため、完全に閉じた神域として扱う。

■ 1F：メイン出入口（神殿に入る視点）
● 役割
1Fは ホールに入るための唯一の生活動線。
扉を開けた瞬間に、
“縦の神殿”が一気に視界に入る圧巻の眺めを作る。

● 扉を開けた瞬間に見えるもの
天井画の光

無色透明の空気の柱

その中に浮くルカ

ルカの右奥にハープ

静かなステージ

客席は空気に溶けて見えない

● 印象
「静けさの中に神が降りている」  
そんな迫力が生まれる。

● 出入口の位置
客席後方

ステージから直接見えない角度

白木の扉で存在感を消す

■ 2F：上層回廊（空中から神域を覗く視点）
● 役割
2Fは 神域を上から覗き込むための階層。
1Fとはまったく違う迫力が生まれる。

● 扉を開けた瞬間に見えるもの
天井画が近い

光が空気に落ちていく

その中に浮くルカ

下に広がる客席が影の海

ステージが遠く下に見える

● 印象
「空中から神域を覗き込むような視点」  
高さ × 透明 × 静けさ が合わさり、
鳥肌が立つような迫力が生まれる。

● 出入口の位置
1F出入口の真上

回廊の外周に配置

ホール内部に直接干渉しない角度

■ 三層構造の最終効果
B1 → 神域（出入口なし）

1F → 神殿に入る視点（圧巻の縦軸）

2F → 神域を上から覗く視点（空中の迫力）

この三層が揃うことで、
ホール全体は “縦の神殿” として完成する。

天井画 → 空気 → ルカ → ステージ
この縦の流れが最大化され、
どの階から見ても世界観が崩れない。
