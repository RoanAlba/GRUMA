# 回転塔

## 役割
- 工場外部で雷エネルギーを生成する中心塔
- 回転運動によって空気を圧縮し、稲妻を誘発する
- 発生した雷を給電装置へ送り、工場全体のエネルギー源となる
- 雷の強弱が工場内部の光や動作サイクルに影響を与える

## 外観

- 塔は細長い円柱形で、上部に向かってわずかに細くなる
- 表面はローズブラスと淡い銀金属の組み合わせで、光を柔らかく反射する
- 中央部に「回転リング」があり、ゆっくりと水平回転している
- 回転リングには細いガラス板が等間隔で取り付けられ、風を切るたびに淡い光が走る
- 塔の最上部には尖った導電針があり、雷を集めやすい構造になっている
- 全体として、静かだが緊張感のある“雷の塔”らしい佇まいを持つ

## 稲妻の発生の仕組み

- 回転塔の中央部にある回転リングが高速で空気を巻き込み、塔内部に圧力差を生む
- 圧力差によって塔の上部に微細な電荷が集まり、導電針へ向かって集中する
- 電荷が一定量を超えると、塔の最上部から稲妻が放出される
- 稲妻は塔の内部構造を通って一度安定化され、給電装置へ送られる
- 雷の強さは回転速度と空気の湿度によって変化し、工場内部の光柱にも影響を与える

## 周囲の環境

- 塔の周囲には常に薄い風が流れ、回転リングの動きに合わせて空気が揺れる
- 塔の根元付近には淡い霧が漂い、雷が強い時ほど濃くなる
- 回転リングが風を切るたびに、低い「フウ…」という音が響く
- 稲妻が発生する瞬間、塔の周囲に青白い光が一瞬だけ走る
- 塔の周囲は常に薄暗く、雷の光が唯一の明滅として存在している

## 工場との配置

- 回転塔は工場本体から少し離れた位置に建っている（およそ数十メートル）
- 塔と工場の間には細いガラス管が地面に沿って伸び、給電装置へ直接つながっている
- 工場側から見ると、塔は常に薄い霧の向こうにぼんやりと見える
- 雷が発生するたび、工場の壁面に青白い光が反射し、内部の光柱とわずかに同期する
- 工場と塔は物理的には離れているが、光と雷によって“ひとつの循環系”として結びついている
