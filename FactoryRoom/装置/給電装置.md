
# 給電装置

## 役割
工場全体にエネルギーを送る中心装置。

## 外観
- 細い縦型ガラス筒
- 内部に白い光柱が立ち上がる
- ローズブラスの縁取り

## 外観の詳細

- ガラス筒は細長く、透明度が高い
- ローズブラスの縁取りが上下にあり、光を柔らかく反射する
- 内部の光柱は白〜淡青で、細い糸のように立ち上がる
- 装置の根元には小さな基部があり、そこからガラス管へ接続される

## 光柱の動き

- 光柱は常に細く立ち上がっているが、雷の強さによって変化する
- 弱い雷：光柱は静かで、白に近い淡い光
- 中くらいの雷：光柱がわずかに揺れ、淡青が混じる
- 強い雷：光柱が脈打つように明滅し、青白い光が走る
- 光柱の揺れはガラス筒の内側に反射し、部屋全体の青い暗さを強める

## ガラス管との接続

- 給電装置の根元から、細いガラス管が横方向へ伸びている
- ガラス管の内部には、光柱から分岐した淡い光が流れ込む
- 光は一定の速度で流れ、雷が強いほど流速がわずかに上がる
- ガラス管は工場の壁へ向かって伸び、他の装置へエネルギーを分配する
- 接続部はローズブラスのリングで固定され、光が漏れないように密閉されている

## 動作サイクル

1. 装置内部の光柱がゆっくりと立ち上がる  
2. 雷の強さに応じて光柱が揺れ、色が変化する  
3. 光柱の一部が根元へ流れ込み、ガラス管へ分岐する  
4. ガラス管の内部を淡い光が一定速度で流れる  
5. 流れた光が工場内の各装置へエネルギーとして届く  
6. 全体の流れは常に循環しており、雷が強いほど循環が速くなる
