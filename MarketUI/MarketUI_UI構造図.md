# MarketUI — UI構造図（正式版）

MarketUI 全体の UI を「ページ単位」「構造単位」で整理した図。  
一覧 → 詳細 → 計器ページ の三本柱で構成される。

---

# ■ 1. MarketUI 全体構造（トップレベル）

MarketUI/
├─ 一覧ページ（List）
├─ 詳細ページ（Detail）
└─ 計器ページ（Meter / Analysis）

---

# ■ 2. 一覧ページ（List）

### ● 役割
- 作品を一覧で閲覧する入口  
- 市場的な UI（実用性重視）

### ● UI構造
- ヘッダー（固定）  
- 作品カード一覧（縦スクロール）  
- カード：左サムネイル × 右情報  
- 情報：タイトル / 価格 / タグ / 説明文（短）  
- 背景：白磁寄りの灰磁  
- 枠線：極細灰磁  
- 影：薄い  
- ホバー表示：禁止（情報は常に可視）

---

# ■ 3. 詳細ページ（Detail）

### ● 役割
- 作品の詳細情報を閲覧  
- 一覧より情報密度が高い

### ● UI構造
- 大きめプレビュー  
- タイトル  
- 説明文（長）  
- タグ  
- 価格  
- 関連作品（横スクロール）  
- 「計器ページへ」ボタン（下部固定）  
- 背景：白磁  
- 枠線：極細灰磁  
- 情報密度：中〜高

---

# ■ 4. 計器ページ（Meter / Analysis）

### ● 役割
- MarketFit 計器を中心とした分析ページ  
- MarketUI の中で唯一 “分析” を担う

### ● UI構造
- 計器エリア（大）  
　→ MarketFit 計器（丸）を中央に配置  
- 補助データエリア（中）  
　→ 閲覧数 / 傾向 / タグ分析  
- AIコメントエリア（小）  
　→ 市場性の解説  
- 背景：白磁寄りの灰磁  
- 遷移：フェード（0.25秒）

---

# ■ 5. ページ間遷移

一覧ページ  
　↓（タップ）  
詳細ページ  
　↓（分析ボタン）  
計器ページ  
　↓（戻る）  
詳細ページへ戻る

※ 計器ページ → 一覧ページへは戻らない（UXの一貫性）

---

# ■ 6. 禁止事項（MarketUI 全体）

- 真鍮UI  
- 光の演出  
- 過度なアニメーション  
- 情報の隠蔽（ホバー表示）  
- 展示室・出品部屋の世界観を持ち込むこと  
- カードの揺れ・回転

---

- MarketUI 全体の UI 構造を固定  
- 一覧・詳細・計器ページの関係を明確化  
- MarketUI_総合仕様 の下位
