# 商品カード（一覧） レイアウト図

## ■ 全体構成
- 左側：サムネイル（小）
- 右側：商品名 → 市場性％ → お気に入り数 → 遷移アイコン
- MarketUI の計器面（パステルブルー）に配置
- ステンレス枠・ガラス質感で統一

## ■ レイアウト図（ASCII）

┌──────────────────────────────┐
│  ┌──────────┐   商品名（短縮）              │
│  │ サムネイル │   ──────────────────────── │
│  │   （小）   │   市場性％（ミニ）           │
│  └──────────┘   ──────────────────────── │
│                     お気に入り数（ミニ）       │
│                     ──────────────────────── │
│                     [詳細] [計器] アイコン     │
└──────────────────────────────┘

## ■ 意図
- 一覧性を最優先した “最小構成のカード”
- 左に視線誘導のサムネイル、右に情報を縦に積む
- 詳細欄・計器ページのどちらにも行ける “入口カード”
