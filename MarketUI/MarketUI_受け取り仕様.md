# MarketUI — 出品部屋からの受け取り仕様（正式版）

このファイルは、出品部屋（ListingRoom）のゲートを通過した作品を  
MarketUI がどのように受け取り、どこに配置し、  
どの UI 状態で初期表示するかを定義する。

---

# ■ 1. 受け取りの基本ルール
- 出品部屋での「吸い込み完了」が MarketUI 受け取りの開始条件  
- 作品データは読み取り専用のまま渡される  
- ExhibitionRoom の空気は完全に切り離される  
- MarketUI は「市場の空気」に変換して受け取る  
  - 白磁光 → 市場光  
  - 静けさ → 情報密度  
  - 儀式性 → 実用性  

---

# ■ 2. 受け取り後の配置ルール
### ● カテゴリ別の初期位置
- 静止画 → Art / Still  
- 動画 → Motion  
- モックアップ → Mockup  
- 未来カード → Future  
- アンティークカード → Antique  

### ● 新規作品の初期位置
- 最上段・左側  
- スクロールなしで見える位置  
- 作品カードは MarketUI 標準カードに変換される

---

# ■ 3. 初期表示の UI 状態
### ● 作品カード
- サムネイル：出品部屋での最終フレーム  
- タイトル：出品部屋の情報を継承  
- 価格：MarketUI 側で自動生成 or 手動設定  
- 市場性％：MarketUI 側で初期値を付与  
- タグ：出品部屋のタグを継承  

### ● 初期アニメーション
- 0.2秒フェードイン  
- 拡大・縮小なし  
- 派手な演出は禁止

---

# ■ 4. 出品部屋との接続条件
### ● 出品部屋が渡す情報
- 作品データ  
- タイトル  
- 説明文  
- タグ  
- 出品形式  
- 注意書き  

### ● MarketUI が追加する情報
- 価格  
- 市場性％  
- カテゴリ位置  
- 商品カードUI  
- 並び順  

---

# ■ 5. 禁止事項（MarketUI 側）
- 出品部屋の UI を持ち込む  
- 儀式的な光・真鍮ラインライト  
- 作品の回転・揺れ  
- 展示室の静けさを再現しようとする  
- 透明台座の再現  

MarketUI は **完全に別世界** として扱う。

---

- MarketUI の初期表示ルールを明確化
