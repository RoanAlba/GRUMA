# RefineRoom UIアニメーション仕様
RefineRoom の UIアニメーションは、
「工房の静けさ」「精密な操作」「儀式の重み」 を壊さないための動きの基準。

目的は、
“動いているのに動いていないように見える”  
という世界観のアニメーションを統一すること。

1. アニメーションの基本原則（Principles）
動きは最小限

フェード中心（透明度の変化）

位置は基本的に動かさない

真鍮だけが光る（儀式具のみ）

0.1〜0.25秒の短い動き

バウンド禁止

音なし

工房の呼吸と同期する

2. プレビューのアニメーション（Preview）
● 呼吸（Breathing）
明るさ：+2% → 0%

時間：2.0〜2.5秒

ループ：ゆっくり

工房の生命感を表現

● 素材が変わった瞬間
明るさ：+3% → 0%（0.2秒）

バウンドなし

真鍮の光なし

音なし

● 図面モード切り替え
0.2秒フェード

スケール変化なし

光の変化なし

3. 素材スロットのアニメーション（Material Slot）
● 素材が置かれた瞬間
真鍮枠：+10%（0.2秒）

白磁：+3%（0.15秒）

プレビューが息を吸う（+2% → 0%）

● ホバー
影が1px濃くなるだけ

浮かない

4. 調整レバーのアニメーション（Adjustment Levers）
● ホバー
つまみ：+3% 明るくなる

影：1px → 2px

時間：0.12秒

● 操作開始
つまみ：+5% 明るくなる（0.1秒）

● 操作中
プレビューに即時反映

レバー自体はアニメーションしない（精密さ優先）

● 操作終了
明るさが元に戻る（0.15秒）

5. 保存レバーのアニメーション（Save Lever）
● 倒し切った瞬間（保存発動）
真鍮：+20%（0.25秒）

大理石：+3% → 0%（0.2秒）

プレビュー：+2%（0.2秒）

● 戻り
0.3秒でゆっくり戻る

バウンドなし

6. 出品レバーのアニメーション（Publish Lever）
● 倒し切った瞬間（出品発動）
真鍮：+25〜30%（0.3秒）

台座：0.2秒遅れて淡く光る

画面全体：0 → 25% 暗転（0.25秒）

プレビュー：深い呼吸（+4% → 0%）

● 戻り
0.4秒でゆっくり戻る

7. 一覧パネル連携時のアニメーション（List Panel Link）
● 読み込み時
画面が 0.15秒だけ暗くなる（20%）

プレビューが息を吸う

真鍮は光らない

● パネル閉じ
スケール：100% → 97%

透明度：100% → 0

時間：0.15秒

8. 図面モードのアニメーション（Blueprint Mode）
● 切り替え
0.2秒フェード

呼吸は弱く維持（+1% → 0%）

線図はアニメーションしない

9. 禁止事項（NG）
バウンド

スライド（横移動・縦移動）

回転

派手な光

色が急に変わる

長いアニメーション（0.3秒以上）

音

ポップアップの飛び出し

UIの縮小表示

レバーの自動アニメーション

→ RefineRoom の静けさと精密さを守るため。

10. 目的（Design Purpose）
RefineRoom のアニメーションは
「工房の空気を乱さず、静かに息づく動き」  
を作るために存在する。

呼吸

弱い光

短いフェード

精密な操作感

儀式具の重み

これらを統一することで、
RefineRoom は “静かに動く工房” として完成する。

✔ 最短まとめ
RefineRoom の動きは 静か・短い・精密

プレビューは呼吸

素材スロットは光＋呼吸

調整レバーは最小限

保存レバーは儀式的

出品レバーは最終儀式

図面モードはフェードのみ

RefineRoom フォルダに保存する
