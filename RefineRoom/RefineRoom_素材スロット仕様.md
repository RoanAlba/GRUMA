# RefineRoom 素材スロット仕様
素材スロットは、RefineRoom における 素材の受け皿・精製の入口。
目的は 素材を静かに置き、工房がそれを受け取る儀式 を UI として成立させること。

1. 素材スロットの役割（Role）
素材を 置く場所

素材カテゴリの 現在の選択状態を示す

精製の “入口” として機能する

UI System の項目とは切り離す（素材専用）

工房の静けさを壊さない “器” として存在する

2. 素材スロットの形状（Shape）
● 形
正方形に近い矩形

角丸なし

枠は 極細の真鍮ライン（0.8〜1.0mm）

● 素材
背景：白磁（青み 5%）

反射：弱い

影：ほぼなし（1%）

● サイズ
プレビューの左下に置ける程度

大きすぎず、小さすぎず

“素材を置く器” としての存在感

3. 素材スロットの配置（Placement）
中央プレビューの左下

素材パネル（左）との距離は近すぎない

調整レバー（右）とは完全に分離

レバー類（右下）とは干渉しない

視線の流れ：

素材パネル → 素材スロット → プレビュー

4. 素材スロットの状態（States）
● ① 空（Empty）
白磁のまま

真鍮枠は弱い光（+0〜2%）

呼吸なし

ホバー：影が1px濃くなるだけ

● ② 素材が置かれている（Filled）
素材のアイコン（小さめ）

真鍮枠が +5% 明るくなる

呼吸アニメーション（+2% → 0%）

プレビューと連動して変化

● ③ 無効（Disabled）
完了カードを読み込んだ場合など

白磁が少し暗くなる（-5%）

真鍮枠は光らない

ホバー反応なし

5. 素材の置き方（Interaction）
● 操作
ドラッグ＆ドロップ ではない

素材パネルで素材を選ぶと、
自動で素材スロットに置かれる

● 理由
ドラッグ操作は世界観に合わない

“置く” という儀式は 選択 で成立する

6. 素材スロットのアニメーション（Animation）
● 素材が置かれた瞬間
真鍮枠が +10% 光る（0.2秒）

白磁が +3% 明るくなる（0.15秒）

プレビューが息を吸う（+2% → 0%）

● 素材が変わった瞬間
光は保存レバーより弱い

呼吸は一度だけ

音なし

バウンドなし

7. 素材スロットとプレビューの連動（Link）
素材スロットは プレビューの入口 なので、
素材が置かれると以下が即時反映される。

プレビューの素材が変わる

光・質感の初期値が変わる

調整レバーの値が素材に合わせてリセットされる

呼吸アニメーションが一度だけ起きる

8. 禁止事項（NG）
ドラッグ＆ドロップ

素材スロットを大きくする

素材スロットを複数作る

素材スロットに説明文を入れる

派手な光

音

ポップアップ

素材スロットを折りたたむ

素材スロットを右側に置く

→ RefineRoom の静けさと儀式性を守るため。

9. 目的（Design Purpose）
素材スロットは
「素材を工房に捧げるための静かな器」  
として存在する。

真鍮の枠

白磁の器

呼吸

静かな光

これらによって、
素材を置く行為が 儀式として成立 する。

✔ 最短まとめ
素材スロットは 素材の入口

プレビュー左下に配置

素材は選択で自動的に置かれる

真鍮枠が光り、白磁が息をする

ドラッグ操作は禁止

RefineRoom フォルダに保存する
