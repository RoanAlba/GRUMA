# RefineRoom 調整レバー仕様
調整レバーは、RefineRoom における 光・質感・数値の微調整を行う操作専用UI。
目的は 素材の美しさを壊さず、静かに調整できる“工房の道具” を成立させること。

1. 調整レバーの役割（Role）
素材の 光量・質感・数値 を調整する

プレビューに 即時反映 する

UI System のアンティーク思想とは接続しない（操作専用）

誤操作を防ぎ、静かに調整できるようにする

保存・出品レバーとは“格”を分ける

2. レバーの形状（Shape）
● 形
縦スライダー

角丸なし

直線的で精密な形状

宝石は使わない（アンティーク思想と切り離す）

● 素材
レール：白金属（マット）

つまみ：陶器（白磁）

影：ほぼなし

反射：弱い

● サイズ
高さ：右パネルの 70〜80%

幅：細め（視線を奪わない）

つまみは小さめ（丸ではなく“短い棒”）

3. レバーの配置（Placement）

右パネル（調整エリア）
┌───────────────┐
│  光量レバー         │
│  質感レバー         │
│  数値レバー         │
│                     │
│  （下）保存レバー   │
│  （下）出品レバー   │
└───────────────┘

右パネルの上から順に並べる

レバー同士の間隔は広め（20〜28px）

保存・出品レバーとは明確に区切る

プレビューと干渉しない位置

4. レバーの状態（States）
● ① 通常（Normal）
レール：白金属

つまみ：白磁

影：1px

光：なし

● ② ホバー（Hover）
影が +1px

つまみが +3% 明るくなる

浮かない（浮くのは UIカードだけ）

● ③ 操作中（Active）
つまみが +5% 明るくなる

レールが +3% 明るくなる

プレビューが即時反応

呼吸アニメーションは維持

● ④ 無効（Disabled）
完了カード読み込み時

つまみが -10% 暗くなる

ホバー反応なし

操作不可

5. レバーのアニメーション（Animation）
● ホバー
影：1px → 2px

明るさ：+3%

時間：0.12秒

イージング：ease-out

● 操作開始
つまみが +5% 明るくなる（0.1秒）

● 操作中
プレビューに即時反映

レバー自体はアニメーションしない（精密さを優先）

● 操作終了
明るさが元に戻る（0.15秒）

6. レバーとプレビューの連動（Link）
調整レバーは プレビューと完全同期 する。

光量 → プレビューの明るさが変わる

質感 → 表面の反射・粗さが変わる

数値 → 素材の内部パラメータが変わる

呼吸アニメーションは維持

保存時はプレビューが +2% 明るくなる

出品時は呼吸が一度だけ深くなる

7. レバーの種類（Types）
● 光量レバー
明るさ

ハイライト

反射量

● 質感レバー
粗さ

滑らかさ

表面の深み

● 数値レバー
素材固有のパラメータ

0〜100 の範囲

精密な調整が可能

8. 禁止事項（NG）
宝石つまみ（アンティーク思想と混ざる）

派手な光

バウンド

横スライダー

角丸

レバーを太くする

レバーをポップアップ化する

レバーに説明文を詰め込む

レバーを中央に置く

レバーを自動で動かす

→ 調整レバーは “操作専用の精密な道具” であり、儀式具ではない。

9. 目的（Design Purpose）
調整レバーは
「素材の美しさを壊さず、静かに精密に調整するための道具」  
として存在する。

白金属のレール

白磁のつまみ

小さな影

即時反映

静かな操作感

これらによって、
工房の世界観を壊さずに 精密な調整作業 が成立する。

✔ 最短まとめ
調整レバーは 操作専用の精密UI

白金属レール＋白磁つまみ

縦スライダー

プレビューと完全同期

ホバーは最小限

儀式具（保存・出品）とは格を分ける

RefineRoom フォルダに保存する
