# 透明ディスプレイUI：全体レイアウト（正式版 / 保存用）

## 1. 概要
本仕様書は、透明ディスプレイに表示される  
「創作ゾーンUI」「判断レーンUI」「出品準備レーンUI」  
の3画面を統合した **全体レイアウト構造** を定義する。

透明ディスプレイは工房（RefineRoom）の中心装置であり、  
UIは“浮遊する透明レイヤー”として表示される。

---

## 2. 全体構造（3レーン）

+--------------------------------------------------------------+
| 【創作ゾーンUI】（揺れの可視化）                            |
| ・棒グラフ（透明）                                           |
| ・総評（短文）                                               |
| ・改善ポイント（最大3つ）                                   |
+--------------------------------------------------------------+
| 【判断レーンUI】（用途・サイズ・改善案）                     |
| ・用途ジャンル                                               |
| ・サイズ                                                     |
| ・改善案（真鍮ラベル）                                       |
+--------------------------------------------------------------+
| 【出品準備レーンUI】（出品前の最終確認）                     |
| ・タイトル                                                   |
| ・タグ                                                       |
| ・カテゴリ                                                   |
| ・価格                                                       |
+--------------------------------------------------------------+

---

## 3. レーン間の関係性

### 3-1. 創作ゾーン → 判断レーン
- 揺れの分析が終わると自動で判断レーンへ  
- 遷移は **0.25秒フェード**  
- 背景は透明のまま

### 3-2. 判断レーン → 出品準備レーン
- 改善案を確認後、右下の真鍮ボタンで遷移  
- 遷移は **0.25秒フェード**  
- UIの位置は変わらない（透明レイヤーのまま）

### 3-3. 出品準備レーン → 完了
- 出品ボタン（真鍮）で完了  
- 工房の中央に “出品完了の白い光” がふわっと灯る  
- アニメは0.3秒以内の静かな演出

---

## 4. 透明ディスプレイUIの共通ルール

### 4-1. 透明性
- UI背景は完全透明  
- 工房の光・灰磁・真鍮が透けて見える  
- UIは“浮いている”ように見える

### 4-2. フォント
- 細め  
- 白磁系の白（#f7f6f4）  
- 影はつけない

### 4-3. ボタン（共通）
- 真鍮の丸ボタン  
- 直径：24〜28px  
- ホバーで影が少し濃くなる  
- 透明ディスプレイの端から24px固定

---

## 5. レイアウトの基準線

### 5-1. 上下の余白
- 各レーンの上下に **24px** の余白  
- レーン間の区切りは線を使わず、**余白だけで区切る**

### 5-2. 横幅
- UI全体の横幅は **透明ディスプレイの80%**  
- 左右中央揃え

### 5-3. 高さ
- 各レーンは内容に応じて自動調整  
- 固定高さは禁止（透明感を損なうため）

---

## 6. 禁止事項（全体）
- 派手なアニメーション  
- スライド遷移  
- 不透明な背景  
- 強い影  
- カラーライト  
- ランダム生成  
- レーンごとに色を変えること

---

## 7. 目的と効果（総括）
- 透明ディスプレイUI全体の統一感を確保  
- 3レーンの役割を明確に分離  
- 工房の世界観（灰磁・真鍮・白磁）と完全統合  
- “透明で静かな創作体験” を実現する
