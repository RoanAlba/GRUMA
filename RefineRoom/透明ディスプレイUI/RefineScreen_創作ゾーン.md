# 透明ディスプレイUI：創作ゾーン（揺れの画面）仕様書（正式版 / 保存用）

## 1. 概要
本仕様書は、工房（RefineRoom）に設置された透明ディスプレイに表示される  
「創作ゾーンUI（揺れの画面）」のレイアウト・構造・世界観を定義する。

創作ゾーンUIは、作品の“揺れ”を静かに可視化し、  
創作の状態を儀式的に確認するための画面である。

---

## 2. UI全体構成（俯瞰）

+--------------------------------------------------------------+
|                     【揺れグラフ（中心）】                   |
|                ・棒グラフ / 透明感のある白線                 |
|                ・背景は完全透明（工房が透ける）             |
+--------------------------------------------------------------+
|                     揺れの総評（短文）                       |
|                ・最大2行 / 静かな文章                       |
|                ・中央揃え / 白磁系の淡い白                   |
+--------------------------------------------------------------+
|                     改善ポイント（3つまで）                  |
|                ・小さな真鍮プレート風のラベル                |
|                ・淡い影（透明度10%）                         |
+--------------------------------------------------------------+
|                     図面モード切替ボタン                     |
|                ・右下固定 / 真鍮の小さな丸ボタン             |
+--------------------------------------------------------------+

---

## 3. 揺れグラフ（メイン要素）

### 3-1. 形式
- 棒グラフ（Bar Graph）
- 本数：5〜7本（作品の性質に応じて変動）
- 透明ディスプレイに馴染む **半透明の白線（#ffffff 40%）**

### 3-2. サイズ
- 最大幅：ディスプレイ幅の 80%
- 最大高さ：ディスプレイ高さの 45%
- 棒の角丸：4px
- 棒の間隔：12px

### 3-3. アニメーション
- **禁止**：派手な伸縮アニメ  
- **許可**：0.2秒のフェードインのみ

### 3-4. 世界観
- 背景は完全透明  
- 工房の光（灰磁・真鍮）が透けて見える  
- 影はつけない（透明ディスプレイの特性）

---

## 4. 揺れの総評（短文）

### 4-1. 内容
- 最大2行  
- 静かで儀式的な文章  
- 例：「今日は落ち着いた揺れです」「集中が深まっています」

### 4-2. スタイル
- フォント：細め  
- 色：白磁系の淡い白（#f7f6f4）  
- 中央揃え  
- 上余白：24px

---

## 5. 改善ポイント（最大3つ）

### 5-1. 表示形式
- 小さな真鍮プレート風のラベル  
- 1つにつき1行  
- 最大3つまで

### 5-2. スタイル
- 背景：真鍮（#b08c4f）20%透明  
- 文字：白磁系の白  
- 影：透明度10%  
- 角丸：6px  
- 横幅：内容に応じて自動調整

---

## 6. 図面モード切替ボタン

### 6-1. 位置
- 右下固定  
- 透明ディスプレイの端から 24px / 24px

### 6-2. デザイン
- 真鍮の丸ボタン（直径24px）  
- ホバーで影が少し濃くなる  
- アイコンは「図面」風の細線

---

## 7. 世界観（透明ディスプレイ特有の要素）

### 7-1. 透明性
- UIの背景は完全透明  
- 工房の光・灰磁・真鍮が透けて見える  
- UIは“浮いている”ように見える

### 7-2. 光
- 工房の光（4300K）がUIを柔らかく照らす  
- UI自身は光らない（発光禁止）

### 7-3. 禁止事項
- 派手なアニメーション  
- 不透明な背景  
- 強い影  
- カラーライト  
- 作品ごとに色を変えること

---

## 8. 目的と効果（総括）
- 透明ディスプレイの特性を活かした“浮遊感のあるUI”を実現  
- 揺れを静かに可視化し、創作の状態を確認できる  
- 工房の世界観（灰磁・真鍮・白磁）と完全に統合  
- 図面モードへの自然な遷移を提供する
