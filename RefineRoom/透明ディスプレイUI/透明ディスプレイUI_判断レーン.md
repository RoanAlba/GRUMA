# 透明ディスプレイUI：判断レーン（正式版 / 保存用）

## 1. 概要
本仕様書は、透明ディスプレイに表示される  
「判断レーンUI」のレイアウト・構造・世界観を定義する。

判断レーンは、作品生成後に確定した  
**用途ジャンル・サイズ・改善案** の3つを  
静かに、儀式的に表示する画面である。

---

## 2. UI全体構成（俯瞰）

+--------------------------------------------------------------+
| 【用途ジャンル】 ラベル / ステッカー / JJ / 背景 / プランナー |
+--------------------------------------------------------------+
| 【サイズ】 4:5 / 1:1 / 3サイズ（JJ） / 4:3 / 9:16 / 3:2      |
+--------------------------------------------------------------+
| 【改善案】                                                   |
|  ・文字スペースを確保                                       |
|  ・コントラストを調整                                       |
|  ・モチーフ密度を下げる                                     |
+--------------------------------------------------------------+
| 右下：改善ボタン（真鍮）                                     |
+--------------------------------------------------------------+

---

## 3. 用途ジャンル（最上段）

### 3-1. 表示内容
- 生成後に確定した用途ジャンルのみ表示  
- リアルタイムの揺れは入れない（創作ゾーンとは別）

### 3-2. デザイン
- 透明背景  
- テキストは白磁系の白（#f7f6f4）  
- フォントは細め  
- 左右中央揃え  
- 上余白：20px  
- 下余白：16px

---

## 4. サイズ（中段）

### 4-1. 表示内容
- 用途ジャンルに応じて自動決定されたサイズ  
- 例：4:5 / 1:1 / 3サイズ（JJ）

### 4-2. デザイン
- テキストのみ  
- 白磁系の白  
- 透明背景  
- 角丸なし  
- 下余白：16px

---

## 5. 改善案（下段）

### 5-1. 表示内容
- 用途ジャンルに最適化された改善案  
- 最大3つ  
- 表示順は固定（上から重要度順）

### 5-2. デザイン
- 真鍮プレート風のラベル  
- 背景：真鍮（#b08c4f）20%透明  
- 文字：白磁の白  
- 角丸：6px  
- 影：透明度10%  
- 横幅：内容に応じて自動調整  
- ラベル間の余白：10px

---

## 6. 改善ボタン（右下固定）

### 6-1. 役割
- 判断レーンに表示された改善案を  
  **そのまま実行するボタン**

### 6-2. デザイン
- 真鍮の丸ボタン（直径28px）  
- ホバーで影が少し濃くなる  
- 透明ディスプレイの端から 24px / 24px

---

## 7. 世界観（透明ディスプレイ特有）

### 7-1. 透明性
- 背景は完全透明  
- 工房の光が透けて見える  
- UIは“浮いている”ように見える

### 7-2. 光と影
- UI自身は光らない  
- 影は極薄（透明度8〜10%）  
- 真鍮はにじむ程度の反射

---

## 8. 禁止事項
- 派手なアニメーション  
- スライド遷移  
- 不透明な背景  
- カラーライト  
- ランダム生成  
- 作品ごとに色を変えること

---

## 9. 目的と効果
- 生成後の確定情報を静かに提示  
- 創作ゾーンとの役割分離  
- 改善ボタンとの連動  
- 工房の世界観と完全統合
