# RefineRoom 総合仕様
RefineRoom は GRUMA における
「作品の方向性・空気・質感を磨き上げる感性の中枢」  
である。

FactoryRoom が“工場”なら、
RefineRoom は “作品の魂を整える部屋”。

本仕様は、RefineRoom の
UI配置・素材・プレビュー・調整・図面・儀式具・アニメーション・フロー  
すべてを統合した親ファイルである。

1. RefineRoom の目的（Purpose）
RefineRoom は以下を担う：

作品の 方向性・空気・質感 を決める

素材の精製結果を 静かに確認 する

調整レバーで 微細な調整 を行う

図面モードで 構造を正確に確認 する

保存レバーで 状態を刻む儀式 を行う

出品レバーで 最終儀式（不可逆） を行う

つまり、
作品の“魂の調整室” である。

2. UI全体構造（Layout Overview）

┌───────────────┬──────────────────────┬───────────────┐
│ 左：素材パネル │   中央：プレビュー   │ 右：調整レバー │
└───────────────┴──────────────────────┴───────────────┘
　　　　　　　　　　　　　　　└── 下：儀式具（保存・出品）
● 左：素材パネル
素材カテゴリの選択。UI System の項目は含めない。

● 中央：プレビュー
工房の中心。呼吸する“空気の器”。

● 右：調整レバー
光・質感・数値の精密調整。

● 下：儀式具
保存レバー・出品レバー。

3. 素材スロット（Material Slot）
プレビュー左下に配置

素材は 選択で自動的に置かれる

真鍮枠が光り、白磁が息をする

プレビューが素材を受け取って呼吸する

ドラッグ操作は禁止

4. プレビュー（Preview）
RefineRoom の中心

枠なし・影なし・角丸なし

呼吸アニメーション（+2% → 0%）

素材変更時は +3% 明るくなる

図面モード切り替えはフェードのみ

完了状態では呼吸が浅くなる

5. 調整レバー（Adjustment Levers）
白金属レール＋白磁つまみ

縦スライダー

光量・質感・数値を調整

プレビューに即時反映

レバー自体はアニメーションしない（精密さ優先）

ホバーは +3% 明るくなるだけ

完了状態では操作不可

6. 図面モード（Blueprint Mode）
プレビュー右上のアイコンで切り替え

0.2秒フェードで線図に変わる

呼吸は弱く維持

レバー操作は線図に即時反映

工房の光は変わらない

完了状態ではさらに呼吸が浅くなる

7. 儀式具（保存・出品レバー）
● 保存レバー
小型T型

倒し切った瞬間に保存

真鍮が +20% 光る

大理石が息をする

プレビューが +2% 明るくなる

UIカードが一覧に追加される

● 出品レバー
大型T型（保存の1.3〜1.4倍）

最終儀式

真鍮が +25〜30% 光る

台座が遅れて光る

画面全体が 25% 暗転

プレビューが深い呼吸

完了カードに変化（不可逆）

8. アニメーション（Animation Rules）
動きは最小限

フェード中心

バウンド禁止

音なし

真鍮だけが光る

呼吸アニメーションは常に維持

図面モードはフェードのみ

儀式具は特別な光を持つ

9. フロー（Flow）

① 素材選び
　↓
② 素材スロットに反映
　↓
③ プレビューが息を吸う
　↓
④ 調整レバーで微調整
　↓
⑤ 図面モードで構造確認（任意）
　↓
⑥ 保存レバーで保存（儀式）
　↓
⑦ 出品レバーで最終儀式（不可逆）

RefineRoom の作業は
「素材 → 調整 → 図面 → 儀式」  
という一本の線で繋がる。

10. 禁止事項（NG）
UIを左右逆にする

プレビューに枠を付ける

ドラッグ＆ドロップ

派手な光

バウンド

音

自動保存

出品の取り消し

図面モードで色を付ける

レバーを中央に置く

プレビューをポップアップ化する

11. RefineRoom の本質（Essence）
RefineRoom は
「作品の方向性・空気・質感を磨くための静かな工房」。

素材を捧げ

空気の器で確認し

精密に調整し

図面で構造を確かめ

儀式具で仕上げる

この一連の流れが
作品の“魂”を整える儀式 となる。

✔ 最短まとめ
RefineRoom は 作品の魂を磨く部屋

UI配置・素材・プレビュー・調整・図面・儀式具・アニメーション・フローを統合

世界観の静けさと儀式性を守る

RefineRoom フォルダに保存する
