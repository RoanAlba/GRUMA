# RefineRoom フロー仕様
RefineRoom のフローは、
「素材 → 配置 → 調整 → 図面 → 儀式（保存・出品）」  
という一連の流れを、工房の静けさを保ったまま成立させるための設計。

目的は、
“迷いなく、静かに、儀式的に進む編集体験” を保証すること。

1. 全体フロー（Overview）

① 素材選び（左パネル）
　↓
② 素材スロットに反映（自動）
　↓
③ プレビューが息を吸う（素材読み込み）
　↓
④ 調整レバーで微調整（右パネル）
　↓
⑤ 図面モードで構造確認（任意）
　↓
⑥ 保存レバーで保存（儀式）
　↓
⑦ 出品レバーで最終儀式（不可逆）

この流れが RefineRoom の“作業の道筋”。

2. ① 素材選び（Material Select）
左パネルで素材カテゴリを選ぶ

素材をクリックすると 素材スロットに自動で置かれる

ドラッグ操作はしない

選択時のアニメーションは最小限（影＋3%）

→ 次の工程：素材スロット

3. ② 素材スロット（Material Slot）
素材が置かれた瞬間、真鍮枠が +10% 光る

白磁が +3% 明るくなる

プレビューが息を吸う（+2% → 0%）

素材の初期値が調整レバーに反映される

→ 次の工程：プレビュー

4. ③ プレビュー（Preview）
RefineRoom の中心

素材の状態を静かに表示

呼吸アニメーション（+2% → 0%）

素材変更時は +3% 明るくなる（0.2秒）

図面モード切り替えはフェードのみ（0.2秒）

→ 次の工程：調整レバー

5. ④ 調整レバー（Adjustment Levers）
光量・質感・数値を調整

プレビューに即時反映

レバー自体はアニメーションしない（精密さ優先）

ホバーは +3% 明るくなるだけ

完了カード読み込み時は操作不可

→ 次の工程：図面モード（任意） or 保存レバー

6. ⑤ 図面モード（Blueprint Mode）※任意
プレビュー右上のアイコンで切り替え

0.2秒フェードで線図に変わる

呼吸は弱く維持

レバー操作は線図に即時反映

工房の光は変わらない

完了状態では呼吸がさらに浅くなる

→ 次の工程：保存レバー or 出品レバー

7. ⑥ 保存レバー（Save Ritual）
倒し切った瞬間に保存

真鍮が +20% 光る（0.25秒）

大理石が +3% → 0%（0.2秒）

プレビューが +2% 明るくなる

UIカードが一覧に追加される

レバーは 0.3秒で戻る

→ 次の工程：作業継続 or 出品レバー

8. ⑦ 出品レバー（Publish Ritual）
最終儀式

倒し切った瞬間に出品

真鍮が +25〜30% 光る（0.3秒）

台座が遅れて光る（0.2秒）

画面全体が 25% 暗転（0.25秒）

プレビューが深い呼吸（+4% → 0%）

完了カードに変化（編集不可）

レバーは 0.4秒で戻る

→ フロー終了（不可逆）

9. フロー全体のアニメーション（Animation Rules）
動きは最小限

フェード中心

バウンド禁止

音なし

真鍮だけが光る

呼吸アニメーションは常に維持

儀式具（保存・出品）は特別な光を持つ

図面モードはフェードのみ

10. 禁止事項（NG）
フローを逆にする

素材スロットを複数にする

プレビューをポップアップ化

レバーを中央に置く

派手なアニメーション

音

自動保存

出品の取り消し

図面モードで色を付ける

11. 目的（Design Purpose）
RefineRoom のフローは
「素材を捧げ、調整し、確認し、儀式で仕上げる」  
という世界観の流れを UI として成立させるために存在する。

素材 → 器 → 調整 → 図面 → 儀式

静けさ

呼吸

真鍮の光

大理石の息

これらがひとつの流れとして繋がることで、
RefineRoom は “静かに動く工房” として完成する。

✔ 最短まとめ
RefineRoom のフローは 素材 → 調整 → 図面 → 儀式

素材スロット → プレビュー → レバー → 図面 → 保存 → 出品

呼吸・光・静けさが全体を貫く

出品は不可逆

RefineRoom フォルダに保存する
