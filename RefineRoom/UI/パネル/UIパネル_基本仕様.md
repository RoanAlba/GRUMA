# UIパネル_基本仕様.md

UIパネルのまとめ（核だけ）
■ UIパネルは素材スロット方式
パネルは 素材を差し替えるだけで無限に作れる

枠・反応エフェクト（青い粒）は共通

形状も共通（薄い石板・白金の縁）

■ 素材はリファインルームで生成
壁の白が素材を綺麗に見せる

窓の景色が背景素材にもなる

アーチが基準線
→ リファインルーム＝UI素材工房

■ パネル素材は無限に追加可能
現在のカテゴリ：

白大理石（細いスジ）

白大理石（太い黒スジ）

白大理石（金スジ）

白大理石（白金スジ）

黒大理石

金属

ガラス

木材

宝石

陶器

樹脂

光素材

→ 全部 UI パネルとして成立する

■ 壁は真っ白キラキラ石で固定
部屋を広く見せる

UI と混ざらない

黒アーチが映える

素材撮影に最適

■ UIは白系に縛られない
パネルはどんな素材でもOK

壁が白だから空間の広さは確保済み

UIは“光の装置”として自由でいい

■ UIパネルとボタンは別ファイル
パネル＝土台

ボタン＝独立素材（宝石以外も作れる）

販売構造が綺麗になる

無限生成が成立する

■ UIパネルとは
UI の“土台”となるパネル部分

素材を差し替えることで無限に生成できる

枠・影・反応（青い粒）は共通

パネルの模様・色・質感だけが変わる

リファインルームで生成した素材をそのまま使える

■ パネルの構造
UIパネルは以下の3層で構成される。

ベース素材層（差し替え可能）

大理石

金属

ガラス

木材

宝石

陶器

樹脂

光素材
→ ここが無限に増える部分

枠（共通）

白金系の細い縁

素材によって縁の色を微調整可能

UIの統一感を保つ役割

反応エフェクト（共通）

操作時に青い粒が縁から湧く

素材に依存しない

UIの“光の装置”としての性質を保つ

■ パネル素材の種類（無限生成）
以下は“例”ではなく、正式に採用されるカテゴリ。

白大理石（細いスジ）

白大理石（太い黒スジ）

白大理石（金スジ）

白大理石（白金スジ）

黒大理石

金属（鉄・白金・真鍮）

ガラス（透明・曇り・色ガラス）

木材（白木・黒木・ウォルナット）

宝石（ルビー・サファイア・エメラルド）

陶器（白磁・青磁）

樹脂（透明・色樹脂）

光素材（発光・粒子・ホログラム）

→ 全部 UI パネルとして成立する。
　全部“売れる素材”になる。

■ パネルのサイズ・形状
基本は縦長の白金枠

角は丸すぎず、直線的

厚みは薄い石板

素材が変わっても形状は共通

形状を固定することで“素材だけ無限”が成立する

■ リファインルームとの関係
リファインルームで生成した素材を
そのままパネルのベース素材に使える

壁の白が素材を綺麗に見せる

窓の景色が背景素材としても使える

アーチが基準線になる

→ リファインルーム＝UI素材生成工房  
→ UIパネル＝生成された素材の“形”

■ 今後追加できる要素
素材ごとの光の反射設定

素材ごとの影の落ち方

素材ごとの縁の色調整

素材ごとの反応エフェクトの強さ

→ 全部後から追加できる。
　今は“素材スロット方式”だけ決めれば十分。
