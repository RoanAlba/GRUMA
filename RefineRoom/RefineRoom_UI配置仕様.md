# RefineRoom UI配置仕様
RefineRoom 内の UIパーツの位置・役割・優先度・視線誘導 を定義する。
目的は 「工房の静けさを壊さず、素材精製の流れが迷わず成立する配置」 を作ること。

1. 全体レイアウト（Overview）
RefineRoom は 左右3分割構造 を基本とする。

┌───────────────┬───────────────┬───────────────┐
│ 左：素材パネル │ 中央：プレビュー │ 右：調整レバー群 │
└───────────────┴───────────────┴───────────────┘
左：素材を選ぶ

中央：結果を見る

右：数値・質感を調整する

この3つが 視線の流れ（左→中央→右） に沿って配置される。

2. 左パネル（素材パネル）
● 役割
素材カテゴリの一覧

素材の切り替え

UI System の項目は含めない（素材専用）

● 配置
画面左端に固定

幅：画面の 22〜26%

背景：白磁パネル（半透明）

影：極弱（1%）

● 操作
スクロールは縦のみ

選択中の項目だけ淡く光る

ホバーは最小限（影1px）

3. 中央（プレビューエリア）
● 役割
精製結果の表示

工房の “静かな器” として存在する

● 配置
画面中央

幅：画面の 40〜48%

背景：工房の空気層（薄いグレー）

影：なし

枠：なし（額縁は不要）

● 特徴
呼吸アニメーション（+2% → 0%）

光の反射は弱め

UIの邪魔をしない

4. 右パネル（調整レバー群）
● 役割
数値・光量・質感などの調整

操作専用の UI（アンティーク思想とは接続しない）

● 配置
画面右端

幅：画面の 22〜26%

背景：白金属の細いバー構造

影：なし

つまみ：陶器（宝石ではない）

● 操作
スライダーは縦方向

レバーは右下に配置（保存・出品）

ホバーは最小限

5. 下部（儀式具エリア）
● 役割
保存レバー

出品レバー

最終操作の儀式具をまとめる

● 配置
右下に 保存レバー

その左に 出品レバー（保存より大きい）

レバー同士は干渉しない距離を確保

台座は大理石＋真鍮ライン

6. UIの優先度（Hierarchy）
● 最優先（視線の中心）
中央プレビュー

素材の結果が最も重要

● 次点
右パネルの調整レバー

素材パネル

● 儀式具
保存レバー

出品レバー
→ 操作頻度は低いが “重み” がある

7. 禁止事項（NG）
UIを左右逆にする

プレビューを小さくする

レバーを中央に置く

素材パネルを折りたたむ

右パネルをポップアップ化する

プレビューに枠線を付ける

背景を濃くする

情報量を増やす（検索・タグなど）

→ RefineRoom の静けさと儀式性を守るため。

8. 目的（Design Purpose）
RefineRoom の UI配置は
「素材 → 結果 → 調整 → 儀式」  
という一連の流れを自然に成立させるために存在する。

左で素材を選び

中央で結果を見て

右で調整し

下で儀式を行う

この流れが 迷いなく、静かに、儀式的に 成立するように設計されている。

✔ 最短まとめ
RefineRoom は 左→中央→右 の3分割

左：素材パネル

中央：プレビュー

右：調整レバー

下：保存レバー＋出品レバー

世界観の静けさと儀式性を守る配置

RefineRoom フォルダに保存する
