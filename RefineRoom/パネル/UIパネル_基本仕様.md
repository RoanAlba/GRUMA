UIパネルのまとめ（核だけ）
■ UIパネルは素材スロット方式
パネルは 素材を差し替えるだけで無限に作れる

枠・反応エフェクト（青い粒）は共通

形状も共通（薄い石板・白金の縁）

■ 素材はリファインルームで生成
壁の白が素材を綺麗に見せる

窓の景色が背景素材にもなる

アーチが基準線
→ リファインルーム＝UI素材工房

■ パネル素材は無限に追加可能
現在のカテゴリ：

白大理石（細いスジ）

白大理石（太い黒スジ）

白大理石（金スジ）

白大理石（白金スジ）

黒大理石

金属

ガラス

木材

宝石

陶器

樹脂

光素材

→ 全部 UI パネルとして成立する

■ 壁は真っ白キラキラ石で固定
部屋を広く見せる

UI と混ざらない

黒アーチが映える

素材撮影に最適

■ UIは白系に縛られない
パネルはどんな素材でもOK

壁が白だから空間の広さは確保済み

UIは“光の装置”として自由でいい

■ UIパネルとボタンは別ファイル
パネル＝土台

ボタン＝独立素材（宝石以外も作れる）

販売構造が綺麗になる

無限生成が成立する
