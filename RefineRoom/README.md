## ■ スライダーつまみの仕様（antique との接続）

スライダーのつまみは、design/antique の思想に基づく
「宝石で構成されたアンティークジュエリー」として扱う。

- つまみは宝石＋金属枠でできた小さなジュエリー
- 宝石の形は自由に変化する（ラウンド・オーバル・ペアシェイプなど）
- 初期値（デフォルト）はラウンド／カボション／淡い宝石
- セッティング台に置くとアンティーク思想に従ってジュエリーへ自動変換される
- 完成したジュエリーは保存され、ジュエリールームに飾られる

UI のつまみは、単なる操作部品ではなく、
アンティーク思想を UI に持ち込むための “世界観の接点” として機能する。
