# RefineRoom（リファインルーム）

RefineRoom は、GRUMA の UI を「磨き上げる」ための部屋。
光・影・質感・操作感・パーツ構造など、UI の完成度を高める工程を扱う。

ここでは素材制作やアイデア出しは行わず、
既にある UI を「整える・精製する・静かに仕上げる」ことだけに集中する。

---

## ■ RefineRoom の役割
- UI の質感・光・影の調整
- パーツの造形・動きの磨き込み
- 世界観（アンティーク思想）と UI の接続を管理する
- UI の“静けさ”と“精密さ”を保つ

---

## ■ UI パーツ一覧

### ● スライダーつまみ（UI仕様／antique 接続）
- つまみは「宝石＋アンティーク金属枠」で構成された小さなジュエリー
- UI の操作部品でありながら、アンティーク思想に基づく宝飾物として扱う

■ デフォルト（初期値）
- 形：ラウンド
- カット：カボション
- 色：淡い宝石（UI を邪魔しない初期値）

※ デフォルトはあくまで初期値であり、変化後の宝石の形・色・カットは完全に自由。

■ 変化後（ユーザー操作・世界の動き）
- ラウンド／オーバル／ペアシェイプ／クッション／マーキスなど自由に変化
- カットも自由（カボション・ファセット・アンティークカットなど）
- 色も自由（濃色・透明・異物色など）

■ 自動変換と保存
- セッティング台に置くと、アンティーク思想に従ってジュエリーへ自動変換される
- 完成したジュエリーは「保存」操作でコレクション化され、ジュエリールームに飾られる

---

### ● スライダー本体（レール）
- 白金属の細いバー
- 極細刻み目で“精密さ”を表現
- 影は弱く、工場の空気層を壊さない

---

### ● レバー（UI 呼び出し用）
- 白金属バー＋白金属つまみ
- 引くと UI が展開し、戻すと消える
- 存在感は薄く、工場の景観を邪魔しない

---

### ● パネル（左：素材パネル）
- 白金属フレーム＋淡い透明層
- 影を落とさず、UI の背景を邪魔しない

---

### ● プレビュー（中央）
- 精製結果を表示する“静かな器”
- 光の反射は弱め
- 背景の工場を邪魔しない

---

### ● 調整レバー（右）
- 白金属の細いバー
- つまみも白金属（操作専用）

---

### ● 項目セレクター
- 白金属ボタン／白金属スライダー／小さなアイコン
- 選択中の項目だけ淡く光る
- アンティーク思想とは直接接続しない（UI 専用）

---

## ■ UI 全体構造（左・中央・右）

RefineRoom の UI は、以下の 3 つの領域で構成される。

┌──────────────────────────────────────────────┐
│                 RefineRoom UI                │
├──────────────────────────────────────────────┤
│  ┌──────────────┐  ┌──────────────────────┐  ┌──────────────┐
│  │   左：素材パネル │  │   中央：プレビュー（静かな器） │  │   右：調整レバー群 │
│  │  （一覧・選択） │  │  （結果の表示領域）          │  │  （光・影・質感） │
│  └──────────────┘  └──────────────────────┘  └──────────────┘

### ● 左：素材パネル
- 素材・項目の一覧を表示
- 白金属フレーム＋淡い透明層
- 影を落とさず、工場の空気を壊さない

### ● 中央：プレビュー（静かな器）
- 精製結果を表示する中心領域
- 光の反射は弱め
- 背景の工場を邪魔しない

### ● 右：調整レバー群
- 光・影・質感などを微調整
- 白金属の細いバーで構成
- 存在感は最小限

- 白金属の極細ライン
- 工場の空気層を壊さないため、存在感は最小
