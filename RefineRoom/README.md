# RefineRoom（リファインルーム）— 完全仕様書

RefineRoom は、GRUMA の UI を「磨き上げる」ための部屋。
光・影・質感・操作感・動き・空気層など、UI の完成度を高める工程を扱う。

ここでは素材制作やアイデア出しは行わず、
既にある UI を「整える・精製する・静かに仕上げる」ことだけに集中する。

---

## ■ RefineRoom の役割
- UI の質感・光・影の調整
- パーツの造形・動きの磨き込み
- 世界観（アンティーク思想）と UI の接続
- UI の“静けさ”と“精密さ”の維持
- 内部装置（RefineRoom_Machines）との同期による “操作の手ごたえ” の実現

---

## ■ UI 全体レイアウト（最終版）
RefineRoom の編集画面は **三層構造** で構成される。

┌──────────────┬──────────────────────┬────────────────────────┐
│ 左：素材＆工程入口 │ 中央：プレビュー（ガイド線） │ 右：内部装置UI（Machines UI） │
└──────────────┴──────────────────────┴────────────────────────┘

---

## ■ 左：素材＆工程の入口（Material & Process Panel）

### ● 素材カテゴリ
- 宝石素材
- テクスチャ素材
- モチーフ素材

### ● 素材一覧
- 白金属フレーム＋透明層
- 選択中の素材だけ淡く光る
- スクロールは静かで慣性なし

### ● 素材選択時の内部装置反応（準備動作）
- 巨大輪がわずかに加速
- 粒子がふわっと舞う
- 加工ベルトが低速で動き始める
- 光ラインに淡い光が走る

### ● 工程選択（5つ）
- 形
- 光
- 色
- 質感
- 仕上げ

工程を選ぶと、右パネルの Machines UI が切り替わる。

---

## ■ 中央：プレビュー（ガイド線レイヤー付き）

### ● プレビューの役割
- 編集の中心
- 内部装置の出力がリアルタイムで反映
- 光の反射は弱め
- 背景の内部装置を邪魔しない

### ● ガイド線
- 素材を選ぶと出現
- 素材を置くと固定
- 色は無彩色、影なし
- フェードイン 0.2 秒

---

## ■ 右：内部装置 UI（Machines UI）
工程に応じて、以下の 5 つの UI が切り替わる。

### ● ① 成形ローラー UI（Shape Roller）
- 形の調整（丸み・厚み・エッジ）
- ダイヤル回転 → 成形ローラーが同速度で回転
- プレビューの形がリアルタイムで変化

### ● ② 光反射板 UI（Reflection Plate）
- 光の角度・強さ・揺れ
- レバー角度 → 反射板が同角度に動く
- プレビューの光が静かに変化

### ● ③ 色タンク UI（Color Tank）
- 色の深さ・濁り・透明度
- ダイヤル回転 → 液体が揺れ、色が変化

### ● ④ 微粒子ライン UI（Particle Line）
- 粒子・ザラつき・粗さ
- スイッチ ON → 粒子ライン開
- スイッチ OFF → 粒子が吸い込まれる

### ● ⑤ プレスアーム UI（Press Arm）
- 仕上げ（決定・保存）
- レバーを下げる → プレスアームが降りる
- プレビューが完成形に変わる

---

## ■ Undo（逆再生）
- 成形 → ローラー逆回転
- 光 → 反射板が逆角度へ戻る
- 色 → 液体が沈む
- 質感 → 粒子が収束
- 仕上げ → プレスアーム上昇

プレビューも同時に逆再生される。

---

## ■ 内部装置の動き（Machines Motion）

### ● 常時稼働
- 巨大輪がゆっくり回転
- ベルトが低速で動く
- 粒子が空気中で揺れる

### ● UI 操作との同期
- つまみ → 光が強くなる
- レバー → 装置がわずかに加速
- スイッチ → 粒子ラインが開閉

### ● 音
- 完全無音  
- 視覚的な重さだけで表現

---

## ■ UI の色・空気層・アニメーション（共通ルール）

### ● 色
- 白金属・透明層・宝石のみ
- 発光色は禁止
- 宝石だけが彩度を持つ

### ● 空気層
- 透明度 0.0〜0.6
- 色なし
- 屈折は最小限

### ● アニメーション
- 速すぎる動き禁止
- バウンド禁止
- 影が揺れすぎる動き禁止
- “静けさ” を最優先

---

## ■ ガイド線 UI（Guide Line UI）
素材を中央に「正しく置く」ための位置決めレイヤー。

### ● ガイド線の役割
- 素材の置き場所を示す
- 位置決めの補助
- 内部装置とは独立した UI レイヤー

### ● 表示タイミング
- 素材選択 → フェードイン
- 素材配置 → 固定
- 工程開始 → フェードアウト

### ● 見た目
- 無彩色（白〜薄灰）
- 影なし
- 線幅 0.5〜1.0px
- 透明度 0.3〜0.5

---

## ■ 工程切り替え UI（Process Switch UI）

### ● 工程一覧
- 形（Shape）
- 光（Light）
- 色（Color）
- 質感（Texture）
- 仕上げ（Finish）

### ● ボタンの見た目
- 白金属の小さなプレート
- 影なし
- 選択中だけ淡く光る（0.15 秒）

### ● 工程選択 → 内部装置の対応ラインが起動
- 形 → 成形ライン
- 光 → 反射板ライン
- 色 → 色タンクライン
- 質感 → 粒子ライン
- 仕上げ → プレスアーム

---

## ■ 内部装置との同期ルール（Machines Sync Rules）

### ● 基本原則
- UI 操作 → 内部装置の動きに反映
- 内部装置 → UI を揺らさない（空気層が緩衝）
- プレビューは 0.1 秒以内に追従
- 音は一切発生しない

---

## ■ RefineRoom 全体フロー（Flow Overview）

1. **素材選び**  
2. **素材配置（ガイド線）**  
3. **工程選択**  
4. **内部装置 UI で調整**  
5. **仕上げ・保存**  
6. **Undo（任意）**

RefineRoom のすべての操作は  
“静けさ・精密さ・内部装置との同期” を最優先に設計されている。
