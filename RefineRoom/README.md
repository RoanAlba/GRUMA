# RefineRoom（リファインルーム）— 完全仕様書

RefineRoom は、GRUMA の UI を「磨き上げる」ための部屋。
光・影・質感・操作感・動き・空気層など、UI の完成度を高める工程を扱う。

ここでは素材制作やアイデア出しは行わず、
既にある UI を「整える・精製する・静かに仕上げる」ことだけに集中する。

---

## ■ RefineRoom の役割
- UI の質感・光・影の調整
- パーツの造形・動きの磨き込み
- 世界観（アンティーク思想）と UI の接続
- UI の“静けさ”と“精密さ”の維持
- 工場（Factory）との同期による “操作の手ごたえ” の実現

---

# ■ UI 全体レイアウト（最終版）

RefineRoom の編集画面は **三層構造** で構成される。

┌──────────────┬──────────────────────┬──────────────────────┐
│ 左：素材＆工程入口 │   中央：プレビュー（ガイド線）   │ 右：工場の機械UI（1つだけ表示） │
└──────────────┴──────────────────────┴──────────────────────┘

---

# ■ 左：素材＆工程の入口（Material & Process Panel）

### ● 素材カテゴリ
- 宝石素材
- テクスチャ素材
- モチーフ素材

### ● 素材一覧
- 白金属フレーム＋透明層
- 選択中の素材だけ淡く光る
- スクロールは静かで慣性なし

### ● 素材選択時の工場反応（準備動作）
- 巨大輪がわずかに加速
- 粒子がふわっと舞う
- 加工ベルトが低速で動き始める
- 光ラインに淡い光が走る

### ● 工程選択（5つ）
- 形
- 光
- 色
- 質感
- 仕上げ

工程を選ぶと、右パネルの機械UIが切り替わる。

---

# ■ 中央：プレビュー（ガイド線レイヤー付き）

### ● プレビューの役割
- 編集の中心
- 工場の出力がリアルタイムで反映
- 光の反射は弱め
- 背景の工場を邪魔しない

### ● ガイド線
- 素材を選ぶと出現
- 素材を置くと固定
- 色は無彩色、影なし
- フェードイン 0.2 秒

---

# ■ 右：工場の機械UI（Factory Machine UI）

工程に応じて、以下の 5 つの機械UIが切り替わる。

---

## ● ① 成形ローラー UI（Shape Roller）
**役割：形の調整（丸み・厚み・エッジ）**

- 白金属の円形ダイヤル
- 表面に極細刻み目
- 回すと奥の成形ローラーが同じ速度で回転
- プレビューの形がリアルタイムで変化
- 工場のローラー・ベルトが微加速

---

## ● ② 光反射板 UI（Reflection Plate）
**役割：光の角度・強さ・揺れ**

- 白金属の細長いレバー
- 倒すと工場奥の反射板が同じ角度に動く
- 光ラインがわずかに強くなる
- プレビューの光が静かに変化

---

## ● ③ 色タンク UI（Color Tank）
**役割：色の深さ・濁り・透明度**

- アンティーク計器のような丸いダイヤル
- 内部に淡い液体が揺れる小窓
- 回すとタンクの液体が揺れ、色が変化
- 宝石内部の光が変わる

---

## ● ④ 微粒子ライン UI（Particle Line）
**役割：質感（粒子・ザラつき・粗さ）**

- 白金属のトグルスイッチ
- 上：粒子ライン開（粒子が舞う）
- 下：粒子ライン閉（粒子が吸い込まれる）
- プレビューの質感が変化

---

## ● ⑤ プレスアーム UI（Press Arm）
**役割：仕上げ（決定・保存）**

- 大型の白金属レバー
- 下げると工場奥のプレスアームが降りる
- 完成ラインの光が走る
- プレビューが完成形に変わる

---

# ■ やり直し（Undo）— 逆再生アニメーション

各工程には “逆再生” が存在する。

- 成形 → ローラーが逆回転
- 光 → 反射板が逆角度へ戻る
- 色 → 液体が沈み、色が吸い込まれる
- 質感 → 粒子が収束
- 仕上げ → プレスアームがゆっくり上昇

プレビューも静かに元へ戻る。

---

# ■ 工場の動き（Factory Motion）

### ● 常時稼働
- 巨大輪がゆっくり回転
- ベルトが低速で動く
- 粒子が空気中で揺れる

### ● UI 操作との同期
- つまみ → 光が強くなる
- レバー → 機械がわずかに加速
- スイッチ → 粒子ラインが開閉

### ● 音
- 完全無音
- 視覚的な重さだけで表現

---

# ■ UI の色・空気層・アニメーション（共通ルール）

### ● 色
- 白金属・透明層・宝石のみ
- 発光色は禁止
- 宝石だけが彩度を持つ

### ● 空気層
- 透明度 0.0〜0.6
- 色なし
- 屈折は最小限

### ● アニメーション
- 速すぎる動き禁止
- バウンド禁止
- 影が揺れすぎる動き禁止
- すべて “静けさ” を最優先

# ■ ガイド線 UI（Guide Line UI）

ガイド線 UI は、素材を中央プレビューに「正しく置く」ための位置決めレイヤー。
工場UIとは独立した、RefineRoom の基礎機能。

---

## ● ガイド線の役割
- 素材の置き場所を示す
- 位置決めの補助
- 工場の動きとは独立した UI レイヤー
- 編集の最初のステップを支える

---

## ● ガイド線の表示タイミング
1. 左パネルで素材を選んだ瞬間 → ガイド線がフェードイン  
2. 素材を中央にドラッグすると → ガイド線が追従  
3. 素材を置いた瞬間 → ガイド線が固定  
4. 工程を開始すると → ガイド線は自動でフェードアウト

---

## ● ガイド線の見た目
- 完全無彩色（白〜薄灰）
- 影なし
- 線幅は極細（0.5〜1.0px）
- 透明度は 0.3〜0.5
- 工場の背景を邪魔しない“静かな線”

---

## ● ガイド線の種類
- 中央基準線（縦・横）
- 素材の外形に合わせた輪郭ガイド
- 配置ポイント（淡い点）
- 回転ガイド（円弧）

※ すべて必要なときだけ表示され、不要なときは自動で消える。

---

## ● ガイド線の動き（アニメーション）
- フェードイン：0.2 秒
- フェードアウト：0.2 秒
- スライド・バウンドは禁止
- 空気層の屈折は最小限（0.1 秒以内）

---

## ● 素材との連動
- 素材をドラッグすると、ガイド線が“吸い付くように”位置補正
- 素材が中央に近づくと、ガイド線が淡く強調
- 素材を置くと、ガイド線が静かに固定される

---

## ● 工場との関係
- ガイド線は工場の動きに影響されない
- 工場の光・粒子・動きがガイド線を揺らすことはない
- 工場UI（右パネル）を操作するとガイド線は自動で消える

---

## ● ガイド線が消える条件
- 工程（形・光・色・質感・仕上げ）を開始したとき
- プレスアームで仕上げを行ったとき
- 素材を変更したとき（新しいガイド線が生成される）
