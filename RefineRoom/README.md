# RefineRoom（リファインルーム）

RefineRoom は、GRUMA の UI を「磨き上げる」ための部屋。
光・影・質感・操作感・動き・空気層など、UI の完成度を高める工程を扱う。

ここでは素材制作やアイデア出しは行わず、
既にある UI を「整える・精製する・静かに仕上げる」ことだけに集中する。

---

## ■ RefineRoom の役割
- UI の質感・光・影の調整
- パーツの造形・動きの磨き込み
- 世界観（アンティーク思想）と UI の接続を管理する
- UI の“静けさ”と“精密さ”を保つ

---

## ■ UI パーツ一覧

### ● スライダーつまみ（UI仕様／antique 接続）
- つまみは「宝石＋アンティーク金属枠」で構成された小さなジュエリー
- UI の操作部品でありながら、アンティーク思想に基づく宝飾物として扱う

■ デフォルト（初期値）
- 形：ラウンド
- カット：カボション
- 色：淡い宝石（UI を邪魔しない初期値）
※ デフォルトは初期値であり、変化後の宝石の形・色・カットは完全に自由。

■ 変化後
- ラウンド／オーバル／ペアシェイプ／クッション／マーキスなど自由
- カットも自由（カボション・ファセット・アンティークカットなど）
- 色も自由（濃色・透明・異物色など）

■ 自動変換と保存
- セッティング台に置くとアンティーク思想に従ってジュエリーへ自動変換
- 完成したジュエリーは保存され、ジュエリールームに飾られる

- 光の反射は弱め
- 背景の工場を邪魔しない

### ● 調整レバー（右）
- 白金属の細いバー

## ■ UI アニメーション仕様（RefineRoom）

RefineRoom の UI は「静けさ」「精密さ」「工場の空気層」を壊さないため、
アニメーションは最小限・低速・滑らかさを基準とする。

### ● スライダーつまみ（宝石）の動き
- 重さのある宝石を前提にした、わずかに粘性のあるスライド
- 加速・減速は弱く、一定速度に近い
- 位置が変わると中央プレビューの光・影が静かに反応
- 離した瞬間、宝石内部の光がわずかに揺れる

### ● レバー（UI 呼び出し）の動き
- 白金属の機構が静かに展開する動き
- 無音
- 展開速度は遅め
- 戻すときは “吸い込まれるように” 静かに閉じる

### ● パネル（左：素材パネル）の展開
- スライドではなく “淡い透明層が静かに現れる”
- フェードイン 0.2〜0.3 秒
- 影を落とさない

### ● プレビュー（中央）の変化
- 瞬間切り替えはしない
- 0.15 秒の淡いフェードで静かに置き換わる
- 光の反射がわずかに揺れる

### ● 調整レバー（右）の動き
- 精密機械の微調整のような直線的な動き
- 反応は即時
- 影はほとんど落とさない
- プレビューが静かに追従する

### ● UI 全体のアニメーション原則
- 速すぎる動きは禁止
- バウンド・弾性・跳ね返りは禁止
- 影が揺れすぎる動きは禁止
- すべての動きは “工場の空気層を壊さない” を最優先とする
