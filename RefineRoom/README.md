# RefineRoom（リファインルーム）— 完全仕様書

RefineRoom は、GRUMA の UI を「磨き上げる」ための部屋。
光・影・質感・操作感・動き・空気層など、UI の完成度を高める工程を扱う。

ここでは素材制作やアイデア出しは行わず、
既にある UI を「整える・精製する・静かに仕上げる」ことだけに集中する。

---

## ■ RefineRoom の役割
- UI の質感・光・影の調整
- パーツの造形・動きの磨き込み
- 世界観（アンティーク思想）と UI の接続
- UI の“静けさ”と“精密さ”の維持
- 工場（Factory）との同期による “操作の手ごたえ” の実現

---

# ■ UI 全体レイアウト（最終版）

RefineRoom の編集画面は **三層構造** で構成される。

┌──────────────┬──────────────────────┬──────────────────────┐
│ 左：素材＆工程入口 │   中央：プレビュー（ガイド線）   │ 右：工場の機械UI（1つだけ表示） │
└──────────────┴──────────────────────┴──────────────────────┘

---

# ■ 左：素材＆工程の入口（Material & Process Panel）

### ● 素材カテゴリ
- 宝石素材
- テクスチャ素材
- モチーフ素材

### ● 素材一覧
- 白金属フレーム＋透明層
- 選択中の素材だけ淡く光る
- スクロールは静かで慣性なし

### ● 素材選択時の工場反応（準備動作）
- 巨大輪がわずかに加速
- 粒子がふわっと舞う
- 加工ベルトが低速で動き始める
- 光ラインに淡い光が走る

### ● 工程選択（5つ）
- 形
- 光
- 色
- 質感
- 仕上げ

工程を選ぶと、右パネルの機械UIが切り替わる。

---

# ■ 中央：プレビュー（ガイド線レイヤー付き）

### ● プレビューの役割
- 編集の中心
- 工場の出力がリアルタイムで反映
- 光の反射は弱め
- 背景の工場を邪魔しない

### ● ガイド線
- 素材を選ぶと出現
- 素材を置くと固定
- 色は無彩色、影なし
- フェードイン 0.2 秒

---

# ■ 右：工場の機械UI（Factory Machine UI）

工程に応じて、以下の 5 つの機械UIが切り替わる。

---

## ● ① 成形ローラー UI（Shape Roller）
**役割：形の調整（丸み・厚み・エッジ）**

- 白金属の円形ダイヤル
- 表面に極細刻み目
- 回すと奥の成形ローラーが同じ速度で回転
- プレビューの形がリアルタイムで変化
- 工場のローラー・ベルトが微加速

---

## ● ② 光反射板 UI（Reflection Plate）
**役割：光の角度・強さ・揺れ**

- 白金属の細長いレバー
- 倒すと工場奥の反射板が同じ角度に動く
- 光ラインがわずかに強くなる
- プレビューの光が静かに変化

---

## ● ③ 色タンク UI（Color Tank）
**役割：色の深さ・濁り・透明度**

- アンティーク計器のような丸いダイヤル
- 内部に淡い液体が揺れる小窓
- 回すとタンクの液体が揺れ、色が変化
- 宝石内部の光が変わる

---

## ● ④ 微粒子ライン UI（Particle Line）
**役割：質感（粒子・ザラつき・粗さ）**

- 白金属のトグルスイッチ
- 上：粒子ライン開（粒子が舞う）
- 下：粒子ライン閉（粒子が吸い込まれる）
- プレビューの質感が変化

---

## ● ⑤ プレスアーム UI（Press Arm）
**役割：仕上げ（決定・保存）**

- 大型の白金属レバー
- 下げると工場奥のプレスアームが降りる
- 完成ラインの光が走る
- プレビューが完成形に変わる

---

# ■ やり直し（Undo）— 逆再生アニメーション

各工程には “逆再生” が存在する。

- 成形 → ローラーが逆回転
- 光 → 反射板が逆角度へ戻る
- 色 → 液体が沈み、色が吸い込まれる
- 質感 → 粒子が収束
- 仕上げ → プレスアームがゆっくり上昇

プレビューも静かに元へ戻る。

---

# ■ 工場の動き（Factory Motion）

### ● 常時稼働
- 巨大輪がゆっくり回転
- ベルトが低速で動く
- 粒子が空気中で揺れる

### ● UI 操作との同期
- つまみ → 光が強くなる
- レバー → 機械がわずかに加速
- スイッチ → 粒子ラインが開閉

### ● 音
- 完全無音
- 視覚的な重さだけで表現

---

# ■ UI の色・空気層・アニメーション（共通ルール）

### ● 色
- 白金属・透明層・宝石のみ
- 発光色は禁止
- 宝石だけが彩度を持つ

### ● 空気層
- 透明度 0.0〜0.6
- 色なし
- 屈折は最小限

### ● アニメーション
- 速すぎる動き禁止
- バウンド禁止
- 影が揺れすぎる動き禁止
- すべて “静けさ” を最優先

# ■ ガイド線 UI（Guide Line UI）

ガイド線 UI は、素材を中央プレビューに「正しく置く」ための位置決めレイヤー。
工場UIとは独立した、RefineRoom の基礎機能。

---

## ● ガイド線の役割
- 素材の置き場所を示す
- 位置決めの補助
- 工場の動きとは独立した UI レイヤー
- 編集の最初のステップを支える

---

## ● ガイド線の表示タイミング
1. 左パネルで素材を選んだ瞬間 → ガイド線がフェードイン  
2. 素材を中央にドラッグすると → ガイド線が追従  
3. 素材を置いた瞬間 → ガイド線が固定  
4. 工程を開始すると → ガイド線は自動でフェードアウト

---

## ● ガイド線の見た目
- 完全無彩色（白〜薄灰）
- 影なし
- 線幅は極細（0.5〜1.0px）
- 透明度は 0.3〜0.5
- 工場の背景を邪魔しない“静かな線”

---

## ● ガイド線の種類
- 中央基準線（縦・横）
- 素材の外形に合わせた輪郭ガイド
- 配置ポイント（淡い点）
- 回転ガイド（円弧）

※ すべて必要なときだけ表示され、不要なときは自動で消える。

---

## ● ガイド線の動き（アニメーション）
- フェードイン：0.2 秒
- フェードアウト：0.2 秒
- スライド・バウンドは禁止
- 空気層の屈折は最小限（0.1 秒以内）

---

## ● 素材との連動
- 素材をドラッグすると、ガイド線が“吸い付くように”位置補正
- 素材が中央に近づくと、ガイド線が淡く強調
- 素材を置くと、ガイド線が静かに固定される

---

## ● 工場との関係
- ガイド線は工場の動きに影響されない
- 工場の光・粒子・動きがガイド線を揺らすことはない
- 工場UI（右パネル）を操作するとガイド線は自動で消える

---

## ● ガイド線が消える条件
- 工程（形・光・色・質感・仕上げ）を開始したとき
- プレスアームで仕上げを行ったとき
- 素材を変更したとき（新しいガイド線が生成される）

---

## ● ガイド線 UI の目的
- 素材の位置決めを正確にする
- 工場UIの操作に集中できるようにする
- RefineRoom の“静けさ”を壊さない補助レイヤーとして機能する

# ■ 工程切り替え UI（Process Switch UI）

工程切り替え UI は、RefineRoom の左パネルで
「形 / 光 / 色 / 質感 / 仕上げ」の 5 工程を選択するための操作部。
選択された工程に応じて、右パネルの工場機械UIが切り替わる。

---

## ● 工程一覧（5つ）
- 形（Shape）
- 光（Light）
- 色（Color）
- 質感（Texture）
- 仕上げ（Finish）

---

## ● ボタンの見た目
- 白金属の小さなプレート状ボタン
- 影なし
- 透明層の上に浮いているような配置
- 選択中の工程だけ淡く光る（0.15 秒フェード）

---

## ● ボタンの動き（アニメーション）
- 押した瞬間：淡い光が内側に流れる（0.1 秒）
- 選択状態：白金属の縁がわずかに明るくなる
- 切り替え時：他のボタンは光を失い静かに戻る
- バウンド・跳ね返りは禁止

---

## ● 工場との連動（ここが重要）
工程を選ぶと、工場の対応ラインが“起動”する。

- 形 → 成形ラインのローラーが低速回転を開始  
- 光 → 反射板ラインの光がスッと走る  
- 色 → 色タンクの液体がわずかに揺れる  
- 質感 → 粒子ラインが淡く開く  
- 仕上げ → プレスアームが待機位置に移動  

※ すべて静かで重い動き。音はない。

---

## ● 右パネルの機械UI切り替え
工程を選ぶと、右パネルの工場UIが即時切り替わる。

- 形 → 成形ローラー UI  
- 光 → 光反射板 UI  
- 色 → 色タンク UI  
- 質感 → 微粒子ライン UI  
- 仕上げ → プレスアーム UI  

切り替えはフェード 0.15 秒。スライドは禁止。

---

## ● ガイド線との関係
- 素材を置いた直後はガイド線が表示されている
- 工程を選んだ瞬間、ガイド線はフェードアウト（0.2 秒）
- 工程切り替え後はガイド線は再表示されない

---

## ● 工程切り替え UI の目的
- 編集の流れを明確にする
- 工場UIを“必要なものだけ”表示する
- 画面を散らさず、静けさを保つ
- RefineRoom の操作を段階的に整理する

# ■ 工場UIと編集画面の同期ルール（Factory Sync Rules）

RefineRoom では、ユーザーの操作（UI）と工場の動き（Factory）が
常にリアルタイムで同期する。  
同期は「強すぎず・弱すぎず・静かで重い」ことを最優先とする。

---

## ● 同期の基本原則
- UI の操作は工場の“動きの強さ”に影響する
- 工場の動きは UI を揺らさない（空気層が緩衝材）
- プレビューは工場の出力として即時反映
- 音は一切発生しない
- すべての反応は 0.1〜0.2 秒以内に収束する

---

## ● ① 成形ローラー UI（形）との同期
- ダイヤル回転 → 成形ローラーが同じ速度で回転
- 回転速度に応じて加工ベルトが微加速
- 粒子がローラー方向に沿って流れる
- プレビューの形がリアルタイムで変化

---

## ● ② 光反射板 UI（光）との同期
- レバー角度 → 反射板が同角度にスッと動く
- 光ラインに淡い光が走る
- 遠景の光がわずかに強くなる
- プレビューの光の揺れが静かに変化

---

## ● ③ 色タンク UI（色）との同期
- ダイヤル回転 → タンク内の液体が揺れる
- 液体の揺れ方で色の深さ・濁りが変化
- 宝石内部の光がわずかに変わる
- プレビューの色が静かに変化

---

## ● ④ 微粒子ライン UI（質感）との同期
- スイッチ ON → 粒子ラインが開き、粒子が舞う
- スイッチ OFF → 粒子が吸い込まれる
- 粒子の密度が質感に反映される
- プレビューの表面が変化

---

## ● ⑤ プレスアーム UI（仕上げ）との同期
- レバーを下げる → プレスアームがゆっくり降りる
- 完成ラインの光が走る
- 粒子が静かに収束
- プレビューが完成形に変わる

---

## ● プレビューとの同期（共通）
- UI 操作 → プレビューが 0.1 秒以内に追従
- フェード切り替えは 0.15 秒
- 過剰な光・影の変化は禁止
- プレビューは常に“静かな器”として振る舞う

---

## ● ガイド線との同期
- 素材を置く → ガイド線が固定
- 工程を開始 → ガイド線がフェードアウト（0.2 秒）
- 工場UI操作中はガイド線は再表示されない

---

## ● Undo（逆再生）との同期
- 成形 → ローラー逆回転
- 光 → 反射板が逆角度へ戻る
- 色 → 液体が沈み色が吸い込まれる
- 質感 → 粒子が収束
- 仕上げ → プレスアームが上昇

プレビューも同時に逆再生される。

---

## ● 同期ルールの目的
- UI と工場を“ひとつの装置”として感じさせる
- 操作に手ごたえを与える
- RefineRoom の静けさと精密さを保つ

# ■ RefineRoom 全体フロー（Flow Overview）

RefineRoom の編集体験は、以下の 5 ステップで構成される。
すべてのステップは工場（Factory）と同期し、静かで精密な流れを保つ。

---

## ● ① 素材選び（Material Select）
- 左パネルで素材カテゴリを選ぶ  
  - 宝石素材  
  - テクスチャ素材  
  - モチーフ素材  
- 素材一覧から 1 つ選択  
- 工場が“準備動作”として微加速  
- ガイド線 UI がフェードイン  
- 次の工程：[素材選び UI](ca://s?q=素材選び_UI_仕様)

---

## ● ② 素材配置（Placement）
- 素材を中央プレビューへドラッグ  
- ガイド線が位置補正をサポート  
- 素材を置くとガイド線が固定  
- 工場が“受け取り動作”として反応  
- 次の工程：[ガイド線 UI](ca://s?q=ガイド線_UI_仕様)

---

## ● ③ 工程選択（Process Switch）
- 左パネルで工程を選ぶ  
  - 形  
  - 光  
  - 色  
  - 質感  
  - 仕上げ  
- ガイド線がフェードアウト  
- 工場の対応ラインが起動  
- 右パネルの工場UIが切り替わる  
- 次の工程：[工程切り替え UI](ca://s?q=工程切り替え_UI_仕様)

---

## ● ④ 工場UIで調整（Factory Machine Control）
選んだ工程に応じて、右パネルに 1 つの機械UIが表示される。

- 成形ローラー（形）  
- 光反射板（光）  
- 色タンク（色）  
- 微粒子ライン（質感）  
- プレスアーム（仕上げ）  

操作すると工場がリアルタイムで反応し、  
プレビューが 0.1 秒以内に追従する。

- 次の工程：[工場同期ルール](ca://s?q=工場同期_詳細仕様)

---

## ● ⑤ 仕上げ・保存（Finish & Save）
- プレスアーム UI を下げると仕上げが実行  
- 工場の完成ラインが光る  
- 粒子が静かに収束  
- プレビューが完成形に変わる  
- 完成品はジュエリールームへ保存  

---

## ● Undo（逆再生）
どの工程でも、Undo を行うと工場が逆再生し、  
プレビューも静かに元へ戻る。

- 成形 → ローラー逆回転  
- 光 → 反射板が逆角度へ戻る  
- 色 → 液体が沈む  
- 質感 → 粒子が収束  
- 仕上げ → プレスアーム上昇  

---

## ● Flow Summary（まとめ）

1. 素材選び  
2. 素材配置（ガイド線）  
3. 工程選択  
4. 工場UIで調整  
5. 仕上げ・保存  
6. Undo（任意）

RefineRoom のすべての操作は、  
“静けさ・精密さ・工場との同期” を最優先に設計されている。
