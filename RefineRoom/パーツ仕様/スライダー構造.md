# スライダー構造仕様
GRUMA世界の UI における「スライダー」の構造と素材仕様を定義する。
目的は 繊細で儀式具のような操作感 と 出っ張り感のない美しい収まり を実現すること。

## 1. 基本思想（Concept）
スライダーは「量の調整」を行う UI だが、
GRUMAでは単なる調整ツールではなく、
工芸品としての存在感 を持たせる。

そのため：

レールは真鍮で統一

つまみは陶器＋真鍮台座

パネルは灰白（セミマット）

全体は“静かで上品な操作具”として成立させる

## 2. スライダーの構成要素（Components）
真鍮レール（Brass Rail）

真鍮台座（Brass Base）

陶器つまみ（Ceramic Knob）

灰白パネル（Gray‑White Panel）

この4つの素材が一体となってスライダーを構成する。

## 3. 真鍮レール仕様（Brass Rail）
● 寸法
高さ：1〜1.5mm

幅：2〜3mm

長さ：用途に応じて可変

角：丸め処理（シャープ禁止）

● 質感
光沢：控えめ（セミグロス）

色味：柔らかい金色（黄味60% / 赤味40%）

表面：密度高め、滑らか

● 役割
つまみの“導線”

UI全体の統一感を作る

出っ張り感を抑えた繊細なラインとして機能

## 4. 真鍮台座仕様（Brass Base）
● 寸法
高さ：1.5〜2.0mm

直径：つまみより +1.5〜2mm

角：丸め処理

● 役割
つまみの根元を隠す

出っ張り感を半分に抑える

陶器つまみを“工芸品”として見せる

レールとの素材統一を作る

## 5. 陶器つまみ仕様（Ceramic Knob）
● 特徴
柄入り陶器（青・白・緑など）

表面は艶あり（陶器特有の光沢）

真鍮台座の中心に据え付ける

● 収まり
台座の上に“乗る”のではなく
台座に“据えられている”ように見えることが重要

## 6. パネルとの関係（Panel Interaction）
● 灰白パネル
セミマット（艶20〜35%）

青み寄りの灰白

真鍮の光を柔らかく受け止める

● 凹み構造
レールと台座が“浮かない”ように
パネル側に 浅い凹み（1mm程度） を作る

これにより、全体が落ち着いて見える

## 7. 視覚効果（Visual Effect）
スライダー全体は以下の効果を持つ：

出っ張り感が少ない

真鍮の細いラインが儀式具のように見える

陶器つまみが工芸品として成立

灰白パネルの静けさと調和

UI全体が“高級家具”のような雰囲気になる

## 8. 禁止事項（NG Rules）
レールを太くする（主張が強くなる）

台座を厚くする（重く見える）

つまみを直接パネルに置く（雑貨感が出る）

真鍮の光沢を強くする（ギラつき → 安っぽい）

レールの角を尖らせる（世界観に合わない）

## 9. 目的（Design Purpose）
スライダー構造の目的は
「繊細・静か・儀式的」な操作感を作ること。

真鍮・陶器・灰白の3素材が
互いを引き立てながら、
出っ張り感のない美しい UI を実現する。

✔ 最短まとめ
レールは真鍮（細く・低く・丸く）

つまみは陶器＋真鍮台座

パネルは灰白（セミマット）

出っ張り感を抑えるために凹み構造を使う

全体が儀式具のように静かで上品になる
