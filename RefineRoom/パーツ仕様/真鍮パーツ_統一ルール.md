# 真鍮パーツの統一ルール
GRUMA世界の UI パネルに使用する「真鍮パーツ」の統一仕様。
つまみ・レール・縁取りなど、金属要素を一貫した質感でまとめる。

## 1. 真鍮の基本質感（Brass Base Quality）
色味：柔らかい金色（黄味 60%、赤味 40%）

光沢：控えめな金属光沢（セミグロス）

表面：滑らかで密度が高い

経年感：基本は無し（儀式具としての清潔感を優先）

目的：
灰白パネルの静けさと、陶器つまみの柔らかさを引き立てるため。

## 2. 真鍮パーツの役割（Role of Brass Parts）
真鍮は UI 全体の「統一感」と「収まり」を作る素材として扱う。

つまみ台座：出っ張り感を抑える“受け皿”

スライダーレール：細く・低く・控えめな導線

パネル縁のライン：全体を引き締める額縁効果

レバーの根元：陶器や木との境界を美しく処理

## 3. 出っ張り感を抑えるための寸法（Anti‑Bulge Design）
やっちゃんの「ゴロンとするのが嫌」を解決するための基準。

● つまみ台座
高さ：1.5〜2.0mm

幅：つまみ直径より +1.5〜2mm

目的：つまみの根元を隠し、安定感を出す

● スライダーレール
高さ：1〜1.5mm

幅：2〜3mm

角：丸める（シャープ禁止）

目的：レールが“主張しない”ようにする

● パネル縁のライン
太さ：0.8〜1.2mm

位置：パネル端から 2〜3mm 内側

目的：視覚的な“収まり”を作る

## 4. 真鍮パーツの統一ルール（Unified Brass Rules）
すべての真鍮パーツは 同じ色味・同じ光沢 を使う

つまみ台座・レール・縁取りは 同じ金属系統

角はすべて 丸め処理（安っぽさ防止）

厚みは 薄く・繊細に（重くしない）

灰白パネルの上で“浮かず・沈まず”のバランスを取る

## 5. 真鍮と他素材の関係（Material Harmony）
● 灰白パネル
真鍮の光を柔らかく受け止める

高級家具のような落ち着いた雰囲気になる

● 陶器つまみ
真鍮台座があることで“雑貨感”が消える

工芸品としての存在感が出る

● ガラス・石・木
真鍮が“境界”として機能し、素材同士の喧嘩を防ぐ

## 6. 禁止事項（NG Rules）
真鍮の光沢を強くしすぎる（ギラつき → 安っぽい）

台座を厚くする（出っ張り感が増える）

レールを太くする（主張が強くなる）

縁取りを二重にする（うるさくなる）

色味を変える（統一感が崩れる）

## 7. 目的（Design Purpose）
真鍮パーツは、
「つまみを落ち着かせる」「出っ張り感を抑える」「全体を上品にまとめる」  
この3つのために存在する。

✔ 最短まとめ
真鍮は UI の“統一素材”

出っ張り感を抑える寸法が重要

灰白パネル・陶器つまみとの相性が最高

角は丸く、光沢は控えめ

世界観の“儀式具感”を作る中心素材
