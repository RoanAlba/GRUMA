# つまみの収まり構造
GRUMA世界の UI パネルにおける「陶器つまみ」の取り付け構造を定義する。
目的は 出っ張り感を抑え、工芸品としての落ち着いた存在感を作ること。

## 1. 基本思想（Concept）
陶器つまみは、そのままパネルに置くと

ゴロンと“乗っているだけ”に見える

根元がむき出しで安っぽくなる

出っ張りが強く、UI全体の静けさを壊す

これを避けるため、
真鍮の台座（受け皿）を必ず介して取り付ける。

## 2. 真鍮台座の役割（Brass Base Role）
真鍮台座は、つまみの“収まり”を作るための必須パーツ。

根元を隠す

出っ張り感を半分に抑える

陶器とパネルの境界を美しく処理

儀式具のような落ち着きを与える

灰白パネルとの相性を高める

→ つまみの価値を底上げする“額縁”の役割。

## 3. 寸法仕様（Dimensions）
● 真鍮台座
高さ：1.5〜2.0mm

直径：つまみより +1.5〜2mm

角：丸め処理（シャープ禁止）

表面：セミグロス、密度高め

● つまみ本体
台座に対して“乗る”のではなく
台座の中心に“据え付けられている”ように見える ことが重要。

## 4. 取り付け構造（Mounting Structure）
● ① パネル
灰白パネル（セミマット）に、
つまみ用の 浅い凹み（1mm程度） を作る。

● ② 真鍮台座
凹みに ぴったり収まるサイズ で配置。
→ これにより、台座が“浮かず・沈まず”安定する。

● ③ 陶器つまみ
台座の中心に固定。
根元は完全に隠れる。

## 5. 視覚効果（Visual Effect）
真鍮台座を使うことで：

つまみが“工芸品”に見える

出っ張りが視覚的に半分になる

UI全体が落ち着く

灰白パネルの静けさと調和する

真鍮ライン・レールとの統一感が出る

やっちゃんが嫌がる
「ゴロンと乗ってる感じ」 が完全に消える。

## 6. 禁止事項（NG Rules）
台座なしで直接パネルに置く

台座を厚くする（重く見える）

台座をつまみより大きくしすぎる

台座の角を尖らせる

台座の光沢を強くする（ギラつき → 安っぽい）

## 7. 目的（Design Purpose）
つまみの収まり構造は、
「出っ張り感を抑える」「根元を隠す」「高級感を作る」  
この3つを満たすために存在する。

✔ 最短まとめ
つまみは必ず真鍮台座に乗せる

台座は“受け皿”として出っ張り感を抑える

パネルに浅い凹みを作り、台座を収める

根元が隠れて工芸品のような雰囲気になる

灰白パネル・真鍮ラインと統一感が出る
