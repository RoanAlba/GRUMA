
# FactoryRoom フォルダについて

このフォルダは、GRUMA の「FactoryRoom（工場）」に関するすべての仕様をまとめた場所です。  
RefineRoom の操作を受けて、作品の形・光・色・質感を実際に加工する工程の仕様が入っています。

## 含まれるファイル

- **FactoryRoom_総合仕様.md**  
  FactoryRoom 全体の親ファイル。構造・目的・工程ラインの全体像をまとめています。

- **FactoryRoom_RefineRoom_総合関係図.md**  
  RefineRoom と FactoryRoom のデータの流れ・儀式・世界観の関係をまとめた図。

- **成形ライン_内部構造.md**  
  形状を作る工程の詳細。

- **光反射ライン_内部構造.md**  
  光の反射・ハイライト・光量を処理する工程の詳細。

- **色処理ライン_内部構造.md**  
  色相・彩度・明度・深みを処理する工程の詳細。

- **微粒子ライン_内部構造.md**  
  質感・凹凸・表面深度を処理する工程の詳細。

- **仕上げライン_内部構造.md**  
  全ラインの統合、保存・出品の最終工程の詳細。

---

FactoryRoom は GRUMA の「身体」を作る場所です。  
RefineRoom の操作はすべてここで実体化されます。
