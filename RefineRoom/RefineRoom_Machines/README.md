# RefineRoom_Machines

RefineRoom_Machines は GRUMA における  
「作品の実体（身体）を生成する内部機構」。  
RefineRoom（操作・魂）で行われた調整・儀式・意図を受け取り、  
形・光・色・質感として実体化する。

---

## 構成ライン（5 Lines）

- 成形ライン（Shape Line）  
  形状・厚み・曲率・内部構造を生成  
- 光反射ライン（Light Line）  
  光量・反射・ハイライト・屈折を処理  
- 色処理ライン（Color Line）  
  色相・彩度・明度・深みを生成  
- 微粒子ライン（Texture Line）  
  粗さ・滑らかさ・凹凸・表面深度を生成  
- 仕上げライン（Finish Line）  
  統合処理・最終光・最終質感・保存・出品儀式  

---

## データの流れ（Flow）

RefineRoom（操作）  
　↓  
RefineRoom_Machines（内部機構）  
　↓  
RefineRoom（結果表示）

操作 → 実体化 → プレビュー  
という循環で作品が生成される。

---

## RefineRoom との接続（Link）

- 素材スロット → 成形ライン  
- 光量レバー → 光反射ライン  
- 色レバー → 色処理ライン  
- 質感レバー → 微粒子ライン  
- 保存レバー → 仕上げライン（保存処理）  
- 出品レバー → 仕上げライン（最終儀式）  
- 図面モード → 成形ライン（線図データ）  

---

## 儀式（Save / Publish）

- 保存：状態の刻印、カード生成、プレビュー +2% 明るく  
- 出品：不可逆固定、完了カード、プレビューが深い呼吸  

---

## 学習システム（Learning System）

RefineRoom の操作ログを吸収し、  
形・光・色・質感の傾向を最適化する。  
時間とともに作品が “自然に良く見える” 方向へ進化。
