# RefineRoom_Machines フォルダについて

このフォルダは、GRUMA の **RefineRoom（右画面）内部に存在する「機械群（Machines）」** に関するすべての仕様をまとめた場所です。

RefineRoom で行われる操作（形・光・色・質感の調整）は、すべてこの内部機構によって実体化されます。  
つまり、RefineRoom_Machines は **作品の身体を実際に生成・加工する中枢部** です。

---

## 含まれるファイル

### RefineRoom_Machines_総合仕様.md
RefineRoom 内部の機械群の全体像・目的・工程ラインの構造をまとめた親ファイル。

### RefineRoom_Machines_総合関係図.md
RefineRoom と内部機構（Machines）のデータの流れ・儀式・世界観的な関係をまとめた図。

### 成形ライン_内部構造.md
形状を生成・変形する工程の詳細。

### 光反射ライン_内部構造.md
光の反射・ハイライト・光量を処理する工程の詳細。

### 色処理ライン_内部構造.md
色相・彩度・明度・深みを処理する工程の詳細。

### 微粒子ライン_内部構造.md
質感・凹凸・表面深度・粒子表現を処理する工程の詳細。

### 仕上げライン_内部構造.md
全ラインの統合、最終的な仕上げ、保存・出品に至る工程の詳細。

---

## RefineRoom_Machines の役割

RefineRoom_Machines は、GRUMA の作品における **「身体」そのものを生成する場所** です。  
RefineRoom で行われた操作・調整・儀式は、すべてこの内部機構によって実体化され、  
形・光・色・質感として作品に反映されます。

RefineRoom（右画面）＝操作・調整・儀式  
RefineRoom_Machines（内部構造）＝実体化・加工・生成

この2つが揃って、初めて GRUMA の作品は完成します。
