
## Etsy品質工程（Etsy Quality Process）

RefineRoom には、作品を Etsy で販売できる品質に整える工程が存在する。

### 1. 視認性の最終調整
- サムネイルでの視認性を高めるための調整が存在する  
- 明るさ・コントラスト・主役の見え方などを確認する  
- 調整方法や UI は未確定  

### 2. 余白の最終調整
- Etsy で好まれる余白バランスに整える工程が存在する  
- 余白量や構図の最終確認を行う  
- 具体的な基準や UI は未確定  

### 3. 色のくすみ・濁りの除去
- 作品の色をクリアに見せるための最終調整が存在する  
- くすみ・濁り・暗さの補正を行う  
- 補正方法や UI は未確定  

### 4. モックアップ整合性の確認
- モックアップ画像との整合性を確認する工程が存在する  
- 違和感や唐突感がないかをチェックする  
- 調整方法は未確定  

### 5. Etsy検索での競合差別化
- Etsy 内での競合との差別化を確認する工程が存在する  
- 差別化の基準や判断方法は未確定  

### 6. タイトル・タグ方向性の確認
- 出品時に使用するタイトルやタグの方向性を確認する工程が存在する  
- 具体的な文言やルールは未確定  

### 7. 他工程との関係
- Etsy品質工程は「マーケティング最適化工程」の後段に位置する  
- 完成確定工程の直前に行われる  
- 印象・形・光・色・質感・テクスチャー・モチーフ・マーケティング状態を参照する  

### 8. 編集画面への反映
- Etsy品質の状態はプレビュー画面に反映される  
- 反映方法の詳細仕様は未確定
