# RefineRoom 工程一覧（事実だけ版）

RefineRoom の工程は「順番が自由」である。
どの工程から開始しても正しく、進行順は固定されない。
ここに記載する順序は構造上の分類であり、実際の作業順とは関係しない。

RefineRoom には、作品を完成させるための工程が存在する。
工程は分類として並んでいるが、詳細仕様は未確定のため記述しない。

---

## 1. デザイン構築工程
作品の基礎となる要素を扱う工程。

### 1-1. 形
- 形を調整する工程が存在する  
- 操作方法や UI は未確定  

### 1-2. 光
- 光を調整する工程が存在する  
- 操作方法や UI は未確定  

### 1-3. 色
- 色を調整する工程が存在する  
- 操作方法や UI は未確定  

### 1-4. 質感
- 質感を調整する工程が存在する  
- 操作方法や UI は未確定  

### 1-5. テクスチャー
- テクスチャーを扱う工程が存在する  
- 出どころは複数方式を併用できる  
  - 素材集から読み込む方式  
  - AI生成による補完方式  
  - 外部制作素材の読み込み方式  
- 選択方法や UI は未確定  

### 1-6. モチーフ
- モチーフを扱う工程が存在する  
- 出どころは複数方式を併用できる  
  - 素材集から読み込む方式  
  - AI生成による補完方式  
  - 外部制作素材の読み込み方式  
- 選択方法や UI は未確定  

---

## 2. 印象決定工程
- 作品全体の印象を決める工程が存在する  
- 判断基準や UI は未確定  

---

## 3. マーケティング最適化工程
- 作品を市場向けに調整する工程が存在する  
- 未来カード・アンティークカードを参照する  
- 使用方法や UI は未確定  

---

## 4. デザイン微調整工程
- 全体のバランスを整える工程が存在する  
- 操作内容や UI は未確定  

---

## 5. Etsy品質工程
- Etsyで売れるための最終調整を行う工程が存在する  
- 視認性・明るさ・余白・色のくすみ除去などを扱う  
- 詳細仕様や UI は未確定  

---

## 6. 完成確定工程
- 完成を確定する工程が存在する  
- 出品時の呼吸アニメーションが発生する  
- 仕様の詳細は未確定
