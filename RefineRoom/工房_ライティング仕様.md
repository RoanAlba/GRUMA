# 工房 ライティング仕様
工房（RefineRoom）の光は、
「創作の温度」「静けさ」「素材の美しさ」 を支えるための基盤。

目的は、
“暖かい白を基調に、机だけ少し明るい”  
という世界観の光を UI として再現すること。

1. 光の基本構造（Lighting Structure）
工房の光は 2層構造。

● ① 部屋全体の光（Ambient Light）
色：暖かい白（少し黄色寄り）

色温度：3500〜3800K

明るさ：中程度

影：柔らかい

反射：弱い

空気を“温める”役割

● ② 机の上の光（Task Light）
部屋より ほんの少し明るい

影が出ない程度の弱いスポット

色温度は Ambient と同じ

机の木目とガラスプレートが美しく見える

2. 光の方向（Light Directions）
工房の光は 4方向の可変式（あなたの世界観の基本）。

● A：上からの柔らかい光（メイン）
影が薄い

作業しやすい

工房の“静けさ”を作る

● B：横からの細い光（アクセント）
宝石のカットが綺麗に見える

彫刻の深さがわかりやすい

● C：下からの淡い光（未来的）
ガラスプレートが浮いて見える

UIの透明感が増す

● D：前後からの弱い光（補助）
影をほぼ消す

図面モードと相性が良い

3. 光の色（Color）
● 基本色
暖かい白（#f7f2e9 付近）

黄色すぎない

オレンジすぎない

“木の机が最も美しく見える色”

● 色温度
3500〜3800K

冷たい白は使わない（出品部屋との対比のため）

4. 光の強さ（Intensity）
● 部屋全体
60〜70%

眩しくない

影は柔らかい

● 机の上
+10〜15% 明るい

ただし“スポットライト感”は出さない

自然に明るい

5. 光のアニメーション（Lighting Animation）
工房の光は 常に静かに呼吸している。

● 呼吸アニメーション
明るさ：+2% → 0%

時間：2.0〜2.5秒

ループ：ゆっくり

目的：工房の“生命感”

● 切り替え時
光は ふわっと 0.2秒 で変わる

点滅しない

急に明るくならない

音なし

6. UIとの連動（UI Integration）
● 保存レバー
保存時に机の光が +3% 明るくなる（0.2秒）

● 出品レバー
出品時は部屋全体が +4% → 0% の呼吸を一度だけ行う

● 素材スロット
素材が置かれた瞬間、机の光が +2% 明るくなる

● プレビュー
プレビューの呼吸と光の呼吸は同期する

7. 禁止事項（NG）
冷たい白（出品部屋との差が消える）

強いスポットライト

影が濃くなる光

点滅

色が変わる光

青い光

光源が見えるデザイン

光のアニメーションを速くする

光を UI の状態表示に使う（光は“空気”であって“通知”ではない）

8. 目的（Design Purpose）
工房の光は
「創作の温度を持つ、静かな暖かさ」  
を表現するために存在する。

暖かい白

机だけ少し明るい

柔らかい影

呼吸する光

静かな切り替え

これらによって、
工房は “作るための空気” を持つ空間になる。

✔ 最短まとめ
工房の光は 暖かい白

部屋全体は柔らかく、机だけ少し明るい

4方向の可変式

呼吸アニメーションで生命感

保存・出品・素材スロットと連動

NG：冷たい白・点滅・強い影

RefineRoom フォルダに保存する
