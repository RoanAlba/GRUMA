# リファインルームUI.md（最新版・混乱ゼロ版）
1. UIの目的
リファインルームUIは、
素材・光・質感・反応・UIパーツなど
“世界の中身を精製するためのUI”。

制作ジャンル（ジュエリー・部屋など）は扱わない。
ここは 精製専用の部屋。

2. UIの呼び出し方法
UIは 縦レバー で出す。

レバーを引く → UIが展開

レバーを戻す → UIが消える

レバーの素材は
白金属バー＋陶器つまみ＋極細刻み目。

3. UIの基本構造（CANVA式）
UIが出たときの画面構成は固定。

左：素材パネル（項目の一覧）

中央：プレビュー（結果を見る場所）

右：調整レバー（数値・質感・光などの調整）

この3カラム構造はリファインUIの“基本形”。

4. 精製項目（UIで扱う内容）
リファインルームで扱うのは “ジャンル” ではなく 精製項目。

色

光

質感

素材

反応

UIパーツ

影

ノイズ

透明度

これらは全部 “リファイン（精製）” のための項目。

5. 項目セレクター
レバーの横にある小さなセレクターで
どの項目を精製するか を選ぶ。

形の候補：

白磁ボタン

白金属スライダー

小さなアイコン

選んだ項目だけ淡く光る。

6. UIの出し方・引っ込め方の挙動
出すとき：ふわっと前に展開

消すとき：静かに溶けるように消える

工場の景観は邪魔しない

UIは半透明で存在感が薄い

UIの影は落とさない

“静けさの器”として存在する。

7. UI素材の仕様
あなたの好みを反映した素材。

白金属バー（極細刻み）

陶器のつまみ（角丸立方体・コロン）

カラフルな陶器パレット（白磁・青磁・薄桃・若草・琥珀・瑠璃）

メモリは極細ライン

8. 景観を邪魔しないルール
UIは工場の空気層を壊さない

光の反射は弱め

UIは前面に浮くが“薄い”

工場の静けさを優先する
