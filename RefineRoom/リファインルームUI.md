# リファインルームUI（外側3点セット統合版）
1. UI全体の役割
リファインルームUIは、
あなたが素材・光・質感・反応などを精製するための“操作画面”。

ここで扱うのは

レバー

つまみ

パネル
という UIの外側の操作部品。

2. レバー（バー）
UIを出したり消したりする“軸”となる部分。

● 素材
白金属

表面に 極細の刻み目

触るとひんやり

工場の世界観に合う静かな質感

● 役割
レバーを引く → UIが展開

レバーを戻す → UIが消える

● デザインの意図
主張しすぎず、背景に溶ける

つまみ（陶器）との対比で美しく見える

3. つまみ（陶器）
あなたが一番こだわった部分。
レバーの“手で触る部分”。

● 形
角丸立方体（コロン）

少し大きめでつまみやすい

● 素材
陶器

つるっとした光沢

ほんのり丸みのある反射

● 色のバリエーション
白磁

青磁

薄桃

若草

琥珀

瑠璃

● デザインの意図
工場の硬質な空気の中で
“柔らかいアクセント” になる

触覚的に気持ちよく、操作の中心になる

4. パネル（左側の素材パネル）
UIが展開したときに左側に出る、
項目一覧を表示するパネル。

● 配置
CANVA式の 左カラム固定

● 素材・見た目
枠は 白金属の細い縁

中身は 淡い陶器色のパレット

半透明で、工場の景観を邪魔しない

UIの影は落とさない（静けさを守る）

● 役割
精製項目（色・光・質感など）を一覧で表示

選択すると右側の調整レバーが変わる

5. UI全体の挙動
レバーを引くと ふわっと前に展開

レバーを戻すと 静かに溶けるように消える

UIは常に前面に浮くが、存在感は薄い

工場の空気層を壊さない

光の反射は弱めで、静けさを優先
