# リファインルームUI（外側3点セット統合版）

## 1. UI全体の役割
素材・光・質感・反応などを精製するための操作画面。
扱うのはレバー／つまみ／パネルの外側UI。

## 2. レバー（バー）
素材：白金属、極細の刻み目、ひんやり。
役割：引くとUI展開、戻すと消える。
意図：主張しすぎず背景に溶ける。

## 3. つまみ（陶器）
形：角丸立方体（コロン）。
素材：陶器、つるっと光沢。
色：自然／青磁／薄桃／若草／琥珀／瑠璃。
意図：硬質な工場に柔らかいアクセント。

## 4. パネル（左側）
配置：CANVA式 左カラム固定。
素材：白金属の縁＋淡い陶器色の中身。
半透明、影なし。
機能：項目一覧 → 右側レバーが変わる。

## 5. UI全体の挙動
引くとふわっと展開、戻すと静かに溶ける。
前面に浮くが存在感は薄い。
光は弱め、空気層を壊さない。
