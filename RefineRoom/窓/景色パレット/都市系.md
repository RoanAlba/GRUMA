# 高層ビル群の飛行景色（GRUMA）

## 概要
この景色は、現実の都市を基盤にしながら、GRUMA の静けさと空気の層が重なった  
“存在しているかもしれない飛行世界”。  
高さは固定されず、都市の層をゆっくり滑るように移動する。  
天候・霧・光・音の変化が、そのときの “飛行の深度” を決める。

---

# 1. 高さのレイヤー

## ● 屋上が見えるビルの高さ
- 低めの高層ビルの屋上が見下ろせる  
- 空調機やヘリポートが見える  
- ガラスの反射が近い  
- 地上のザワザワは “遠い気配” として沈む  
→ 都市のリアルさが最も強い層

## ● ビルの谷間を抜ける高さ
- 左右にビルの壁が迫る  
- 光が反射して細い線になる  
- ザワザワが少しだけ近くなるが、うるさくはない  
- 速度はゆっくり、滑るように進む  
→ 都市の密度を感じる層

## ● 屋上が見えないほど高いビルの高さ
- 見上げても屋上が見えない  
- ガラスの壁が延々と続く  
- 霧が途中にかかり、光が縦に伸びる  
- 下の音はほぼ届かない  
→ 現実の都市の限界を超えた層

## ● すべてのビルより高く飛ぶ高さ
- 都市全体が足元に広がる  
- 道路の光が線になり、車の動きが粒になる  
- 超高層ビルの頭だけが霧の上に島のように浮かぶ  
- ザワザワは完全に下に沈む  
→ “空の層” に触れる直前の高さ

---

# 2. 天候のレイヤー（まとめ）

## ● 晴れ（クリア）
- 空が高く、深い青  
- ビルのガラスが柔らかく光を返す  
- 遠くのビルの輪郭がくっきり見える  
- 空気が軽く、飛行が最も“現実寄り”  
- ザワザワは薄い気配として下に沈む  
→ 都市の構造が最もはっきり見える層

## ● 薄霧
- ビルの輪郭が少し柔らかくなる  
- 光がわずかに滲む  
- 都市の湿度が感じられる  
- 飛行が少しゆっくりに感じられる  
→ 現実と非現実の境界が薄くなる層

## ● 中霧
- ビルの中腹が霧に沈む  
- オフィスの光がぼんやり滲む  
- 廊下の光が線のように浮く  
- 飛行速度がさらにゆっくりに感じられる  
→ “都市の湿度” が最も強く出る層

## ● 深霧
- ビルの形が輪郭だけになる  
- 光が粒になって沈む  
- 音が完全に吸われる  
- 飛行感は “静止に近い滑り”  
→ GRUMA の静けさが最大になる層

## ● 夜（晴れ）
- 道路の光が線になる  
- 車の動きが粒になる  
- ビルの窓の光が浮かぶ  
- 空は黒ではなく “深い青”  
→ 光が主役になる層

## ● 夜（霧）
- 光が霧に吸われて柔らかく滲む  
- ビルの窓の光がぼんやり浮く  
- 道路の光が短い線になる  
- 音は完全に消える  
→ “静かな都市の夜” の極致

---

# 3. 音の扱い

## ● 霧がないとき（晴れ・クリア）
- 昼間の都会のザワザワが “気配だけ”  
- 車の音、人のざわめき、空調の低い唸り  
- すべてが遠く、薄く、下の層に沈む  

## ● 霧があるとき
- 音は完全に消える  
- 空気が布で包まれたように重くなる  
- 光だけが存在を主張する  
→ 音の切り替わりが “層を移動している” 感覚を作る

---

# 4. 光の性質
- 色は白・淡い金・淡い青  
- ガラスの反射は強すぎない  
- 夜景の光は線になり、霧の中では粒になる  
- 空は黒ではなく “深い青”  
→ リアル寄りの都市が GRUMA の空気に変換される

---

# 5. 飛行の感覚
- 速く飛ばない  
- 風切り音はない  
- 空気の抵抗もない  
- ただ “横に滑る”  
- 高さはゆっくり変動し、層を移動する  
→ 現実と非現実の境界線を滑る飛行
