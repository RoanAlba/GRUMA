# まとめ（空想系）

## ● 空想系とは
- GRUMA の空気の中で “実在していそうな空想” を扱う領域  
- スチームパンクを中心軸にしつつ、他の空想も受け入れる柔らかい箱  
- 現実系とは別レイヤーに存在し、窓を通して現れる

## ● 空想系に共通する特徴
- 彩度の低い金属・白木・灰色  
- 過剰な装飾を排した静かな造形  
- 空の色によって金属が揺れる  
- 音が少なく、動きがゆっくり  
- 霧や光の層と相性が良い

## ● 空想系の役割
- GRUMA の “非現実レイヤー” を担う  
- 現実の都市や部屋と対比されることで世界の奥行きを作る  
- スチームパンク気球を基準に、他の空想を追加していく構造
