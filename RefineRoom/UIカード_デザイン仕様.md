# UIカードのデザイン仕様
保存した UI セットを一覧表示するための「UIカード」のデザイン仕様を定義する。
目的は 静かで上品な“儀式札”のようなカード を作り、
複数の UI セットから迷わず選べるようにすること。

## 1. カードの基本形状（Shape）
形状：角丸長方形（R = 6〜8px）

厚み：極薄（2〜3px）

影：1%だけ（気配程度）

出っ張り感：ゼロ

世界観：薄い石板のような静けさ

## 2. カードの背景（Background）
色：灰白（青み 5〜10%）

艶：セミマット（20〜35%）

表面：密度高めで滑らか

パネル素材と同じ質感で統一する

目的：
一覧パネル全体が“整った棚”のように見えるため。

## 3. 真鍮の縁取り（Brass Frame）
太さ：0.8〜1.2mm

色味：柔らかい金（黄60% / 赤40%）

光沢：控えめ（セミグロス）

角：丸め処理

役割：カードを“儀式札”として成立させる

## 4. サムネイル（Thumbnail）
● 配置
カード上部に配置

横幅の 60% 程度

影は弱く、情報量は最小限

● 内容
スライダー（真鍮レール＋陶器つまみ）

パネルの色味

レバーの有無

UIの“雰囲気”が一目でわかる程度でOK

● 目的
やっちゃんが迷わない

UIセットの違いが直感でわかる

## 5. テキスト情報（Text）
表示項目：

UIセット名（例：UIセット01）

保存日時

主要構成（スライダー数など）

文字色：濃い灰（黒は使わない）

フォント：細めで静か

配置：カード下部に整列

## 6. カードの操作（Interaction）
タップ：その UI セットを呼び出す

長押し：名前変更 or 削除

選択時の反応：

真鍮の縁が 0.2秒だけふわっと光る

音は出さない

儀式具のような静かな反応

## 7. カードのサイズ（Size）
幅：180〜220px

高さ：240〜280px

一覧パネルでは 2〜3列 で並べる

スクロールは縦方向のみ

## 8. 禁止事項（NG Rules）
真鍮縁を太くする（うるさくなる）

背景を白にする（安っぽくなる）

サムネを大きくしすぎる（情報過多）

影を濃くする（出っ張り感が出る）

黒文字を使う（世界観が壊れる）

## 9. 目的（Design Purpose）
UIカードは
「静か・上品・儀式的」な一覧表示  
を実現するための部品。

灰白 × 真鍮 × 小サムネ の組み合わせで
GRUMAの世界観を崩さず、
複数の UI セットから迷わず選べるようにする。

✔ 最短まとめ
灰白の薄い板に真鍮の細縁

上に小さなサムネ

下に静かな文字

出っ張りゼロ

選択時だけ真鍮が光る

一覧パネルに並べてもゴチャつかない
