# RefineRoom 図面モード仕様
図面モードは、RefineRoom の素材を 正確な線・寸法・構造として確認するための技術モード。
目的は 「工房の空気を壊さず、図面だけが静かに浮かぶ」 という状態を作ること。

1. 図面モードの役割（Role）
素材の 形状・寸法・構造 を確認する

調整レバーの値を 正確に反映した線図 を表示する

工房モードとは異なる“技術視点”を提供

ただし世界観の静けさは維持する

出品レバー・保存レバーとは干渉しない

2. 図面モードの切り替え（Switch）
● 切り替え方法
プレビュー右上の 小さなアイコン（白金属）

角丸なし

ホバーで +3% 明るくなるだけ

クリックで図面モードに切り替わる

● 切り替えアニメーション
0.2秒でフェード

工房の光は変わらない

プレビューだけが “線図” に変わる

バウンドなし

音なし

3. 図面モードの表示（Display）
図面モードでは、プレビューが 線図（Blueprint） に変わる。

● 線の特徴
色：薄い黒（#222）

太さ：0.8〜1.0px

影なし

アンチエイリアスは弱め

線は静かで揺れない

● 背景
工房の空気色（薄い灰白）

呼吸アニメーションは 弱く維持

大理石の気配は入れない

● 表示内容
外形線

内部構造（必要な場合）

寸法線（控えめ）

中心線

角度線（必要な場合）

図面モードは RefineRoom_Machines の形状データを線図化して表示している。

4. 図面モードの UI（UI Behavior）
● 調整レバー
通常通り動く

レバー操作は 線図に即時反映

線図はアニメーションせず、静かに変わる

● 素材スロット
素材アイコンは表示される

図面モードでも素材変更は可能

● 保存レバー
図面モードでも保存可能

保存時の光は通常と同じ（+20%）

● 出品レバー
図面モードでも出品可能

出品時の儀式は通常と同じ

5. 図面モードのアニメーション（Animation）
● 切り替え時
プレビューが 0.2秒で線図に変わる

フェードのみ

スケール変化なし

光の変化なし

● レバー操作時
線図が即時変化

アニメーションなし（精密さ優先）

● 呼吸
明るさ：+1% → 0%

時間：2.5秒

工房モードより弱い呼吸

図面の邪魔をしない

6. 図面モードの状態（States）
● ① 通常（工房モード）
素材が立体で表示

呼吸は通常強度

光の反射あり

● ② 図面モード（Blueprint）
線図のみ

呼吸は弱い

光の反射なし

立体感なし

UIは最小限の変化で維持

● ③ 完了状態（Completed）
線図の呼吸がさらに浅くなる

編集不可

出品済みの印象を保つ

7. 禁止事項（NG）
図面モードで色を付ける

線を太くする

図面をポップアップ化する

図面に影を付ける

図面を回転させる

図面切り替え時の派手なアニメーション

図面モード専用の音

図面モードで UI System の情報を表示する

→ 図面モードは “静かな技術モード” であり、派手さは不要。

8. 目的（Design Purpose）
図面モードは
「工房の空気を保ったまま、素材の構造だけを静かに浮かび上がらせる」  
ために存在する。

線は細く

背景は空気

呼吸は弱く

光は変わらない

UIはそのまま

これによって、
工房の世界観を壊さずに 技術的な確認作業 が成立する。

✔ 最短まとめ
図面モードは 静かな技術モード

プレビューが線図に変わる

背景は工房の空気色

呼吸は弱く維持

レバー操作は即時反映

切り替えはフェードのみ

RefineRoom フォルダに保存する
