# 黒曜石ハンドル 最終仕様（やっちゃん最終決定版）

## 1. 全体サイズ
- 全長：7.5〜9cm
- 黒曜石：4.5〜6cm（全体の60〜65%）
- 真鍮リング（王冠）：1.2〜1.8cm（全体の15〜20%）
- 印面：1.5〜2cm（全体の20〜25%）

## 2. 黒曜石の質
- 種類：半透明寄りの黒曜石
- 特徴：黒の中に青い光がふっと浮かぶタイプ
- 質感：ガラス質で滑らか、内部に光が少し入る

## 3. ファセット仕様（宝石風ランダム）
### 3-1. 流れの方向（最重要）
- 斜め方向の流れ（確定）
- 光が斜めに走り、黒曜石の青い反射が真鍮リングにも落ちる
- 王冠ごと美しく見える唯一の流れ

### 3-2. 密度（中密度）
- 大きい面：20〜30%
- 中くらい：50%
- 小さい面：20〜30%
- 大小混在で“宝石らしい流れ”を作る
- 間延びしない、ザラザラしない

### 3-3. ファセット角度
- 浅め〜中間
- 深すぎると光が強く跳ねて王冠が負けるため避ける
- 浅めだと内部に光が入り、青い線が浮かぶ

## 4. 真鍮リング（王冠）仕様
- ギザギザ：小さく繊細（高さ2〜3mm）
- 形状：均一＋わずかな揺らぎ
- 透かし模様なし
- 真鍮の厚み：3〜4mm
- 目的：黒曜石の斜め流れと調和し、王冠ごと美しく見える

## 5. 全体の印象
- 黒曜石の青い光 → 真鍮リングの王冠 → 印面へと光が流れる
- 静かな格を持つ儀式具として成立
- 左上の王冠の“静かな格”を抽出し、完全オリジナルとして再構成
