# 出品後_自動チェック表_上級者版（正式版）

## 1. 初動チェック（24〜72時間）
- [ ] クリック数が「強い」判定  
- [ ] お気に入りが複数ついている  
- [ ] 滞在時間が十分  
- [ ] 反応速度が弱くない（弱い＝即危険判定）

→ 初動が弱い場合：未来予測％と勝率を必ず確認する

---

## 2. 未来予測％（Future Forecast）
※ 上級者基準：70％以上が必須

- [ ] 未来予測％が **70％以上**  
- [ ] 類似商品の伸びと比較して遅れていない  
- [ ] タグの強さが十分  
- [ ] 価格帯が競合とズレていない  

→ **70％未満は “危険ライン”**

---

## 3. 勝率（Win Rate）
※ 上級者基準：75％以上が必須

- [ ] 勝率が **75％以上**  
- [ ] サムネが競合より強い  
- [ ] トレンドと一致している  
- [ ] レビュー数・価格帯で負けていない  

→ **75％未満は “危険ライン”**

---

## 4. 危険ラインの判定（上級者版）
以下のうち **1つでも危険ライン** に入ったら取り下げ候補。

- 初動反応が弱い  
- 未来予測％ < 70％  
- 勝率 < 75％  

※ 初心者版の「2つ以上」ではなく、  
　 **上級者版は “1つでもアウト”**。

---

## 5. 取り下げ前の修正チェック
- [ ] サムネを改善した  
- [ ] タグを再調整した  
- [ ] タイトルを最適化した  
- [ ] 価格を微調整した  

→ 修正後、再度「初動 → 未来予測％ → 勝率」を評価する

---

## 6. 最終判断
### ● 改善した場合
- [ ] 継続  
- [ ] 7日後に再チェック  
- [ ] 30日後に最終チェック  

### ● 改善しない場合
- [ ] 取り下げ  
- [ ] RefineRoom 工程3（マーケティング最適化）へ戻す
