# GRUMA — アンティーク思想で動く UI × 工場システム

GRUMA は、  
**「アンティークの思想・色・質感」を  
現代 UI と巨大工場の構造に落とし込むプロジェクト。**

UI（RefineRoom）と工場（FactoryRoom）が  
**ひとつの装置として同期して動く** 世界をつくる。

---

# ■ GRUMA の全体構造

GRUMA は 2 つの部屋で構成される。

## 1. RefineRoom（リファインルーム）— UI 操作卓
宝石 UI を “磨き上げる” 部屋。  
光・形・色・質感を静かに調整する。

→ RefineRoom.md

---

## 2. FactoryRoom（ファクトリールーム）— 工場本体
RefineRoom の操作に応じて動く巨大工場。  
売れる確率アルゴリズムと学習システムを内部で処理する中枢。

→ FactoryRoom.md

---

# ■ GRUMA の目的

- アンティーク思想を UI と工場に落とし込む  
- “静けさ・重さ・精密さ” を持つ UI を作る  
- 工場と UI をひとつの装置として統合  
- 使うほど賢くなる学習システムを持つ  
- 世界観・操作感・確率計算を完全同期させる
