# GRUMA – RefineRoom × RefineRoom_Machines 統合README（最新版）

GRUMA は「作品を生み、育て、完成させる工房世界」です。  
この README は、GRUMA の中心となる 2 つの層  
RefineRoom（操作・魂）と  
RefineRoom_Machines（実体生成・身体）  
の関係をまとめた “現行世界観の総合案内” です。

---

# 1. GRUMA の全体構造（2層構造）

GRUMA は以下の 2 層で構成されます。

- **RefineRoom（魂）**  
  操作・調整・鑑賞・プレビュー・儀式を行う部屋  
  「どうしたいか」を決める場所

- **RefineRoom_Machines（身体）**  
  RefineRoom の操作を受け取り、  
  形・光・色・質感を実体として生成する内部機構  
  5つのラインが階層的に動く巨大な機構

RefineRoom が “魂”、  
RefineRoom_Machines が “身体” を担当します。

---

# 2. RefineRoom（操作・魂）

RefineRoom は作品の方向性・意図・儀式を司る部屋。

## ● 主な構成
- 素材スロット  
- 調整レバー（形・光・色・質感）  
- 図面モード（線図の確認）  
- 保存レバー（軽い決意）  
- 出品レバー（最終決断）  
- プレビュー（呼吸アニメーション）

## ● 役割
- 「どうしたいか」を決める  
- 儀式として操作を行う  
- 結果を静かに鑑賞する  
- 作品の魂を整える

RefineRoom の操作内容はすべて  
**RefineRoom_Machines に送られ、実体化されます。**

---

# 3. RefineRoom_Machines（実体生成・身体）

RefineRoom_Machines は、作品の “身体” を生成する内部機構。

## ● 5つの構成ライン（5 Lines）
- **成形ライン（Shape Line）**  
  形状・厚み・曲率・内部構造を生成

- **光反射ライン（Light Line）**  
  光量・反射・ハイライト・屈折を処理

- **色処理ライン（Color Line）**  
  色相・彩度・明度・深みを生成

- **微粒子ライン（Texture Line）**  
  粗さ・滑らかさ・凹凸・表面深度を生成

- **仕上げライン（Finish Line）**  
  統合処理・最終光・最終質感・保存・出品儀式

## ● データの流れ
RefineRoom（操作）  
　↓  
RefineRoom_Machines（実体生成）  
　↓  
RefineRoom（結果表示）

操作 → 実体化 → プレビュー  
という循環で作品が生成されます。

## ● 図面モードとの関係
図面モードの線図データは  
**成形ラインの形状情報を元に生成** されます。

---

# 4. 儀式（Save / Publish）

- **保存**  
  状態の刻印、カード生成、プレビューが +2% 明るくなる

- **出品**  
  不可逆固定、完了カード、プレビューが深い呼吸になる

儀式は GRUMA の “決断の装置” です。

---

RefineRoom の UI は
