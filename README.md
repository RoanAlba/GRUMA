# GRUMA – RefineRoom × FactoryRoom 統合README

GRUMA は「作品を生み、育て、完成させる工房世界」です。  
この README は、GRUMA の中心となる 2つの部屋  
**RefineRoom（操作の部屋）** と  
**FactoryRoom（加工の部屋）**  
の関係をまとめた “入口の案内板” です。

---

# 1. GRUMA の全体構造

GRUMA は以下の2つの部屋で構成されます。

- **RefineRoom**：操作・調整・儀式具・プレビュー  
- **FactoryRoom**：形・光・色・質感の加工、保存・出品の最終処理  

RefineRoom が「魂」、FactoryRoom が「身体」を担当します。

---

# 2. RefineRoom（操作の部屋）

RefineRoom は作品を「どうしたいか」を決める部屋です。

- 素材スロット  
- 調整レバー（形・光・色・質感）  
- 図面モード  
- 保存レバー  
- 出品レバー  
- プレビュー（呼吸アニメーション）  

RefineRoom の操作はすべて FactoryRoom に送られ、  
作品の実体として加工されます。

---

# 3. FactoryRoom（加工の部屋）

FactoryRoom は RefineRoom の操作を受けて  
作品の実体を作る巨大工場です。

5つの工程ラインで構成されます：

1. 成形ライン  
2. 光反射ライン  
3. 色処理ライン  
4. 微粒子ライン  
5. 仕上げライン  

最後に保存・出品の儀式が行われ、  
作品は完成します。

---

# 4. データの流れ（RefineRoom → FactoryRoom → RefineRoom）

1. **RefineRoom**  
　操作（素材・レバー・図面モード）  
　　　↓  
2. **FactoryRoom**  
　5ラインで加工（形 → 光 → 色 → 質感 → 仕上げ）  
　　　↓  
3. **RefineRoom**  
　プレビュー更新・カード生成・儀式の反映  

この循環によって作品は成長します。

---

# 5. 儀式具（保存・出品）

### ● 保存レバー  
- FactoryRoom が現在の状態を記録  
- カード生成  
- プレビューが +2% 明るくなる  

### ● 出品レバー  
- FactoryRoom が最終儀式を実行  
- 完了カード生成  
- 画面が 25% 暗転  
- プレビューが深い呼吸（+4% → 0%）  

保存＝状態の刻印  
出品＝不可逆の完成儀式

---

# 6. 図面モード

RefineRoom の図面モードは  
FactoryRoom の内部構造（形状・寸法・内部線）を  
線図として表示します。

図面モード中は光・色・質感が弱まり、  
線図が最優先されます。

---

# 7. 学習システム

RefineRoom の操作ログは  
FactoryRoom の学習システムに蓄積されます。

- 形の傾向  
- 光の好み  
- 色の傾向  
- 質感の傾向  
- 最終工程の成功率  

時間とともに作品が“自然に良く見える方向”へ最適化されます。
