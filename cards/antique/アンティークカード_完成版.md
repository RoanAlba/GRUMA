# アンティークカード（ウォルナット背景版）完成仕様書

## 1. カード構造
- 背景：ウォルナット（縦木目・低コントラスト・暖色寄り）
- 紙：アンティーク紙（静域仕様）
- 影：紙の縁に0.3〜0.5pxの自然影
- 情報レイヤー：タイトル／番号／矢印（静のUI）

## 2. レイヤー順序（固定済み）
1. ウォルナット背景
2. アンティーク紙
3. 紙の影
4. 情報レイヤー（タイトル・番号・矢印）

## 3. 背景仕様（ウォルナット）
- 木目方向：縦
- 色温度：紙に合わせて暖色寄り
- コントラスト：低め
- 紙が沈まず浮きすぎない“展示物の静けさ”を確保

## 4. アンティーク紙の仕様
- 紙の縁に薄い影を付与
- 木目の凹凸に自然に馴染む影
- 紙の中央は完全な静域
- 読みやすさを最優先した明度調整

## 5. 情報レイヤー（静のUI）
- タイトル：紙上部に控えめに配置
- 番号：小さく端に寄せる
- 矢印：細線で紙の雰囲気を壊さない
- 全体として“揺らぎゼロ”の静的デザイン

## 6. 書き出し構造（完成）
- レイヤーはすべて統合済み
- 境界・透明度の整合性チェック済み
- ファイルとして扱える構造に変換済み

## 7. 備考
- 未来カード（浅瀬光 × 金属）との世界観衝突なし
- GRUMAカードの「静と動」の二面性が成立
