# GRUMA UI 商品制作方針（保存版）

## 1. 文字なしで制作する
- 用途を限定しないため
- ユーザーが自由に文字を入れられる
- 多言語対応が自然にできる

## 2. 高単価商品を目指す
- リアル質感を中心にする
- 白素材 × 白金の高級感
- セット販売で単価を上げる
- UI全体の統一感を重視する

## 3. リアル質感で統一する
- 白バーチ（太い木目）
- 白ガラス（透明感）
- 白石膏（マット）
- 白セラミック（弱光沢）
- 白金の縁（細い金属ライン）

## 4. 選べる種類を多くする
- 素材
- 色
- 形
- 光の有無
- 配置
- バリエーションを増やし、選択の自由度を高める

## 5. UIの世界観
- 静かで儀式的
- 余白が広い
- 動きは最小限
- 丸（つまみ）と直線（パネル）の対比
- 白素材を中心とした落ち着いた空気感

## 6. 売れる確率をつける仕組み
- 素材ごとに売れやすさを評価
- ボタンごとに売れやすさを評価
- パネルごとに売れやすさを評価
- セットごとに売れやすさを評価
- 市場傾向・汎用性・質感を基準にスコア化する

## 7. GitHubで管理する目的
- 制作方針の固定
- 商品制作の基準化
- 後から見返せるようにする
- GRUMA全体の品質を統一する
