# Factory Library — 素材管理システム仕様書
# （Factory_UI の左パネルに相当する領域）

## 1. Library の役割
Factory Library は、Factory（旧RefineRoom）で制作する
**デジタル商品の構成要素（＝素材）を管理する中枢システム**。

ここで扱う素材はすべて “商品を構成するための要素” であり、
テンプレート・モックアップ・背景・装飾など、
あなたが作る作品の部品として存在する。

---

## 2. Library の構造（カテゴリ）
Library は以下のカテゴリで構成される。

- **Backgrounds（背景）**
　写真背景、グラデ背景、抽象背景、透明背景など。

- **Decorations（装飾）**
　花、線、図形、フレーム、アイコン、パターンなど。

- **Shapes（図形）**
　基本図形、複合図形、マスク用シェイプ。

- **Typography（テキストプリセット）**
　見出し、サブ見出し、本文、装飾文字、タイトル用プリセット。

- **Mockup Elements（モックアップ素材）**
　スマホ枠、PC枠、紙面、額縁、立体オブジェなど。

- **UI Components（UIパーツ）**
　ボタン、カード、パネル、ウィジェットなど。

- **Effects（エフェクト素材）**
　光、粒子、ぼかし、グロー、ノイズ、反射素材など。

---

## 3. Library の操作
### ● 素材のプレビュー
- ホバーで拡大プレビュー
- クリックで詳細情報（サイズ、色、タグ）

### ● Workspace への配置
- ドラッグ＆ドロップで中央の Workspace に配置
- 配置時に自動でレイヤーが生成される

### ● タグ検索
- 「花」「金」「透明」「ガラス」「ピンク真鍮」など
- あなたの世界観に合わせたタグを自由に追加可能

### ● お気に入り登録
- よく使う素材を “★” で固定表示

---

## 4. 素材の内部データ構造
素材は以下の情報を持つ。

- **ID**（一意の識別子）
- **カテゴリ**（Backgrounds / Decorations など）
- **タグ**（pink_brass, glass, elegant など）
- **解像度 / サイズ**
- **色属性**（主要色・アクセント色）
- **使用回数**（学習システム用）
- **作成日 / 更新日**

---

## 5. Library と Factory 内部機構の連動
- Library から素材を配置すると、Factory 内部の “加工機構” が起動し、
　光・影・質感の処理が自動で行われる。
- Inspector の調整は、Library 素材の “派生レイヤー” に適用される。
- Export 時には、Library の素材情報を元に最適化が行われる。

---

## 6. Library のUIレイアウト
- 左側に縦長のパネル
- 上部にカテゴリタブ
- 下部にタグ検索バー
- 中央に素材グリッド
- ホバーで拡大プレビュー

---

## 7. Library の世界観（GRUMA仕様）
- Library は Factory の “素材倉庫” に相当する。
- 素材は透明ガラスの棚に並び、光が反射して輝いている。
- ピンク真鍮の縁取りがカテゴリごとに異なる色で光る。
- 素材を取り出すと、棚の奥で “小さな機械” が静かに動く。

---

## 8. 備考
- RefineRoom 時代の素材もそのまま Library に統合可能
- 旧ファイルは過去の記録として保持し、貼り換え不要
- 新規素材はすべて Factory Library に登録される
