# （RefineRoom の名称変更後の正式UI）

## 1. Factory の役割
Factory は、やっちゃんが作る **さまざまなジャンルのデジタル商品**  
（テンプレート、モックアップ、装飾素材、背景、UIパーツなど）  
を制作するための “工房そのもの”。

RefineRoom の名称変更であり、別部屋ではない。

---

## 2. Factory UI の全体構造
Factory UI は、以下の 4 つの領域で構成される。

- **Workspace（作業領域）**  
　商品を実際に組み立てる・配置する・編集する中心エリア。

- **Library（ライブラリ）**  
　背景、装飾、図形、テキストプリセット、モックアップ素材など  
　Factory 内で使う “素材” を一覧で管理する場所。

- **Inspector（詳細パネル）**  
　選択した要素の色・影・光・サイズ・配置・効果などを調整。

- **Export Panel（書き出しパネル）**  
　完成したデジタル商品を PNG / JPG / PDF / MP4 などで書き出す。

---

## 3. Workspace（作業領域）
- キャンバス中央に商品が表示される
- ズーム（拡大縮小）
- グリッド表示 ON/OFF
- スナップ（整列補助）
- レイヤー構造の可視化（左下に小さく表示）

---

## 4. Library（ライブラリ）
Factory 内で使う “素材” を管理する場所。  
※素材という言葉は “商品を構成する要素” の意味で、  
　あなたの世界に存在する正しい概念。

カテゴリ例：
- 背景（Backgrounds）
- 装飾（Decorations）
- 図形（Shapes）
- テキストプリセット（Typography）
- モックアップ素材（Mockup Elements）
- UIパーツ（UI Components）
- エフェクト素材（Effects）

操作：
- ドラッグ＆ドロップで Workspace に配置
- プレビュー表示
- タグ検索

---

## 5. Inspector（詳細パネル）
選択した要素の詳細設定を行う。

### 調整項目
- 色（Color）
- 影（Shadow）
- 光（Light）
- 透明度（Opacity）
- サイズ（Scale）
- 回転（Rotation）
- 位置（Position）
- 効果（Glow / Blur / Noise など）

---

## 6. Export Panel（書き出し）
完成したデジタル商品を出力する。

### 静止画
- PNG / JPG
- 解像度プリセット（1:1 / 4:5 / 16:9 / 9:16）
- 背景透過（ON/OFF）

### PDF
- A4 / US Letter / カスタムサイズ
- 余白設定

### 動画（必要な場合）
- MP4 / MOV
- ループ設定
- 解像度選択

---

## 7. Factory の内部処理（世界観）
- Factory は常に稼働しており、  
　Workspace の操作に応じて内部の機構が静かに動く。
- 光・影・質感の調整は、工房内部の “見えない機械” が処理している。
- 書き出し時には、工房奥の “圧縮炉” が稼働し、  
　最適化されたデータとして商品が生成される。
