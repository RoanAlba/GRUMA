# アンティーク設計の全体構造

## ■ 起点：つまみ（爪）
- 爪は主役。高く・太く・長く、石を掴む形。
- 爪の高さ＝裏側の深さを示す。

## ■ 裏側の構造
- 深い石座
- 段差の多い構造
- 光穴が丸ではない（花形・涙形）
- 細い線・開いた裏側

## ■ 裏側 → 表への変換
- 高い爪 → 深い石座の存在を表現
- 段差 → 表の立体感として現れる
- 光穴の形 → 表の影の形に反映
- 細い枠 → 裏側の繊細さを表現
- 隙間 → 空気の通り道として表に現れる

## ■ 色の思想
- 濁りを美とする
- 色ムラを揃えない
- 黒金属で沈ませる
- 白はミルク色
- 異物色（小さな青・緑）を混ぜる

## ■ 全体の連鎖
爪 → 裏側の深さ → 段差 → 影の形 → 色の濁り → 異物色 → アンティークの色思想
