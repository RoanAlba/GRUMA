# GRUMA_世界接続図.md

## 1. 全体構造（GRUMAの世界のつながり）
GRUMA世界は、以下の主要空間で構成される。

- 工房（RefineRoom）
- 出品部屋（SubmissionRoom）
- 展示室（ExhibitionRoom）
- 外界（Outside World）
- 世界観（WorldView）
- レイアウト構造（Layout Structure）

---

## 2. 接続図（テキスト版）

GRUMA/
├─ 工房（RefineRoom）
│    ├─ 制作レーン
│    ├─ 出品準備レーン
│    └─ → 出品部屋へ接続
│
├─ 出品部屋（SubmissionRoom）
│    ├─ 作品最終確認
│    ├─ 出品台座
│    ├─ 出品ゲート
│    └─ → 外界へ接続
│
├─ 展示室（ExhibitionRoom）
│    ├─ 作品展示
│    ├─ 観覧UI
│    └─ → 工房へ戻る導線
│
├─ 外界（Outside World）
│    ├─ 出品された作品が存在する領域
│    └─ → GRUMAへは戻らない（片方向）
│
├─ 世界観（WorldView）
│    ├─ 素材（白磁・灰磁・真鍮）
│    ├─ 光（白磁光）
│    └─ 各部屋へ設定を供給
│
└─ レイアウト構造（Layout Structure）
     ├─ 画面構成
     ├─ UI配置
     └─ 各部屋の位置関係を定義

---

## 3. 流れ（動線まとめ）

1. 工房で制作  
2. 出品準備レーンで出品操作  
3. 出品部屋へ遷移  
4. 出品ゲートから外界へ  
5. 展示室は工房と相互接続  
6. 世界観とレイアウト構造は全体に影響

---

## 4. 世界の性質（統合）
- 工房 → 出品部屋 → 外界 は **片方向の儀式的流れ**
- 展示室は工房と循環する **双方向の空間**
- 世界観は全空間の素材・光・静けさを統一
- レイアウト構造は全体の位置関係を支える

---

## 5. 完成イメージ（総括）
GRUMA世界は  
**「制作 → 儀式 → 旅立ち → 展示」**  
という静かな循環で構成される。

- 工房：創造  
- 出品部屋：儀式  
- 外界：旅立ち  
- 展示室：共有  
- 世界観：統一  
- レイアウト構造：骨格

これが GRUMA 世界の全接続図である。
