# 世界観フォルダ_構成（正式版 / 保存用）

## 1. 世界観フォルダの目的
GRUMA世界全体の「素材・光・空気・構造・接続」を統一し、  
各部屋（工房・出品部屋・展示室）が同じ世界として成立するための  
“基準ファイル” をまとめるフォルダ。

---

## 2. 世界観フォルダの完成構成
世界観/
├─ 世界観_素材.md
├─ 世界観_光.md
├─ 世界観_空気感.md
├─ 世界観_構造.md
└─ GRUMA_世界接続図.md

---

## 3. 各ファイルの役割
- **世界観_素材.md**  
  → 白磁・灰磁・真鍮・透明素材など、世界の物質ルール

- **世界観_光.md**  
  → 白磁光の性質、色温度、影の濃度、光源の扱い

- **世界観_空気感.md**  
  → 静けさ、余白、音の扱い、世界の雰囲気の基準

- **世界観_構造.md**  
  → GRUMA世界の階層構造・空間の分類・世界の骨格

- **GRUMA_世界接続図.md**  
  → 工房・出品部屋・展示室・外界のつながりを示す図

---

## 4. 世界観フォルダの運用ルール
- 各部屋の仕様はここに置かない  
- 世界の“基準”だけを置く  
- 部屋ごとの詳細は各部屋フォルダに置く  
- 世界観は「全体の辞書」になるように保つ
