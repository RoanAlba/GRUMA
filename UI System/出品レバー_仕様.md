# 出品レバー（大レバー）仕様
作品を「本番に出品する」ための最終儀式としての大レバーの仕様を定義する。
目的は 重み・緊張感・儀式性 を最大限に高め、
GRUMA の世界観における“最終決断”を象徴する装置とすること。

## 1. レバーの役割（Role）
UI セットを 本番に出品する最終操作

保存レバーとは完全に別の行為

「軽い保存」ではなく “決断” の儀式

出品後は取り消し不可（世界観上の重み）

## 2. レバーの形状（Shape）
種類：大型 T型レバー

全長：85〜110mm

つまみ：黒マット革（保存レバーより大きい）

軸：真鍮（太め・存在感あり）

根本：真鍮の厚いリング（儀式具の台座）

厚み：しっかりある（存在感を出す）

→ “工房の中心にある儀式具” の雰囲気。

## 3. 素材（Material）
軸：真鍮（黄55% / 赤45%）

つまみ：黒マット革（艶 5〜10%）

根本：重厚な真鍮台座

表面：セミマット（ギラつき禁止）

→ 保存レバーより重厚で、世界観の中心。

## 4. 動き（Motion）
動作：倒す（押すでも引くでもない）

可動角度：40〜55°

重さ：しっかり重い（保存レバーの 2.5〜3倍）

戻り：自動では戻らない

倒したまま 0.5秒固定 → 自動でゆっくり戻る（1.2秒）

→ “決断の重さ” を身体で感じる動き。

## 5. 反応（Interaction）
● 倒した瞬間
真鍮台座が 0.3秒だけ深く光る

画面が一瞬だけ暗転（儀式感）

出品処理が開始される

音は出さない（静けさ重視）

● 戻るとき
光がゆっくり消える

UI が出品完了状態に切り替わる

## 6. 配置（Placement）
編集画面：右下（保存レバーより大きく、離して配置）

私用UI：表示しない

画面端からの余白：40〜48px

他の UI と絶対に干渉しない位置

→ “最後に触れる場所” としての存在感。

## 7. 保存レバーとの関係（Relation）
保存レバー：軽い決意

出品レバー：最終決断

形・大きさ・重さ・光り方が明確に違う

誤操作を防ぐため、距離を離して配置

## 8. 禁止事項（NG Rules）
保存レバーと同じ大きさにする

ボタン式にする（儀式性が消える）

軽い動きにする（重みが消える）

強い光沢（安っぽくなる）

音を鳴らす（世界観が壊れる）

## 9. 目的（Design Purpose）
出品レバーは
「作品を世界に送り出すための最終儀式」  
として存在する。

重さ・光・動きのすべてが
“決断の瞬間” を演出し、
GRUMA の世界観の中心となる。

✔ 最短まとめ
大型 T型レバー

黒マット革 × 重厚真鍮

重い倒し動作

光は深く 0.3秒

保存レバーとは完全に別物

UI System フォルダに保存する
