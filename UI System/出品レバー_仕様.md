# 出品レバー 仕様
出品レバーは、完成した作品を 外の世界へ送り出すための最終儀式具。
目的は 決断の重み・儀式性・世界観の中心性 を UI に落とし込むこと。

1. レバーの役割（Role）
作品を 正式に出品する

保存レバーとは違い、最終決断 を扱う

誤操作を絶対に許さない

世界観の“儀式の中心”として存在する

出品後は作品が 不可逆的に確定 する

2. レバーの形状（Shape）
● 大型T型レバー
保存レバーより 明確に大きい。
儀式具としての存在感を持つ。

● 素材
本体：重厚な金（真鍮より深い金色）

持ち手：黒革（保存レバーより太く、重厚）

軸：黒鉄＋真鍮の刻印ライン

反射：保存レバーより強め（儀式性のため）

● サイズ
保存レバーの 1.3〜1.4倍

握りは両手でも片手でも扱える太さ

“重さ” を感じる見た目

3. レバーの配置（Placement）
RefineRoom の 右下の特別な台座

台座は大理石＋真鍮ライン

保存レバーとは距離を置く

「ここが最終儀式の場所」とわかる配置

4. 操作方法（Interaction）
● 倒す方向
奥から手前へ倒す（保存レバーと同じ方向）

ただし角度は深い：40〜45°

● 重み
保存レバーより明確に重い

倒すときに “静かな抵抗” を感じる

途中で手を離すとゆっくり戻る（0.4秒）

5. 出品の発動タイミング（Trigger）
出品は レバーが倒れ切った瞬間 に発動。

倒れ始めでは発動しない

戻る途中でも発動しない

倒し切った位置でのみ発動

誤操作防止のため、倒し切り位置は明確に重い

6. 出品時の儀式アニメーション（Ritual Animation）
出品レバーの核心。
保存レバーとは “格” が違う動きになる。

● ① 真鍮が強く光る
明るさ：+25〜30%

時間：0.3秒

光は保存レバーより長く残る

“決断の光”

● ② 台座の真鍮ラインが連動して光る
レバーの光が台座に伝わる

0.2秒遅れて淡く光る

世界観の“儀式回路”の表現

● ③ 画面全体が静かに暗転
透明度：0 → 25%

時間：0.25秒

保存より深い暗転

“世界が息を止める瞬間”

● ④ 出品完了の静かな息
大理石パネルが +4% → 0%

時間：0.25秒

呼吸が一度だけ起きる

音はなし

● ⑤ 一覧パネルに出品済みマークが付く
UIカードに “完了状態” が反映

完了カードは光らない・呼吸浅い

選択不可になる

7. 出品後の状態（After Publish）
レバーはゆっくり戻る（0.4秒）

真鍮の光が消える

工房の空気が静かに戻る

UIカードは 完了状態 に固定

完了カードは編集不可

完了カードは一覧パネルの下に寄る

8. 禁止事項（NG Rules）
ボタン式の出品

レバーを軽くする

派手な光

音

ポップアップ

出品後にアニメーションを連続させる

出品を取り消せるようにする

完了カードを再編集可能にする

→ 出品は “不可逆の儀式” であるため。

9. 目的（Design Purpose）
出品レバーは
「作品を世界へ送り出すための最終儀式」  
として存在する。

保存より重い

光は強い

暗転は深い

呼吸は一度だけ

完了は不可逆

このすべてが、
GRUMA の世界観における “出品の重み” を表現する。

✔ 最短まとめ
出品レバーは 最終儀式具

保存レバーより大きく重い

倒し切った瞬間に出品

真鍮が強く光り、台座も光る

画面が深く暗転

完了カードに変化

出品は不可逆

UI System に保存する
