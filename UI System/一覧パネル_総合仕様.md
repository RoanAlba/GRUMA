# 一覧パネル 総合仕様
保存した UI セットを一覧表示する 一覧パネル全体の構造・動き・操作体系 を統合した仕様。
目的は 静けさ・儀式性・視認性・誤操作防止 をすべて満たす UI を作ること。

1. パネル全体のレイアウト
（詳細：レイアウト仕様）

画面中央に固定

幅：画面の 68〜72%

高さ：画面の 72〜78%

背景：灰白（大理石の気配 10〜15%）

枠：細い真鍮ライン（1.2mm）

上：ヘッダー

下：カード一覧（スクロール）

2. ヘッダー
（詳細：ヘッダー仕様）

左：タイトル「保存したUI」

右：真鍮の ×

高さ：48〜56px

下端に真鍮ライン

背景はパネルと同じ灰白

出現時のみ真鍮が一瞬光る

3. カード配置
（詳細：カード配置仕様）

1列固定

カード幅：パネルの 88〜92%

縦余白：20〜28px

並び順：新しい順

完了カードは自動で下に寄る

スクロールは弱い慣性、バウンドなし

4. UIカード（サムネ・情報欄・状態）
● サムネ
（詳細：サムネ仕様）

スライダー3本まで

パネル色の気配

アクセント1つ

情報量は最小限

● 情報欄
（詳細：情報欄仕様）

UIセット名

保存日時

左寄せ・控えめ

● 状態
（詳細：選択状態仕様）

未選択：静か

選択：真鍮が光る＋1px浮き＋四隅の光

完了：光なし・呼吸浅い

5. ホバー・クリック・削除
● ホバー
（詳細：ホバー仕様）

影が1px濃くなる

真鍮が3%明るくなる

浮かない

● クリック
（詳細：クリック動作仕様）

真鍮が光る（0.2秒）

1px浮く

四隅の光が点灯

完了カードは反応しない

● 削除
（詳細：削除動作仕様）

カード右上の真鍮点から開始

必ず確認モーダル

光らずに静かに霧散して消える

6. 並び替え禁止
（詳細：並び替え禁止仕様）

ドラッグ不可

並び替えボタンなし

並び替え設定なし

完全自動整列（新しい順）

7. アニメーション
（詳細：アニメーション仕様）

● 出現
スケール：95% → 100%（0.18秒）

透明度：0 → 100%

真鍮縁が一瞬光る

暗幕：40%

● 消失
スケール：100% → 97%（0.15秒）

透明度：100% → 0

真鍮は光らない

8. 閉じ方
（詳細：閉じ方仕様）

× ボタン

暗幕タップ

Escキー

パネルは 0.15秒で静かに消える

9. 禁止事項（NG）
2列レイアウト

派手なアニメーション

音を鳴らす

情報を増やす（検索・フィルターなど）

並び替え機能

カードの色変更

背景を濃くする

10. 目的（Design Purpose）
一覧パネルは
「静かに中央に現れ、静かに選び、静かに消える儀式の場」  
として存在する。

真鍮の額縁

灰白の静けさ

控えめな光

1列の儀式札

誤操作のない操作体系

これらすべてが GRUMA の世界観を支える。

✔ 最短まとめ
一覧パネルの 全仕様を統合した親ファイル

レイアウト／ヘッダー／カード／動き／削除／禁止事項

世界観の静けさを守るための総合設計

UI System に保存する
