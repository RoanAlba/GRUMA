
# 一覧パネルのレイアウト仕様
保存した UI セットを一覧表示するパネルの 大枠の構造・比率・余白・領域分割 を定義する。
目的は 静けさ・視認性・世界観の統一 を保ちながら、
迷わず UI セットを選べるレイアウトを作ること。

## 1. パネル全体のサイズ（Panel Size）
画面中央に固定

幅：画面幅の 68〜72%

高さ：画面高さの 72〜78%

角丸：なし（直線的な世界観）

枠：細い真鍮ライン（1.2mm）

→ 儀式具の“額縁”としての存在感。

## 2. パネルの背景（Background）
色：灰白（青み 5%）

質感：大理石の“気配”を 10〜15%

影：薄い影を 2%（浮きすぎない）

## 3. パネルの領域分割（Sections）
一覧パネルは 上：ヘッダー / 下：カード領域 の2分割。

┌──────────────────────────┐
│        ヘッダー（固定）        │  ← 高さ 48〜56px
├──────────────────────────┤
│                                │
│     カード一覧（スクロール）     │
│                                │
└──────────────────────────┘

## 4. ヘッダー領域（Header）
※ 詳細は別仕様（一覧パネル_ヘッダー仕様.md）

高さ：48〜56px

背景：パネルと同じ

下端に真鍮ライン

左にタイトル、右に ×

## 5. カード一覧領域（Card List Area）
ヘッダーの下からパネル下端まで

スクロール可能

左右余白：24〜32px

上下余白：20〜28px

→ “棚を静かに見ていく” 感覚。

## 6. 背景暗幕（Overlay）
一覧パネルが出ている間、
背後に 40% の暗幕 を敷く。

色：黒に近い灰

透明度：40%

アニメーション：0.18秒でフェードイン

→ 儀式の“場”を作る。

## 7. 出現位置（Position）
常に画面中央

スクロールしても位置は固定

呼び出しボタンの位置と干渉しない

## 8. アニメーション（Animation）
※ 詳細は別仕様（一覧パネル_アニメーション仕様.md）

出現：95% → 100%（0.18秒）

消失：100% → 97%（0.15秒）

真鍮縁が一瞬光る（出現時のみ）

## 9. 禁止事項（NG Rules）
角丸を付ける

背景を濃くする

枠線を太くする

2列レイアウトにする

パネルを画面端に寄せる

派手なアニメーション

暗幕を濃くしすぎる

→ 世界観の“静けさ”を守るため。

## 10. 目的（Design Purpose）
一覧パネルのレイアウトは
「静かに中央に現れる儀式の場」  
として存在する。

真鍮の額縁・灰白の背景・広い余白によって
GRUMA の世界観を壊さず、
UI セットを迷わず選べる構造を作る。

✔ 最短まとめ
パネルは中央固定

幅 70% / 高さ 75%

上：ヘッダー、下：カード一覧

真鍮の額縁

暗幕 40%

UI System に保存する
