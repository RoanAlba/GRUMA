# UIカードの選択状態仕様
一覧パネル内の UIカードにおける 選択・未選択・完了 の状態を定義する。
目的は 世界観を壊さず、静かに、迷わず選べる状態表示 を作ること。

## 1. 状態の種類（States）
UIカードには 3 種類の状態がある。

未選択（Default）

選択中（Selected）

完了（Completed）

それぞれの状態は「光」「呼吸」「浮き」の 3要素で表現する。

## 2. 未選択（Default）
● 見た目
真鍮縁：通常の明るさ

影：1%（気配程度）

浮き：なし

呼吸：通常（ゆっくり 3秒周期）

● 目的
一覧の中で“静かに並ぶ”状態

情報量ゼロの静けさ

## 3. 選択中（Selected）
● 見た目
四隅に極小の淡い光（1〜2px）

真鍮縁が 0.2秒だけふわっと光る（タップ時）

カードが 1px 浮く

呼吸：少し浅く（2秒周期）

● 動き
タップした瞬間：

明るさが 1%だけ上がる（0.1秒）

すぐ元に戻る

四隅の光は 常時点灯（弱い）

● 目的
“選ばれている” を静かに示す

うるさくせず、儀式具のような反応

## 4. 完了（Completed）
● 見た目
真鍮縁：少しだけ暗く（彩度 -10%）

四隅の光：消灯

カードの明るさ：-2%（控えめ）

呼吸：さらに浅く（1.5秒周期）

● 動き
浮き：なし

タップしても光らない（完了の静けさ）

● 目的
“もう触らなくていい” 空気を作る

選択中との階層差を自然に見せる

## 5. 状態遷移（Transition）
● 未選択 → 選択
真鍮縁が光る（0.2秒）

四隅の光がふわっと点灯（0.15秒）

カードが 1px 浮く

● 選択 → 未選択
四隅の光が 0.2秒でゆっくり消える

浮きが戻る

● 選択 → 完了
光がすべて消える

呼吸が浅くなる

明るさが -2%

● 完了 → 選択（基本的に不可）
完了状態は“固定”

再編集時のみ解除される

## 6. 禁止事項（NG Rules）
チェックマークを表示する

枠線を太くする

色を変えて状態を示す

強い光や点滅

浮きを 2px 以上にする

音を鳴らす

派手なアニメーション

→ 世界観の“静けさ”を守るため。

## 7. 目的（Design Purpose）
UIカードの選択状態は
「静かに灯る儀式札」  
として存在する。

光・浮き・呼吸の3つで
世界観を壊さずに状態を伝える。

✔ 最短まとめ
選択：四隅の淡い光＋1px浮き

未選択：静かに並ぶ

完了：光なし・呼吸浅い

真鍮縁はタップ時だけ光る

UI System フォルダに保存する
