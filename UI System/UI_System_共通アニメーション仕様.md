# UI System 共通アニメーション仕様
GRUMA の UI 全体に共通する 動き・光・影・速度・禁止事項 を定義する。
目的は 世界観の静けさ・儀式性・統一感 を UI の動きで保証すること。

1. アニメーションの基本原則（Principles）
動きは最小限

位置は基本的に動かさない

フェード中心（透明度の変化）

真鍮だけが光る

バウンド禁止

音なし

0.1〜0.25秒の短い儀式的な動き

“動いているのに動いていないように見える” が正解

2. 使用するアニメーションの種類（Types）
● フェード（Fade）
透明度の変化のみ

速度：0.12〜0.20秒

イージング：ease-out

UI System の基本動作

● スケール（Scale）
95% → 100%（出現）

100% → 97%（消失）

速度：0.15〜0.18秒

拡大・縮小は儀式のときだけ

● 光（Brass Glow）
真鍮部分のみ光る

明るさ：+10〜20%

速度：0.15〜0.25秒

出現時・選択時・保存時のみ使用

● 呼吸（Breathing）
明るさ：+2% → 0%

速度：1.5〜2.5秒

プレビューや工房の “静かな生命感” を表現

3. 画面全体のアニメーション（Global）
● 出現（Appear）
スケール：95% → 100%

透明度：0 → 100%

真鍮縁が一瞬光る（+15%）

速度：0.18秒

● 消失（Disappear）
スケール：100% → 97%

透明度：100% → 0

真鍮は光らない

速度：0.15秒

● 暗幕（Overlay）
出現：透明度 0 → 40%（0.18秒）

消失：40% → 0（0.15秒）

4. UIカードの共通アニメーション（Cards）
● ホバー
影：1% → 2%

真鍮：+3%

速度：0.12秒

浮かない

● クリック（選択）
真鍮：+15%

浮き：1px

四隅の光：点灯

速度：0.15〜0.20秒

● 削除
明るさ：100% → 0%

スケール：100% → 92%

透明度：100% → 0

速度：0.20〜0.25秒

真鍮は光らない

5. レバー（儀式具）のアニメーション（Levers）
● 保存レバー
倒す：手動

戻る：0.3秒

保存時：真鍮が+20%光る（0.25秒）

大理石が息をする（+3% → 0%）

● 出品レバー（別仕様）
保存レバーより重い

光は強め（+25%）

儀式性が最も強い

6. イージング（Easing）
UI System で使用するイージングは 2種類のみ。

ease-out（基本）

ease-in（閉じるとき）

これ以外は使用しない。

7. 禁止アニメーション（NG）
バウンド

スライド（横移動・縦移動）

回転

弾む動き

色が急に変わる

ポップアップの飛び出し

長いアニメーション（0.3秒以上）

音付きアニメーション

UIが跳ねる動き

角丸のアニメーション

→ 世界観の静けさを壊すため完全禁止。

8. 目的（Design Purpose）
UI System の共通アニメーションは
「静かに息づく儀式の動き」 を作るために存在する。

真鍮の光

大理石の呼吸

短いフェード

小さなスケール変化

これらを統一することで、
どの画面でも GRUMA の空気が途切れない。
