# 一覧パネルの閉じ方仕様
一覧パネルを閉じるための 操作方法・動き・反応 を定義する。
目的は 静けさ・儀式性・誤操作防止 を両立した閉じ方を作ること。

## 1. 閉じ方の種類（Close Methods）
一覧パネルの閉じ方は 3つだけ。

① ヘッダー右上の × を押す

② 暗幕（背景）をタップする

③ Escキー（PCのみ）

これ以上は増やさない。

## 2. × ボタンで閉じる（Close Button）
※ 詳細はヘッダー仕様に準拠

● 動き
タップ → 真鍮が 0.12秒だけ光る

パネルが 0.15秒でフェードアウト

スケール：100% → 97%

暗幕：40% → 0%

● 目的
最も“儀式的”な閉じ方

誤操作が少ない

## 3. 暗幕タップで閉じる（Overlay Tap）
● 動き
暗幕をタップすると

パネルが 0.15秒で静かに消える

真鍮縁は光らない（儀式ではないため）

● 条件
暗幕の透明度は 40%

暗幕のタップ範囲は全面

パネル本体はタップしても閉じない

● 目的
“軽い閉じ方”

一覧を見終わった後の自然な動作

## 4. Escキーで閉じる（PC）
● 動き
Esc → パネルが即座に閉じる

アニメーションは通常通り（0.15秒）

真鍮縁は光らない

● 目的
PC操作の利便性

ただし儀式性は弱め

## 5. 禁止事項（NG Rules）
スワイプで閉じる（世界観に合わない）

下に引っ張って閉じる（モバイルUI的でNG）

ダブルタップで閉じる

パネル内の空白タップで閉じる

閉じる操作に音を付ける

派手なアニメーション

→ GRUMA の“静けさ”を守るため。

## 6. 閉じる時のアニメーション（Animation）
※ アニメーション仕様と整合性あり

● パネル
透明度：100% → 0%（0.15秒）

スケール：100% → 97%

イージング：ease-in

● 暗幕
40% → 0%（0.15秒）

● 真鍮縁
光らせない（出現時のみの儀式）

## 7. 閉じた後の状態（After Close）
呼び出しボタンは元の状態

選択状態は保持されない（一覧を閉じたらリセット）

スクロール位置は保持しない（常に先頭から）

## 8. 目的（Design Purpose）
一覧パネルの閉じ方は
「静かに儀式を終えるための動作」  
として存在する。

× は儀式的

暗幕タップは自然

Esc は実用的

この3つで、世界観と操作性を両立させる。

✔ 最短まとめ
閉じ方は「×」「暗幕タップ」「Esc」の3つ

パネルは 0.15秒で静かに消える

真鍮は光らない

UI System に保存する
