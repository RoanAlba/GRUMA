# UIカードのホバー仕様
UIカードにカーソルを乗せたときの 最小限の反応 を定義する。
目的は 世界観の静けさを保ちながら、触れる場所を示すこと。

## 1. ホバーの役割（Role）
「ここは触れる場所だよ」と静かに示す

情報を出さない

動きを大きくしない

世界観を乱さない

## 2. ホバー時の反応（Reaction）
● ① 影が 1px だけ濃くなる
影：1% → 2%

時間：0.12秒

イージング：ease-out
→ “存在が少しだけ確定する” 程度。

● ② 真鍮縁がほんの少し明るくなる
明るさ：+3%

時間：0.12秒
→ 光らせすぎない。気配だけ。

● ③ 浮きは なし
→ 選択状態（1px浮き）と区別するため。

## 3. ホバーで起きないこと（Not Included）
情報を出さない

サムネを動かさない

色を変えない

枠線を太くしない

カードを浮かせない

アニメーションを派手にしない

音を鳴らさない

→ 世界観の静けさを守る。

## 4. ホバー解除時（Unhover）
影：2% → 1%（0.12秒）

真鍮縁：明るさ +3% → 0%

動きはすべて静かに戻る

## 5. 選択状態との関係
ホバーは「気配」

選択は「光＋1px浮き」

完了は「光なし・呼吸浅い」

→ 3つの階層が自然に区別できる。

## 6. 目的（Design Purpose）
UIカードのホバーは
「触れたときにだけ、静かに息をする儀式札」  
として存在する。

強い反応は不要。
気配だけで十分。

✔ 最短まとめ
ホバー＝マウスを乗せただけの反応

UIカードでは「影1px濃くなる＋真鍮が3%明るくなる」だけ

浮かない

情報を出さない

世界観の静けさを守る

UI System に保存する
