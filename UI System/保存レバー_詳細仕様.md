# 保存レバー 詳細仕様
保存レバーは、RefineRoom で作業した内容を UIセットとして保存するための儀式具。
目的は 軽い決意を形にする“静かな儀式” を提供すること。

1. レバーの役割（Role）
現在の UI設定を 保存 する

保存した UIセットを 一覧パネルに追加 する

操作を “ボタン” ではなく 儀式化 する

誤操作を防ぐ（押すのではなく倒す）

2. レバーの形状（Shape）
● T型レバー（T字）
上部：横棒

下部：縦の軸

握りやすい形状

決定レバーと同じ系統の儀式具

● 素材
本体：落ち着いた金（真鍮寄りのゴールド）

持ち手：黒革（マット、シボ少なめ）

反射：弱め（大理石の上で柔らかく光る）

● サイズ
出品レバーより 小さめ

高さ：出品レバーの 70〜75%

横棒の長さ：手の幅より少し短い

3. レバーの配置（Placement）
RefineRoom の右下

大理石パネルの上

真鍮ラインがレバーの根元に集まる

他の UI と干渉しない位置

4. 操作方法（Interaction）
保存レバーは 押すのではなく、倒す。

● 動き
倒す方向：手前に倒す（下方向）

倒す角度：30〜35°

倒した後、ゆっくり自動で戻る（0.3秒）

● 触感（世界観的表現）
静かな重み

金属の軋みはない

革の握りが手に馴染む

5. 保存の発動タイミング（Trigger）
保存は レバーが倒れ切った瞬間 に発動する。

倒れ始めでは発動しない

戻る途中でも発動しない

倒れ切った位置でのみ発動

→ 誤操作防止。

6. 保存時のアニメーション（Saving Ritual）
保存が発動した瞬間、以下の儀式が起きる。

● ① 真鍮縁が光る
明るさ：+20%

時間：0.25秒

イージング：ease-out
→ “保存が成立した” を示す儀式。

● ② 大理石パネルが一瞬だけ息をする
明るさ：+3% → 0%

時間：0.2秒
→ 工房が保存を受け取った表現。

● ③ UIカードが一覧パネルに追加される
一覧パネルが開いていれば最上段に追加

開いていなければ次回開いたときに反映

7. 保存後のフィードバック（Feedback）
保存後は 控えめなフィードバック のみ。

レバーがゆっくり戻る

真鍮の光が消える

工房の空気が静かに戻る

ポップアップは出さない

音は鳴らさない

8. 保存内容（Saved Data）
保存されるのは以下の内容。

スライダー位置

素材設定

光・質感の設定

選択中の素材カテゴリ

UIセット名（自動生成 or ユーザー名）

保存日時

保存されないもの：

レバーの位置

一時的な UI System の状態

一覧パネルのスクロール位置

9. 禁止事項（NG Rules）
ボタン式の保存

レバーを軽くする

レバーを光らせすぎる

保存音

保存完了ポップアップ

保存アニメーションを派手にする

レバーを連打できるようにする

保存を自動化する

→ 保存は “儀式” であり、軽い操作にしてはいけない。

10. 目的（Design Purpose）
保存レバーは
「工房に刻むための小さな儀式」  
として存在する。

ボタンではなくレバー

光と重み

静かなフィードバック

誤操作のない構造

これらによって、
保存という行為に “意味” と “重み” を与える。

✔ 最短まとめ
保存レバーは “軽い決意の儀式具”

倒し切った瞬間に保存

真鍮が光り、大理石が息をする

UIカードが一覧に追加

ポップアップなし・音なし

UI System に保存する
