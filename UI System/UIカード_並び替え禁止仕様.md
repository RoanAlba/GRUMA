## 1. 並び替え禁止の理由（Reason）
UIカードは “儀式札” であり、動かす対象ではない

自動整列（新しい順）が最も迷わない

ドラッグ操作は世界観に合わない

誤操作が起きやすい

並び替え機能は情報量を増やし、静けさを壊す

## 2. 禁止する操作（Forbidden Actions）
ユーザーは以下の操作を 一切できない。

ドラッグでカードを上下に動かす

カードを別の位置に移動する

並び替えボタンで順番を変える

完了カードを上に持ってくる

お気に入り順に並べる

手動で順番を保存する

## 3. 許可される並び（Allowed Order）
一覧パネルの並びは 完全自動。

新しい順（上に新しい）

完了カードは自動で下に寄る

削除後は静かに詰める

並び替えの設定項目は存在しない

## 4. UI上の表現（UI Behavior）
カードはドラッグしても動かない

並び替えアイコンは存在しない

並び替えメニューも存在しない

並び替えに関するヒントや説明も出さない

並び替えを示すカーソル（上下矢印）も出さない

## 5. 禁止事項（NG Rules）
並び替え機能を後から追加する

並び替えの設定を作る

並び替えのショートカットを作る

並び替えのアニメーションを作る

並び替えのヒントを表示する

## 6. 目的（Design Purpose）
UIカードの並び替え禁止は
「儀式札は動かさず、静かに並ぶ」  
という世界観を守るために存在する。

ユーザーが迷わず、
自動整列された UI セットを
静かに選べるようにする。

✔ 最短まとめ
並び替え＝順番を変えること

やっちゃんの世界では 禁止が正解

並び替え禁止仕様は必要

自動で「新しい順」だけ

UI System に保存する
