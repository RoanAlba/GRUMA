# 保存レバー（小レバー）仕様
UI セットを「自分用として保存」するための小レバーの仕様を定義する。
目的は 軽い操作で“お気に入り登録”のように保存できる儀式具 を作ること。

## 1. レバーの役割（Role）
編集画面で作った UI セットを保存する

保存した内容は私用UIに反映される

出品とは別の行為（混ぜない）

「軽い決意」の操作として存在する

## 2. レバーの形状（Shape）
種類：T型レバー（小型）

全長：38〜45mm

つまみ部分：黒マット革（小さめ）

軸：真鍮（細め）

厚み：出っ張り感を抑えるため 控えめ

→ “小さな儀式具” の雰囲気。

## 3. 素材（Material）
軸：真鍮（黄60% / 赤40%）

つまみ：黒マット革（艶 10〜15%）

根本：薄い真鍮リング（固定具）

→ 大レバーより控えめで、上品。

## 4. 動き（Motion）
動作：倒す（押すでも引くでもない）

可動角度：25〜35°

重さ：軽め（大レバーの 40〜50%）

戻り：ゆっくり自動で戻る（0.3秒）

→ “保存” らしい軽い儀式感。

## 5. 反応（Interaction）
● 倒した瞬間
真鍮の根本が 0.15秒だけ光る

UIセットが保存される

音は出さない（静けさ重視）

● 長押し
反応なし（保存は一動作のみ）

## 6. 配置（Placement）
編集画面：右下 or 左下

私用UI：基本的に表示しない（保存は編集画面だけ）

画面端からの余白：32〜40px

→ “作業の終わりに触れる位置”。

## 7. 一覧パネルとの関係（Relation）
保存レバーを倒す → UIセットが保存される

呼び出しボタン → 保存されたセットを一覧で見る

役割が完全に分離している

## 8. 禁止事項（NG Rules）
大レバーと同じ大きさにする

つまみを大きくする（出っ張り感NG）

金属光沢を強くする（安っぽくなる）

ボタン式にする（儀式感が消える）

保存と出品を同じレバーにする

## 9. 目的（Design Purpose）
保存レバーは
「軽い決意で、自分用に残すための儀式具」  
として存在する。

大レバー（出品）とは明確に差をつけ、
世界観の静けさと操作の軽さを両立させる。

✔ 最短まとめ
保存レバーは 小型の T型レバー

黒マット革 × 真鍮

軽い倒し動作で保存

光は 0.15秒だけ

編集画面にだけ置く

UI System フォルダに保存する
