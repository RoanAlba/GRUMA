宝石転送アニメーション（保存用仕様）
1. 目的
宝石庫で選んだ宝石を、
工房の回転台横の“宝石待機スポット”へ転送する儀式的な動きを定義する。

工房の世界観の“魔法の根幹”になる動作。

2. 全体の流れ
宝石を選んだ瞬間から、
回転台の横に宝石が現れるまでの一連の動き。

タップ → 浮上 → 分解 → 粒子化 → 移動 → 再構成 → 待機

すべて 静か・ゆっくり・光が弱く揺れる。

3. アニメーション詳細
● ① タップ
宝石がわずかに浮く（2〜3mm）

周囲のベルベットがほんの少し暗くなる

宝石の縁に 弱い金色の光 が走る

→ 「選ばれた」感を出す。

● ② 浮上
宝石がゆっくりと上方向へ5cmほど浮く

下に落ちる影が少し薄くなる

光源が宝石を中心に寄る

→ 儀式の始まり。

● ③ 分解
宝石の輪郭がゆっくりとほどける

表面の光が“ひび割れ”のように分離

破片ではなく 光の面 が剥がれるイメージ

→ 物質ではなく“光の存在”として扱う。

● ④ 粒子化
剥がれた光が 細かい粒子 になる

粒子は赤系なら赤、青系なら青など宝石の色を保持

粒子はゆっくりと渦を巻きながら集まる

→ 工房の“静かな魔法”の象徴。

● ⑤ 移動
粒子が細い帯になって工房側へ移動

速度はゆっくり（急がない）

帯の中心が弱く脈打つ

音はない（完全な静寂）

→ 工房の空気が少しだけ揺れる。

● ⑥ 再構成
回転台の横の“宝石待機スポット”で粒子が集まる

ゆっくりと宝石の形に戻る

最後に表面が“スッ”と光って完成

→ 儀式の終わり。

● ⑦ 待機状態
宝石は静止

下に薄い影

ほんの少しだけ光が揺れる

ロボットアームが取りに来るまで動かない

4. 色の扱い
粒子化の色は宝石の色をそのまま使う。

ルビー → 赤い粒子

サファイア → 青い粒子

エメラルド → 緑の粒子

ダイヤ → 白金色の粒子

色の粒子が工房を横切ることで、
“宝石が工房に来た” という感覚が生まれる。

5. 速度
全体のアニメーションは 約3秒。

速すぎない

遅すぎない

儀式としてちょうどいい長さ

6. UI反応
選択した宝石のケースは少し暗くなる

他の宝石は選択不可状態になる

転送が終わると宝石庫は自動で閉じる

7. 次の工程
宝石が待機スポットに現れたら、
次は ロボットアームが宝石を取りに来る。
