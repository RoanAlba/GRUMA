# 参考画像エリア仕様（修正版）

## 1. 目的
デザイン生成モードに入る前に、ユーザーが参考にしたい画像を「好きなだけ」並べて雰囲気を決めるためのエリア。

## 2. 位置
工房の左側の壁面に沿った横長スペース。  
ガラス板の上に画像が浮かぶように配置される。

## 3. 内装・質感
- ガラス板：薄い曇りガラス（マット）
- フレーム：黒アイアン
- 背景：淡いグレーのアーチ
- 光：弱いスポットライト
- 影：浅くて柔らかい

## 4. 画像の扱い
- **画像は無制限に置ける**
- 横方向にスクロールして並べられる
- 画像はふわっと浮く
- 選択した画像は少し明るくなる
- 非選択は少し暗くなる
- 画像の順番はドラッグで入れ替え可能
- 削除すると光の粒になって消える

## 5. 操作フロー
1. 画像をアップロード
2. ガラス板の上にふわっと配置される
3. 必要なら順番を入れ替える
4. デザイン生成モードに進むと、参考画像の雰囲気を反映したデザインが生成される

## 6. デザイン生成との連動
- 参考画像の「色」「雰囲気」「線の細さ」「世界観」を読み取る
- 宝石の色の傾向もここで決まる
- ただし宝石のサイズ・形は後の工程で決まる

# 参考画像エリア仕様

参考画像エリアは、下にたくさんの画像が静かに並ぶ一覧（縦スクロール）と、上に選抜した画像だけが並ぶ横スクロール帯の二階構造でできている。下の一覧では画像がタイル状に並び、スマホの写真アプリのように縦にスクロールして自由に眺めることができる。気に入った画像があれば、その画像を軽く長押しする。すると画像がふわっと浮き、上の横スクロール帯へ自動で移動して“選抜メンバー”として追加される。ドラッグ操作やボタンは不要で、ただ触るだけで選抜に入る。

横スクロール帯には選んだ画像だけが静かに横一列に並び、少数精鋭の比較エリアとして機能する。ここでは方向性を決めるための候補を落ち着いて見比べることができる。もし気が変わった場合は、横帯の画像を下方向に軽くスワイプするだけで、画像はふわっと浮いて元の一覧へ戻る。削除ではなく“選抜から外れる”だけなので、一覧にはそのまま残っている。

この仕組みによって、一覧でたっぷり迷いながら、気に入ったものだけを横帯に集め、最終的に横帯だけを見て方向性を決めるという流れが自然に成立する。操作はすべて静かで軽く、工房全体の世界観に合った触り心地になっている。
