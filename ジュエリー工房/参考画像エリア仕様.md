# 参考画像エリア仕様（修正版）

## 1. 目的
デザイン生成モードに入る前に、ユーザーが参考にしたい画像を「好きなだけ」並べて雰囲気を決めるためのエリア。

## 2. 位置
工房の左側の壁面に沿った横長スペース。  
ガラス板の上に画像が浮かぶように配置される。

## 3. 内装・質感
- ガラス板：薄い曇りガラス（マット）
- フレーム：黒アイアン
- 背景：淡いグレーのアーチ
- 光：弱いスポットライト
- 影：浅くて柔らかい

## 4. 画像の扱い
- **画像は無制限に置ける**
- 横方向にスクロールして並べられる
- 画像はふわっと浮く
- 選択した画像は少し明るくなる
- 非選択は少し暗くなる
- 画像の順番はドラッグで入れ替え可能
- 削除すると光の粒になって消える

## 5. 操作フロー
1. 画像をアップロード
2. ガラス板の上にふわっと配置される
3. 必要なら順番を入れ替える
4. デザイン生成モードに進むと、参考画像の雰囲気を反映したデザインが生成される

## 6. デザイン生成との連動
- 参考画像の「色」「雰囲気」「線の細さ」「世界観」を読み取る
- 宝石の色の傾向もここで決まる
- ただし宝石のサイズ・形は後の工程で決まる
