# ◆ 宝石トレイ.md（本文：第1版）
■ 1. 目的
宝石を工程に入れる前に一時的に置くための台。
ロボットアームが宝石を拾い上げる最初の位置として機能し、
宝石の扱い工程・セッティング台・ネックレス制作工程と連動する。

■ 2. 形状
形：浅い皿状

直径：工程サイズの宝石を 5〜8個 置ける程度

深さ：宝石が転がらない程度の浅さ

縁：ゆるやかな立ち上がり（宝石が外に出ないため）

■ 3. 内側の構造
底面は 完全なフラット

細い影のような紋様が入る（彫りではなく“影で見える程度”）

紋様の役割：

宝石の位置を把握しやすくする

光の方向を確認しやすくする

宝石同士がぶつからないよう、
ごく浅い区切り線 が入っていてもよい（目立たない程度）

■ 4. 素材
アンティークゴールド

黒ずんだシルバー

ローズブラス

いずれも
光を反射しすぎず、宝石の色を邪魔しない金属 を採用。

■ 5. 宝石を置く工程との整合性
宝石はまずこのトレイに置かれる

ロボットアームが宝石の上 3〜5mm で停止

ゆっくり下降してつまむ

つまんだ後、0.2秒静止して安定を確認

その後、セッティング台またはライン台へ移動する

（詳細は 宝石の扱い.md と ロボットアームの動き仕様.md に準拠）

■ 6. 光の扱い
スポットライトの光が柔らかく当たる位置に配置

宝石の色が見やすいよう、
底面はマット寄りの質感

宝石が浮いて見える工房の演出と矛盾しない浅さ

■ 7. 工房画像との整合性
金属の質感は工房の世界観に合わせて控えめ

宝石が整然と並ぶ“静けさ”を保つデザイン

ロボットアームの動線（トレイ → セッティング台 → ライン台）と自然につながる配置

■ 8. 状態
第1版

宝石の扱い.md と整合済み

ロボットアーム仕様と連動済み

ネックレス制作工程.md とも矛盾なし
