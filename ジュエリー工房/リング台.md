# ◆ リング台.md（本文：第1版）
■ 1. 目的
指輪に使用する宝石を、
指輪の高さ・角度を想定した位置で確認するための台。
セッティング台よりも “高さ” と “角度の見え方” を重視する。

■ 2. 形状
形：小さな円柱

直径：工程サイズの宝石より少し大きい程度

高さ：指輪の石座を想定した高さ

上面：ごく浅いくぼみ（宝石が転がらない程度）

側面：控えめなミル打ち、または滑らかな金属面

■ 3. 上面の仕様
くぼみの深さ：宝石の高さの 1/5〜1/4

角度：

完全な水平ではなく、1〜2°だけ前に傾ける

指輪を着けたときの“見え方”に近づけるため

紋様：

セッティング台よりさらに控えめ

光の方向を確認するための薄い影のラインのみ

■ 4. 素材
アンティークゴールド

黒ずんだシルバー

ローズブラス

いずれも
光を反射しすぎず、宝石の色を邪魔しない金属。

■ 5. 宝石を置く工程との整合性
ロボットアームが宝石をつまんでリング台の上へ移動

1〜2mm上で停止

0.2秒静止

ゆっくり置く

置いた後、宝石がわずかに揺れて光が変わる

ピンセットで角度を微調整（1〜2°単位）

（詳細は 宝石の扱い と ロボットアームの動き仕様 に準拠）

■ 6. 指輪特有の確認ポイント
宝石の“高さ”が指輪の印象を大きく左右するため、
リング台はセッティング台より高い

光の入り方が指輪の角度に近い

宝石の“正面”がどこになるかを決める工程に使う

指輪のデザイン図と連動して、
宝石の向きが自動でガイド表示される

■ 7. 工房画像との整合性
工房の静けさを壊さない落ち着いた金属質

スポットライトの点光源に対応

宝石が浮いて見える演出と矛盾しない高さ

セッティング台 → リング台 → 完成形 の流れが自然につながる
