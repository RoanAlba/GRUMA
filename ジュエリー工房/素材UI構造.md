# 素材切り替えUI構造（JewelryStudio Material UI）

ジュエリー工房・リファインルーム共通で使用する  
「素材切り替えUI」の構造定義。

ユーザーは素材を選ぶだけでよく、  
光は自動制御システムが最適化する。

----------------------------------------
## 1. UIの基本思想

- 素材は「壁・アイアン・机」の3カテゴリのみ選択
- 光は自動制御のため選択項目に含めない
- 床は固定（黒ベルベット）
- 選択肢はすべて「素材選択肢ファイル.md」を参照
- 初期状態は「初期セット.md」を参照

----------------------------------------
## 2. UIカテゴリ構造

### ● 壁（Wall）
- 選択肢：素材選択肢ファイルの「壁」項目を参照
- 選択すると即時反映
- 光の自動制御が壁の明るさに応じて調整される

### ● アイアン（Iron Frame）
- 選択肢：素材選択肢ファイルの「アイアン」項目を参照
- 選択すると即時反映
- 光の自動制御が線の沈み込みを補正

### ● 机（Desk）
- 選択肢：素材選択肢ファイルの「机」項目を参照
- 選択すると即時反映
- 光の自動制御が反射量を調整

----------------------------------------
## 3. 光（Light System）

光はUIに表示しない。  
以下の要素を参照して自動で最適化される。

- 壁の明るさ
- アイアンの色
- 机の素材
- 外の天候
- 宝石の種類

自動で切り替わる光：
- 青緑光（基準）
- 白金光
- 琥珀光
- 縁光（エッジライト）

----------------------------------------
## 4. UIの表示形式

### ● 3つの大きなボタン
- 壁
- アイアン
- 机

### ● ボタンを押すと素材一覧が展開
- 一覧は素材選択肢ファイルを参照して生成
- 選択すると即時反映
- 光は裏側で自動調整

----------------------------------------
## 5. 初期状態の読み込み

UI起動時に以下を読み込む：

- 壁：初期セット.md → 白いキラキラ石
- アイアン：初期セット.md → 黒アイアン（サテン）
- 机：初期セット.md → ガラス
- 光：自動制御（青緑基準）
- 床：黒ベルベット（固定）
