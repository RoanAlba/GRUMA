# 工房レイアウト構造

工房の作業工程とUI配置をまとめた基礎構造。

----------------------------------------
## 1. 左側：デザイン作業エリア（大）

- 参考画像を置く
- デザイン画を生成して大きく表示
- 分解モードで宝石なしの枠を作る
- 枠データを回転台に送る

----------------------------------------
## 2. 右側：小型モニター（設計図専用）

- 正面図 / 側面図 / 上面図
- 数値情報の確認
- デザイン画の補助表示

※ モニターは大きくしない（儀式性を守る）

----------------------------------------
## 3. 中央：回転台

- 枠データを受け取って出現
- ロボットアームと同期して動作
- 宝石セットの儀式を行う場所

----------------------------------------
## 4. 右奥：ロボットアーム

- 枠に宝石をセットする
- 動きはゆっくりで儀式的
- 先端ツールは交換式

----------------------------------------
## 5. 作業工程（全体の流れ）

1. 左側に参考画像を置く  
2. 左側でデザイン画を生成  
3. 分解して宝石なしの枠を作る  
4. 枠を回転台に送る  
5. ロボットアームが宝石をセット  
6. リファインルームで仕上げ  

----------------------------------------
