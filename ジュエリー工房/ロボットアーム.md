# ロボットアーム仕様（Gem-Arm v1）

ジュエリー工房の机に固定されている  
宝石セッティング専用のロボットアーム。

----------------------------------------
## 1. 役割
- 宝石をつまむ
- 宝石をはめる
- 宝石を外す
- 彫刻パーツの付け替え
- 回転ステージとの同期動作

----------------------------------------
## 2. 動きの特徴
- 動きは機械的で正確（選択：B）
- 速度はゆっくり（儀式性を壊さない）
- つまむ瞬間だけ力が変化する
- 宝石を扱う時だけ先端が光る（弱い青緑）

----------------------------------------
## 3. 取り付け位置
- 机の右奥に固定
- ベースは黒アイアン
- アームは細い黒アイアン＋関節部に薄い金属光

----------------------------------------
## 4. 先端ツール
- 宝石用ピンセット型
- 彫刻パーツ用の小型クランプ
- 先端は交換式

----------------------------------------
## 5. 回転ステージとの連動
- ステージが止まる → アームが動く
- アームが止まる → ステージが回る
- 同時動作はしない（儀式性のため）
