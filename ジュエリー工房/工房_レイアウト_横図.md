# 工房_レイアウト_横図.md（正式版：第1版）

## ■ 1. 目的
工房を横方向（サイドビュー）から見たときの  
各台の高さ・位置・光の方向・ロボットアームの動線を統一するためのレイアウト図。

## ■ 2. 工房の横図（サイドビュー）

（奥）
   ┌─────────────── 透明ディスプレイ ───────────────┐
   │   ・高さ：胸〜頭の位置                                   │
   │   ・図面モードあり                                       │
   └──────────────────────────────────────┘

        ↑ スポットライト（上から45°）
        │

【ロボットアーム（高い位置から前方へ伸びる）】
        │
        ▼

  ┌───────────┐
  │ セッティング台 │  ← 高さ：中（基準）
  └───────────┘
        │
        ▼（用途別へ）

  ┌───────────┐      ┌───────────┐
  │ リング台       │      │ ライン台       │
  │（少し高い）     │      │（低め・横長）   │
  └───────────┘      └───────────┘

        ▼（最終工程）

  ┌───────────┐
  │ リファイン仕上げ │  ← 高さ：中〜やや高め
  └───────────┘

（手前）
  ┌───────────┐
  │ 宝石トレイ     │  ← 一番低い位置
  └───────────┘

## ■ 3. 高さの基準
| 台 | 高さ | 役割 |
|---|---|---|
| 宝石トレイ | 低い | 宝石が最初に着地する場所 |
| セッティング台 | 中（基準） | 光・角度の確認 |
| リング台 | やや高い | 指輪の高さを再現 |
| ライン台 | 低め | ネックレスの横並びを確認 |
| リファイン仕上げ | 中〜やや高め | 最終の光調整 |
| 透明ディスプレイ | 高い | 図面・情報表示 |

## ■ 4. ロボットアームの動線（横方向）
ロボットアームは高い位置から前方へ伸びる。

宝石トレイ  
　↓（つまむ）  
セッティング台  
　↓（用途別へ）  
リング台 or ライン台  
　↓  
リファイン仕上げ  

## ■ 5. 光の方向
工房の光は上から45°が基本。

- 宝石トレイ：柔らかい光  
- セッティング台：基準光  
- リング台：前傾1〜2°に合わせて光が当たる  
- ライン台：横方向の光が少し強い  
- リファイン仕上げ：光が宝石の表面を走る演出  

## ■ 6. 宝石の見え方（横方向）
- 宝石トレイ：低い → “置かれている”  
- セッティング台：中 → “浮いて見える”  
- リング台：高い → 指輪の高さを再現  
- ライン台：低い → 横並びの流れ  
- リファイン仕上げ：中〜高 → 完成品が浮かび上がる
