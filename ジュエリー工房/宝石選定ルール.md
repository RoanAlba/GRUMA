# 宝石選定ルール

1. サイズが一致していること（最優先）
- デザイン生成モードで決まった宝石サイズと完全一致するものだけ候補になる。

2. 形（カット）が一致していること
- ラウンドならラウンド、オーバルならオーバルなど、枠の爪・台座に合わせる。

3. 色は自由（色違いOK）
- 同じサイズ・同じ形なら色違いの宝石を自由に選べる。
- 例：ルビー（赤）、ピンクサファイア（ピンク）、ガーネット（深赤）など。

4. 宝石庫に出現する宝石
- 上記条件を満たす宝石だけが宝石庫に並ぶ。
- 完全に関係ない宝石（サイズ違い・形違い・色系統違い）は出てこない。

5. 宝石庫の動作
- デザインが決まると、条件に合う宝石だけが自動で並ぶ。
- 色違いが3〜5個ふわっと出現し、選ぶ楽しさを残す。
