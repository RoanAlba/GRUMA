# ◆ 宝石庫_UI構造.md（整理版：第2版）
※ このファイルは「宝石庫.md」で定義された空間設定を前提としている。
■ 1. 目的
宝石庫における UIの配置・画面構成・操作フロー を定義する。
宝石庫.md で定義された “儀式的な空間” を前提に、
ユーザーが宝石を選ぶ体験を美しく、静かに、迷いやすく設計する。

■ 2. 画面全体構成
画面は 左・中央・右 の3分割。

● 左：宝石棚（3段）
上段：透明系

中段：色石

下段：大きめ・特殊カット

並ぶ宝石は 最大5〜6個

宝石は棚に“浮いているように”表示

選択されていない宝石は少し暗く表示

● 中央：宝石の拡大ビュー
宝石をタップすると中央に拡大

360度回転

光の当たり方を変更できる

背景は黒または紺のベルベット

宝石の“本来のサイズ”で表示（工程サイズではない）

● 右：宝石情報パネル
宝石名

色

カット

サイズ（デザインと一致）

特徴（透明度・彩度など）

「この宝石を使う」ボタン

■ 3. 宝石の表示ルール
宝石は光の粒から形になる（宝石庫.md に準拠）

選択中の宝石は弱い金色の縁取り

非選択時は少し暗く

ホバー時（またはタップ前）はわずかに浮く演出

宝石の影は極めて薄く、浮遊感を保つ

■ 4. 操作フロー
宝石庫がふわっと開く

宝石が光の粒から形になる

宝石をタップして中央に拡大

光の角度を変えて確認

「この宝石を使う」を押す

宝石が光の粒にほどける

工房へ転送（宝石の転送アニメーション.md に準拠）

■ 5. UIアニメーション
タップ時：宝石がわずかに浮く

選択時：金色の縁取りが出る

非選択時：暗くなる

拡大ビュー：ゆっくり回転

転送時：宝石が光の粒に分解される

全体の動きは 静かでゆっくり

■ 6. 工房との連動
「この宝石を使う」ボタン押下 → 転送アニメーション開始

画面は宝石庫 → 工房へフェード

粒子の流れは画面をまたいで連続

工房側では宝石トレイがハイライトされる

完了後、ロボットアームが宝石を認識して動作開始

■ 7. 工房画像との整合性
宝石庫は工房より暗い

光源は弱いスポットライトのみ

宝石だけが光る

UIの色は黒・紺・金を基調

工房の“静けさ”と矛盾しない演出

■ 8. 状態
第2版（整理版）

宝石庫.md と完全連動

宝石の転送アニメーション.md と整合

工房_動線図.md と矛盾なし
