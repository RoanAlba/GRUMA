# 初期セット（JewelryStudio InitialSet）

ジュエリー工房・リファインルーム共通で使用する  
「最初に読み込まれる空間構成」。

素材選択肢ファイルの中から、  
初期状態で適用される素材を定義する。

----------------------------------------
## 1. 壁（Wall）
- 白いキラキラ石（クォーツ系）
  - 微細な光粒
  - 反射弱め
  - 石積みの境界 0.5〜1mm

----------------------------------------
## 2. アイアン（Iron Frame）
- 黒アイアン（サテン寄り）
  - エッジ 0.5〜1mm 面取り
  - 接合部は段差をつけて見せる工芸仕様

----------------------------------------
## 3. 机（Desk）
- ガラス（透明・ノイズなし）

----------------------------------------
## 4. 光（Light System）
- 自動制御（青緑光を基準）
  - 壁・アイアン・机・天候・宝石に応じて自動調整
  - 白金光 / 琥珀光 / 縁光 に自動切替

----------------------------------------
## 5. 床（Floor）
- 黒ベルベット（毛足短い）
  - 宝石の舞台
  - ここだけは固定
