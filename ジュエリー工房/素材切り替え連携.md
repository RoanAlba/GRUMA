# 素材切り替え連携（Material UI Integration）

ジュエリー工房（JewelryStudio）と  
リファインルーム（RefineRoom）が  
同じ素材切り替えUIを使用するための連携仕様。

----------------------------------------
## 1. 参照ファイル

- 素材選択肢ファイル.md  
- 初期セット.md  
- 素材UI構造.md  

----------------------------------------
## 2. 連携の目的

- 工房とリファインで同じ素材セットを使用する  
- UI の修正を1箇所で済ませる  
- 世界観の統一（壁・アイアン・机の整合性）  
- 光の自動制御を両方で同じロジックにする  

----------------------------------------
## 3. 連携の仕組み

### ● 工房（JewelryStudio）
- UI を直接読み込む  
- 初期セットをそのまま適用  
- 素材変更は即時反映  
- 光は自動制御（UIに表示しない）

### ● リファインルーム（RefineRoom）
- 工房と同じ UI を参照  
- 初期セットも工房と同じ  
- 素材変更は工房と同じロジックで反映  
- 光は自動制御（工房と同じアルゴリズム）

----------------------------------------
## 4. UI の共通化ポイント

- 壁・アイアン・机の3カテゴリのみ選択  
- 光は UI に出さず自動制御  
- 床は固定（黒ベルベット）  
- 選択肢はすべて素材選択肢ファイルから生成  
- 初期状態は初期セットから読み込む  

----------------------------------------
## 5. 反映の流れ

1. 初期セット.md を読み込む  
2. 素材選択肢ファイル.md を読み込む  
3. 素材UI構造.md を読み込む  
4. 工房・リファインの両方に UI を適用  
5. 光の自動制御が素材に応じて動作  

----------------------------------------
