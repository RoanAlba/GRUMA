# つまみのジュエリー化と保存ルール

## ■ つまみは宝石でできている
スライダーのつまみは、宝石と金属枠から構成される小さなアンティークジュエリーとして扱う。
UI 要素でありながら、アンティーク思想に基づく宝飾物として設計される。

## ■ つまみはアンティークジュエリーに自動変換される
宝石をセッティング台に置くと、アンティーク思想に従って
形・枠・爪・色・裏側構造が自動で決定され、ジュエリーとして完成する。

## ■ 変換されたジュエリーは保存される
完成したジュエリーは「保存」操作によってコレクション化される。
保存されたジュエリーは別室（ジュエリールーム）に飾られ、後から閲覧・再利用できる。

## ■ 宝石の形は自由に変わる
つまみの宝石は、ラウンド・オーバル・ペアシェイプ・クッション・マーキスなど
多様な形に変化する。特定の形（例：スクエアカボション）に固定しない。

## ■ デフォルト設定（初期値）
自由に変更できるが、世界を動かすための初期値として以下を採用する。
- デフォルト形：ラウンド
- デフォルトカット：カボション
- デフォルト色：淡い宝石（UI 全体の世界観と統一）

※ デフォルトは初期値であり、変化後の宝石の形・色・カットは完全に自由。

## ■ UI とアンティーク思想の接続
つまみの造形・色・裏側構造は、design/antique の思想に基づいて決定される。
UI とアンティークジュエリーの世界観を統一するための基礎ルールとなる。
