# つまみのジュエリー化と保存ルール

## ■ つまみは宝石でできている
スライダーのつまみは、宝石と金属枠から構成される小さなアンティークジュエリーとして扱う。

## ■ つまみはアンティークジュエリーに自動変換される
宝石をセッティング台に置くと、アンティーク思想に基づき、
形・枠・爪・色・裏側構造が自動で決定され、ジュエリーとして完成する。

## ■ 変換されたジュエリーは保存できる
完成したジュエリーは「保存」操作によってコレクション化される。
保存されたジュエリーは別室（ジュエリールーム）に飾られる。

## ■ 宝石の形は固定しない
つまみの宝石は、ラウンド・オーバル・ペアシェイプ・クッションなど、
多様な形に変化する。スクエアカボションに固定しない。

## ■ UI とアンティーク思想の接続
スライダーのつまみは UI 要素であると同時に、
design/antique の思想に基づく宝飾物として設計される。
