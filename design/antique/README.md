# antique フォルダについて

このフォルダは、GRUMA における「アンティーク設計思想」をまとめた場所です。
アンティーク特有の色・形・裏側構造・連鎖の考え方を、4つの基礎ファイルとして整理しています。

## ■ ファイル構成
- 01_アンティーク設計の全体構造  
  アンティーク思想の全体像と、爪・裏側・影・色の連鎖構造。

- 02_色の設計思想  
  濁り・ムラ・黒金属・ミルク色・異物色など、アンティーク特有の色の哲学。

- 03_形の設計思想  
  爪の造形、段差、歪み、影の形、空気の隙間など、形の特徴と理由。

- 04_裏側の癖と表への変換ルール  
  裏側の構造（光穴・段差・深さ）が、表のデザインにどう影響するかの変換ルール。

## ■ 読み方
01 → 02 → 03 → 04 の順で読むと、アンティークの世界観が連鎖的に理解できます。
