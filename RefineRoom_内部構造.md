# RefineRoom_Machines（リファインルーム・マシンズ）
GRUMA における「作品の身体」を生成する内部装置世界。  
RefineRoom（魂）で行われた操作・意図・儀式を受け取り、  
形・光・色・質感・仕上げとして実体化する中枢。

RefineRoom_Machines は “部屋” ではなく、  
RefineRoom の裏側に広がる巨大な内部世界として存在する。

---

# 1. 役割（Roles）
RefineRoom_Machines の役割は以下の通り。

- 作品の実体生成（5ライン）
- 外部塔とのエネルギー同期（雷）
- 内部光柱によるエネルギー循環
- 操作と実体化の同期（0.1〜0.2秒）
- Undo の自然な逆再生
- 内部世界の奥行き・生命感の維持
- 静寂儀式（竪琴）による内部の静止

---

# 2. 内部世界の構造（World Structure）
RefineRoom_Machines は以下の層で構成される。

- **中央エネルギー柱（Energy Pillar）**  
  外部塔の雷と同期し、内部世界の中心軸となる光柱。

- **エネルギー循環管（Energy Tubes）**  
  ガラス管のような透明管。光が流れ、装置群へ分岐する。

- **装置群（Machines Cluster）**  
  5ラインの装置が壁際に配置され、光を受けて動作する。

- **粒子レイヤー（Particle Layer）**  
  微細な光粒子が空気中に漂い、工程に応じて動きが変化する。

- **奥行き構造（Depth Layer）**  
  遠景に大型装置が存在し、雷の強さで振動が変わる。

---

# 3. 5ライン構造（Five Lines）
RefineRoom_Machines は 5 つのラインで構成される。

1. **成形ライン（Shape Line）**  
   形状・厚み・曲率・内部構造を生成。

2. **光反射ライン（Light Line）**  
   光量・反射・角度・揺れを処理。

3. **色処理ライン（Color Line）**  
   色相・彩度・濁り・透明度を生成。

4. **微粒子ライン（Texture Line）**  
   粒子・粗さ・ザラつき・表面深度を生成。

5. **仕上げライン（Finish Line）**  
   統合処理・最終光・最終質感・不可逆固定。

---

# 4. 外部塔との同期（Energy Sync）
RefineRoom_Machines は外部の回転塔とゆるやかに同期する。

- 雷が弱い → 光柱は白く静か  
- 雷が中 → 淡青が混じり、壁面に揺らぎ  
- 雷が強い → 青白く脈動し、装置の反射が鋭くなる  

ガラス管の光も雷に応じて速度と色が変化する。

---

# 5. 内部の音（Internal Sounds）
RefineRoom_Machines は “静けさを壊さない音” だけを持つ。

- 光柱の低い脈動音「ゴォ…」  
- ガラス管の細い共鳴音「サラ…」「チリ…」  
- 小型装置の律動音「コト…コト…」  
- 大型装置の共鳴音「ボウ…」  
- 雷が強い時の金属音「キン…」  

すべてが干渉せず、内部世界は “透明な楽器” のように調和する。

---

# 6. 工程切り替え（Line Switching）
工程を切り替えると、対応するラインが起動する。

- 成形 → ローラーが低速回転  
- 光 → 反射板に光が走る  
- 色 → タンクの液体が揺れる  
- 質感 → 粒子ラインが開く  
- 仕上げ → プレスアームが待機位置へ移動  

切り替え時、ガラス管の光が一度だけ「ピッ」と明滅する。

---

# 7. Undo（逆再生）
Undo は内部装置の動きを自然に巻き戻す。

- 成形 → ローラー逆回転  
- 光 → 反射板が逆角度へ戻る  
- 色 → 液体が沈む  
- 質感 → 粒子が収束  
- 仕上げ → プレスアーム上昇  

プレビューも同時に逆再生される。

---

# 8. 竪琴（静寂儀式装置）
RefineRoom_Machines 内には竪琴が存在する。

- 竪琴を起動すると内部音がすべて止まる  
- 光柱は弱まり、ガラス管の光は呼吸のように明滅  
- 粒子はゆっくり沈む  
- 内部世界が “眠る” ような静の状態になる  

演奏が終わると、内部世界はゆっくり再起動する。

---

# 9. 内部光の種類（Light Types）
内部装置には 3 種類の光が存在する。

1. **流れる光（ライン）**  
   ガラス管を「ス…」と走る光。

2. **粒子の光（光の砂）**  
   空中に漂う微細な光粒子。

3. **脈動する光（呼吸の光）**  
   装置中心部の淡い明滅。

---

# 10. 内部世界の導線（Flow Path）
内部世界は自然な導線で構成される。

- 中央の光柱が視界に入る  
- ガラス管の光が進む方向を示す  
- 装置群が四方に広がる  
- 奥の大型装置へ続く  
- 外部塔へ向かう通路がある  

内部世界は “ひとつの循環系” として機能する。

---

# 11. RefineRoom との関係（Integration）
RefineRoom の操作はすべて RefineRoom_Machines に入力される。

- 形 → 成形ライン  
- 光 → 光反射ライン  
- 色 → 色処理ライン  
- 質感 → 微粒子ライン  
- 仕上げ → 仕上げライン  
- 素材 → 成形ライン  
- 図面モード → 成形ラインの線図出力  
- 保存 → 仕上げライン  
- 出品 → 仕上げライン（不可逆固定）

操作 → 実体化 → プレビュー  
という循環で作品が生成される。

---

# 12. 目的（Purpose）
RefineRoom_Machines の目的は以下の通り。

- 作品の実体を生成する  
- 操作と実体化を同期させる  
- 内部世界の生命感を維持する  
- 雷と光柱による世界観の深層を形成する  
- Undo を自然に巻き戻す  
- GRUMA 全体の奥行きを生み出す
