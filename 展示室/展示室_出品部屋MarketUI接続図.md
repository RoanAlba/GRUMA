# 展示室 → 出品部屋 → MarketUI 接続図（正式版）

作品が展示室から市場へ届くまでの  
全ステップを視覚化した接続図。

---

# ■ 1. 全体フロー図

展示室  
　↓（裏側：出品準備レーン）  
出品部屋  
　↓（ゲートの儀式）  
MarketUI

---

# ■ 2. 展示室 → 出品準備レーン
- タグ整合性チェック  
- データ最終整理  
- モックアップ生成  
- 作品データの最終版確定  
- UIは非表示（裏側処理）

---

# ■ 3. 出品準備レーン → 出品部屋
- 作品が「儀式の空気」に切り替わる  
- 読み取り専用UIに変換  
- 出品ボタンが有効化される

---

# ■ 4. 出品部屋 → MarketUI
- ゲートが開く  
- 作品が吸い込まれる  
- MarketUIの空気に変換  
- 初期表示ルールに従って配置される

---

# ■ 5. このファイルの役割
- 3つの部屋の関係を明確化  
- 作品フローの誤解を防ぐ  
- 世界観の境界線を固定する
