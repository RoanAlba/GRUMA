# 展示室 作品詳細UI 図面（正式版 / 保存用）

## 1. 概要
本図面は、展示室の「作品詳細UI」のレイアウト構造・寸法・比率を定義する。  
作品詳細UIは、作品を静かに鑑賞するための“儀式的な閲覧空間”であり、  
余白・光・中央配置を重視した構造を採用する。

---

## 2. 画面全体の構成（俯瞰図）

+--------------------------------------------------------------+
|                                                              |
|                     【作品メイン画像】                       |
|                （中央・大きく・余白多め）                    |
|                                                              |
+--------------------------------------------------------------+

---

## 3. 寸法仕様（px基準）

### 3-1. 作品メイン画像
- 最大幅：画面幅の **70%**
- 最大高さ：画面高さの **60%**
- 角丸：6px
- 影：極薄（静けさ優先）
- 上余白：**60px**

### 3-2. タイトル（作品名）
- フォントサイズ：20〜24px
- 色：#333
- 中央揃え
- 上余白：24px

### 3-3. 説明文（Description）
- フォントサイズ：14〜16px
- 色：#555
- 最大3〜4行
- 中央揃え
- 上余白：12px

### 3-4. 素材情報（Material Info）
- フォントサイズ：12〜13px
- 色：#8c8c8c
- 中央揃え
- 上余白：10px

### 3-5. 閉じるボタン（真鍮）
- 直径：28px
- 色：真鍮（#b08c4f）
- 位置：右上固定（画面右上から 24px / 24px）
- ホバー：影が少し濃くなる

---

## 4. レイアウト比率（重要）
- 画面中央に縦軸を置き、全要素を **中央揃え** にする
- 作品画像：全体の **最重要要素（視線の中心）**
- テキスト類：画像の下に静かに積む
- 閉じるボタン：右上に固定し、視線を邪魔しない位置

---

## 5. 階層構造（Z-index）
1. 閉じるボタン（最上位）
2. 作品メイン画像
3. タイトル
4. 説明文
5. 素材情報
6. 背景（白磁）

---

## 6. 禁止事項
- 派手なアニメーション
- 枠線の強調
- 作品画像の拡大アニメーション
- 長文説明
- 背景を動かす

---

## 7. 目的と効果
- 作品を静かに鑑賞できる“儀式的な空間”を作る
- 本棚UIとの世界観を統一する
- 作品の魅力を最大限に引き出す
- 工房 → 展示室 → 詳細 の流れを自然に繋ぐ
