# ■ 2. UI要素の配置（詳細）

## ● 2-1. 左エリア（空白）
- 何も置かない  
- 視線を中央へ誘導するための “呼吸スペース”  
- ExhibitionRoom の静けさを継承する役割

---

## ● 2-2. 中央エリア（作品プレビュー）
- 透明台座＋作品  
- 作品は画面中央に固定  
- スクロールしても動かない  
- 影は極薄  
- 動画の場合も静かにループ（音なし）

**中央は “儀式の主役” の位置。**

---

## ● 2-3. 右エリア（出品情報）
- 読み取り専用  
- スクロールは右エリアのみ発生  
- 表示項目（最小構成）  
  - タイトル  
  - 説明文  
  - タグ  
  - 出品形式  
  - 注意書き  
- 情報は壁に浮かぶ透明ディスプレイとして表示  
- ExhibitionRoom より情報密度は少し高いが、MarketUI よりは低い

---

## ● 2-4. 右下（出品ボタン）
- 真鍮の円形ボタン  
- 直径：32px  
- 常に画面右下に固定  
- スクロールしても位置は変わらない  
- 押下時：わずかに暗くなる（0.1秒以内）  
- 押すとゲートが開き、儀式が始まる

---

# ■ 3. 視線誘導（UX設計）

1. 入室 → **中央の作品**  
2. 作品を確認 → **右側の情報**  
3. 情報を確認 → **右下の出品ボタン**  
4. ボタン押下 → **奥のゲートへ視線が移動**  

→ **一本道で迷わない UI 動線**  
→ ExhibitionRoom の静けさを保ちながら、MarketUI へ繋ぐ

---

# ■ 4. 配置ルール（UIの原則）
- UIは “壁に浮かぶ透明ディスプレイ” として扱う  
- 作品の上に UI を重ねない  
- 情報量は最小限  
- アニメーションはフェードのみ  
- 色は白磁・灰磁・真鍮の 3 色に限定  
- MarketUI の UI を流用しない（世界観が違うため）

---
